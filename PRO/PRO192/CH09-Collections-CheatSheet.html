Collection Framework Hierarchy

Iterable
  └── Collection
        ├── List (ordered, duplicates OK)
        │     ├── ArrayList (array-backed, fast random access)
        │     ├── LinkedList (doubly-linked, fast add/remove)
        │     └── Vector (synchronized, legacy)
        ├── Set (no duplicates)
        │     ├── HashSet (hash table, O(1) ops, no order)
        │     ├── LinkedHashSet (insertion order maintained)
        │     └── TreeSet (sorted, O(log n) ops)
        └── Queue / Deque
              ├── LinkedList (also implements Deque)
              ├── ArrayDeque (stack/queue, faster than Stack)
              └── PriorityQueue (heap-based, natural order)
Map (key-value, separate hierarchy)
  ├── HashMap (hash table, O(1), no order)
  ├── LinkedHashMap (insertion order)
  ├── TreeMap (sorted by key, O(log n))
  └── Hashtable (synchronized, legacy)
Tip
For most use cases: ArrayList for lists, HashMap for key-value, HashSet for uniqueness, ArrayDeque for stacks/queues

List — ArrayList vs LinkedList

// ArrayList — preferred for most cases
List<String> list = new ArrayList<>();
list.add("Alice");           // append
list.add(0, "Bob");          // insert at index
list.remove("Alice");        // remove by value
list.remove(0);              // remove by index
list.get(1);                 // O(1) random access
list.size();
list.contains("Bob");
list.isEmpty();
list.clear();
list.set(0, "Charlie");      // replace at index
Collections.sort(list);      // sort
list.sort(Comparator.naturalOrder());

// LinkedList — when you need efficient insertion at head/tail
LinkedList<String> ll = new LinkedList<>();
ll.addFirst("first");
ll.addLast("last");
ll.removeFirst();
ll.pollFirst();  // removes and returns, null if empty

Map — HashMap, TreeMap, LinkedHashMap

Map<String, Integer> scores = new HashMap<>();

// Basic operations
scores.put("Alice", 95);
scores.put("Bob", 87);
int alice = scores.get("Alice");       // 95
int charlie = scores.getOrDefault("Charlie", 0); // 0
scores.containsKey("Bob");             // true
scores.containsValue(87);             // true
scores.remove("Bob");
scores.size();

// Iterate entries
for (Map.Entry<String, Integer> entry : scores.entrySet()) {
    System.out.println(entry.getKey() + "=" + entry.getValue());
}

// Java 8+ methods
scores.putIfAbsent("Dave", 75);
scores.merge("Alice", 5, Integer::sum);  // Alice becomes 100

// Compute

Sets & Queues / Deques

Sets
// HashSet — no duplicates, no order
Set<String> set = new HashSet<>();
set.add("apple");
set.add("banana");
set.add("apple");  // ignored — duplicate
set.size();        // 2
set.contains("apple");  // true
set.remove("banana");

// Set operations
Set<Integer> a = new HashSet<>(Arrays.asList(1,2,3,4));
Set<Integer> b = new HashSet<>(Arrays.asList(3,4,5,6));

// Union
Set<Integer> union = new HashSet<>(a);
union.addAll(b);  // {1,2,3,4,5,6}

// Intersection
Set<Integer> intersect = new HashSet<>(a);
intersect.retainAll(b);  // {3,4}

// Difference
Set<Integer> diff = new HashSet<>(a);
diff.removeAll(b);  // {1,2}
TreeSet (sorted)
TreeSet<String> sorted = new TreeSet<>();
sorted.add("banana"); sorted.add("apple"); sorted.add("cherry");
// Iterates in order: apple, banana, cherry
sorted.first();       // "apple"
sorted.last();        // "cherry"
sorted.floor("b");    // "banana" (largest <= "b")
sorted.ceiling("b");  // "banana" (smallest >= "b")
sorted.headSet("c");  // {"apple","banana"}
sorted.tailSet("b");  // {"banana","cherry"}
Queues & Deques
// Queue (FIFO)
Queue<String> queue = new LinkedList<>();
queue.offer("first");   // add to tail
queue.offer("second");
queue.peek();    // view head without removing: "first"
queue.poll();    // remove and return head: "first"

// ArrayDeque (more efficient than LinkedList for queue/stack)
Deque<Integer> deque = new ArrayDeque<>();
deque.offerFirst(1);  // add to front
deque.offerLast(2);   // add to back
deque.peekFirst();    // view front
deque.peekLast();     // view back
deque.pollFirst();    // remove front
deque.pollLast();     // remove back

// Stack (use Deque, not legacy Stack class)
Deque<String> stack = new ArrayDeque<>();
stack.push("bottom");
stack.push("top");
stack.peek();    // "top"
stack.pop();     // "top"

Iterators

// Using Iterator explicitly
List<String> list = new ArrayList<>(Arrays.asList("a","b","c"));
Iterator<String> iter = list.iterator();
while (iter.hasNext()) {
    String s = iter.next();
    if (s.equals("b")) {
        iter.remove();  // safe removal during iteration!
    }
}

// ListIterator (bidirectional, for List only)
ListIterator<String> li = list.listIterator(list.size());
while (li.hasPrevious()) {
    System.out.println(li.previous()); // iterate backwards
}
Warning
Never modify a collection while iterating with for-each — throws ConcurrentModificationException. Use Iterator.remove() or collect to a list then remove.
for-each (Iterable)
// Internally uses Iterator
for (String s : list) {
    System.out.println(s);  // don't remove here!
}

// Safe removal pattern
list.removeIf(s -> s.startsWith("a"));  // Java 8+

Collection Streams Overview (Java 8+)

Streams Preview
Full coverage in Volume II Chapter 1
List<String> names = Arrays.asList("Alice","Bob","Charlie","Dave");

// filter + map + collect
List<String> result = names.stream()
    .filter(n -> n.length() > 3)   // Charlie, Dave, Alice
    .map(String::toUpperCase)       // CHARLIE, DAVE, ALICE
    .sorted()                       // ALICE, CHARLIE, DAVE
    .collect(Collectors.toList());

// count
long count = names.stream()
    .filter(n -> n.startsWith("A"))
    .count();  // 1

// reduce
int totalLen = names.stream()
    .mapToInt(String::length)
    .sum();  // 5+3+7+4 = 19

// groupingBy
Map<Integer, List<String>> byLength = names.stream()
    .collect(Collectors.groupingBy(String::length));
Tip
Stream operations are lazy — nothing executes until a terminal operation (collect, count, reduce) is called
