Determinant ‚Äî 2√ó2 Definition & 3√ó3 Formula
Definition ‚Äî 2√ó2
For A = [[a, b], [c, d]], the determinant is:
$$\det\begin{pmatrix}a & b \\ c & d\end{pmatrix} = ad - bc$$
A is invertible $\iff \det A \neq 0$.
3√ó3 Expansion Formula
For A = [[a,b,c],[d,e,f],[g,h,i]]:
$$\det A = a(ei-fh) - b(di-fg) + c(dh-eg)$$ $$= aei + bfg + cdh - ceg - afh - bdi$$
Signs on row 1 alternate: $+\,-\,+$.
Sign Pattern
The sign of position $(i,j)$ is $(-1)^{i+j}$. The grid alternates:
$$\begin{pmatrix}+&-&+&\cdots\\-&+&-&\cdots\\+&-&+&\cdots\end{pmatrix}$$
Cofactor Expansion (Laplace Expansion)
Definition 3.1 ‚Äî Cofactor
Given n√ón matrix A, let A_ij = (n‚àí1)√ó(n‚àí1) matrix obtained by deleting row i and column j. The (i,j)-cofactor is:
$$c_{ij}(A) = (-1)^{i+j} \det(A_{ij})$$
Definition 3.2 ‚Äî Cofactor Expansion along Row 1
$$\det A = a_{11}c_{11}(A) + a_{12}c_{12}(A) + \cdots + a_{1n}c_{1n}(A)$$ $$= \sum_{j=1}^{n} a_{1j}\,c_{1j}(A)$$
Theorem 3.1.1 ‚Äî Cofactor Expansion (Laplace)
det A can be computed by cofactor expansion along any row or column: multiply each entry by its cofactor and sum. The result is always det A.
$$\det A = \sum_{j=1}^{n} a_{ij}\,c_{ij}(A) \quad \text{[expand along row }i\text{]}$$ $$\det A = \sum_{i=1}^{n} a_{ij}\,c_{ij}(A) \quad \text{[expand along col }j\text{]}$$
Strategy
Choose the row or column with the most zeros to minimize computation.

Triangular Matrix Determinant & Block Form
Theorem 3.1.4 ‚Äî Triangular Matrices
If A is upper or lower triangular, then:
$$\det A = a_{11} \cdot a_{22} \cdot a_{33} \cdots a_{nn}$$ $$= \text{product of main diagonal entries}$$
Example
$$A = \begin{pmatrix}a&0&0&0\\u&b&0&0\\v&w&c&0\\x&y&z&d\end{pmatrix} \implies \det A = abcd$$
Theorem 3.1.5 ‚Äî Block Triangular
For block matrices with A (k√ók) and B square:
$$\det\begin{pmatrix}A & X \\ 0 & B\end{pmatrix} = \det A \cdot \det B$$ $$\det\begin{pmatrix}A & 0 \\ Y & B\end{pmatrix} = \det A \cdot \det B$$
Theorem 3.1.6 ‚Äî Linearity in a Column
Fix all columns except column j. The map x ‚Ü¶ det[c‚ÇÅ ¬∑¬∑¬∑ x ¬∑¬∑¬∑ c‚Çô] is linear:
$$T(\mathbf{x}+\mathbf{y}) = T(\mathbf{x}) + T(\mathbf{y}) \quad \text{and} \quad T(a\mathbf{x}) = a\,T(\mathbf{x})$$
¬ß3.2 Determinants and Matrix Inverses
Theorems 3.2.1‚Äì3.2.5 ¬∑ Definition 3.3
Product Theorem & Invertibility
Theorem 3.2.1 ‚Äî Product Theorem (Cauchy, 1812)
For any n√ón matrices A and B:
$$\det(AB) = \det A \cdot \det B$$
Extends inductively: $\det(A_1 A_2 \cdots A_k) = \det A_1 \cdot \det A_2 \cdots \det A_k$. In particular: $\det(A^k) = (\det A)^k$.
Theorem 3.2.2 ‚Äî Invertibility Criterion
A is invertible ‚ü∫ det A ‚âÝ 0. When invertible:
$$\det(A^{-1}) = \frac{1}{\det A}$$
Theorem 3.2.3 ‚Äî Transpose
For any square matrix A:
$$\det(A^T) = \det A$$
Key Identities
  • $A$ orthogonal ($A^{-1} = A^T$) $\implies \det A = \pm 1$
  • $\det(AB) = \det(BA)$ for any $A$, $B$ of the same size
  • If $A_1 \cdots A_k$ invertible, then each $A_i$ is invertible
Adjugate and Inverse Formula
Definition 3.3 ‚Äî Adjugate
The adjugate (classical adjoint) of A is the transpose of the cofactor matrix:
$$\operatorname{adj}(A) = [c_{ij}(A)]^T$$ $$\text{so } [\operatorname{adj} A]_{ij} = c_{ji}(A)$$
For $2\times 2$: $\operatorname{adj}\!\begin{pmatrix}a&b\\c&d\end{pmatrix} = \begin{pmatrix}d&-b\\-c&a\end{pmatrix}$.
Theorem 3.2.4 ‚Äî Adjugate Formula
For any square A:
$$A \cdot \operatorname{adj}(A) = \det(A) \cdot I = \operatorname{adj}(A) \cdot A$$
If det A ‚âÝ 0, then:
$$A^{-1} = \frac{1}{\det A}\,\operatorname{adj}(A)$$
Theoretical vs. Practical
The adjugate formula gives a closed-form expression for A‚Åª¬π, useful in theory. For computation, Gaussian elimination is far more efficient (n¬≤ cofactors each requiring (n‚àí1)√ó(n‚àí1) determinants).
Useful Result
$\det(\operatorname{adj} A) = (\det A)^{n-1}$ for n√ón A, n ‚â• 2.
Cramer's Rule
Theorem 3.2.5 ‚Äî Cramer's Rule
If A is an invertible n√ón matrix, the unique solution to Ax = b is:
$$x_k = \frac{\det(A_k)}{\det(A)}, \quad k = 1, 2, \ldots, n$$
where A‚Çñ is A with column k replaced by b.
Example ‚Äî 2√ó2
System: $ax + by = e$, $cx + dy = f$ ($A$ invertible, $\det A = ad-bc \neq 0$).
$x_1 = \dfrac{ed-bf}{ad-bc}$,  $x_2 = \dfrac{af-ce}{ad-bc}$
Note ‚Äî Practical Limits
Cramer's Rule is theoretically elegant but computationally expensive for large n. Computing all n determinants costs roughly the same as solving by Gaussian elimination. Its main value is theoretical (e.g., proving solution formulas exist).

Definitions 3.4‚Äì3.5 ¬∑ Theorems 3.3.2‚Äì3.3.3
Eigenvalues ‚Äî Definition & Characteristic Equation
Definition 3.4 ‚Äî Eigenvalue & Eigenvector
For n√ón matrix A, a scalar Œª is an eigenvalue of A if:
$$A\mathbf{x} = \lambda\mathbf{x} \quad \text{for some nonzero } \mathbf{x} \in \mathbb{R}^n$$
Such a nonzero x is an eigenvector corresponding to Œª.
Equivalent Condition
$\lambda$ is an eigenvalue $\iff$ the system $(\lambda I - A)\mathbf{x} = \mathbf{0}$ has a nontrivial solution $\iff$ $(\lambda I - A)$ is not invertible $\iff$
$$\det(\lambda I - A) = 0$$
This is the characteristic equation.
Example
$A = \begin{pmatrix}3&5\\1&-1\end{pmatrix}$, $\mathbf{x} = \begin{pmatrix}5\\1\end{pmatrix}$: $A\mathbf{x} = \begin{pmatrix}20\\4\end{pmatrix} = 4\mathbf{x} \implies \lambda = 4$ is an eigenvalue.
Key Facts
  • Every $n\times n$ matrix has exactly $n$ eigenvalues over $\mathbb{C}$ (counting multiplicity)
  • $A$ and $A^T$ always share the same eigenvalues (same characteristic polynomial)
  • $\lambda = 0$ is an eigenvalue $\iff$ $A$ is not invertible
Characteristic Polynomial
Definition 3.5 ‚Äî Characteristic Polynomial
For n√ón matrix A, the characteristic polynomial is:
$$c_A(x) = \det(xI - A)$$
It has degree $n$ with leading term $x^n$. For $2\times 2$:
$$c_A(x) = x^2 - (\operatorname{tr} A)\,x + \det A$$ $$\text{where } \operatorname{tr} A = a_{11} + a_{22} \text{ (trace of }A\text{)}$$
Theorem 3.3.2 ‚Äî Finding Eigenvalues & Eigenvectors
Let A be n√ón.
  1. The eigenvalues $\lambda$ are the roots of $c_A(x) = 0$.
  2. The $\lambda$-eigenvectors are the nonzero solutions to the homogeneous system $(\lambda I - A)\mathbf{x} = \mathbf{0}$.
Example ‚Äî 2√ó2
A = [[3,5],[1,‚àí1]]:
$$c_A(x) = \det\begin{pmatrix}x-3 & -5 \\ -1 & x+1\end{pmatrix} = x^2 - 2x - 8 = (x-4)(x+2)$$
Eigenvalues: $\lambda_1 = 4$, $\lambda_2 = -2$.
Cayley-Hamilton Theorem
Every square matrix $A$ satisfies its own characteristic polynomial: $c_A(A) = \mathbf{0}$.
¬ß3.3 Eigenvectors, Eigenspaces, and Multiplicity
Definition 3.7 ¬∑ Theorems 3.3.3‚Äì3.3.5
Eigenspaces & Algebraic vs. Geometric Multiplicity
Eigenspace
The Œª-eigenspace of A is:
$$E_\lambda = \operatorname{null}(\lambda I - A) = \{\mathbf{x} : A\mathbf{x} = \lambda\mathbf{x}\}$$
It is a subspace of $\mathbb{R}^n$. Eigenvectors for $\lambda$ are all nonzero vectors in $E_\lambda$.
Definition 3.7 ‚Äî Algebraic Multiplicity
The algebraic multiplicity of eigenvalue $\lambda$ is the number of times it occurs as a root of $c_A(x)$.
Geometric Multiplicity
The geometric multiplicity of $\lambda$ is $\dim(E_\lambda)$ = number of basic eigenvectors for $\lambda$.
Always: $1 \leq$ geometric mult $\leq$ algebraic mult.
Theorem 3.3.3 ‚Äî A-Invariance (2√ó2)
Let x ‚âÝ 0, and let L_x be the line through the origin containing x. Then:
$$\mathbf{x} \text{ is an eigenvector of } A \iff L_{\mathbf{x}} \text{ is } A\text{-invariant}$$
Procedure ‚Äî Find Basic Eigenvectors
  1. Form $(\lambda I - A)$
  2. Row-reduce the augmented matrix $[(\lambda I - A) \mid \mathbf{0}]$
  3. Write the general solution; each free parameter gives one basic eigenvector
Diagonalization ‚Äî A = PDP‚Åª¬π
Definition 3.6 ‚Äî Diagonalizable
An n√ón matrix A is diagonalizable if there exists an invertible matrix P such that:
$$P^{-1}AP = D = \operatorname{diag}(\lambda_1, \lambda_2, \ldots, \lambda_n)$$ $$\text{i.e., } A = PDP^{-1}$$
P is called a diagonalizing matrix for A.
Theorem 3.3.4 ‚Äî Diagonalization Theorem
A is diagonalizable ‚ü∫ A has n eigenvectors x‚ÇÅ, ‚Ä¶, x‚Çô such that P = [x‚ÇÅ x‚ÇÇ ¬∑¬∑¬∑ x‚Çô] is invertible. When this holds:
$$P^{-1}AP = \operatorname{diag}(\lambda_1, \ldots, \lambda_n)$$
where $\lambda_i$ is the eigenvalue corresponding to column $\mathbf{x}_i$ of $P$.
Theorem 3.3.1 ‚Äî Powers via Diagonalization
If $A = PDP^{-1}$ then for each $k \geq 1$:
$$A^k = P D^k P^{-1}$$ $$\text{where } D^k = \operatorname{diag}(\lambda_1^k, \lambda_2^k, \ldots, \lambda_n^k)$$
Key Observation
The columns of P can be arranged in any order ‚Äî the eigenvalues on the diagonal of D follow the same order as the eigenvector columns of P.
¬ß3.3 Conditions for Diagonalizability
Theorems 3.3.5‚Äì3.3.6 ¬∑ Diagonalization Algorithm
Conditions for Diagonalizability
Theorem 3.3.5 ‚Äî Multiplicity Criterion
$A$ is diagonalizable $\iff$ for every eigenvalue $\lambda$ of algebraic multiplicity $m$, the system $(\lambda I - A)\mathbf{x} = \mathbf{0}$ has exactly $m$ parameters (equivalently, $\dim(E_\lambda) = m$, i.e., geometric mult = algebraic mult for each $\lambda$).
Theorem 3.3.6 ‚Äî Distinct Eigenvalues Sufficient
If an $n\times n$ matrix $A$ has $n$ distinct eigenvalues, then $A$ is diagonalizable.
Diagonalization Algorithm
  1. Find eigenvalues: solve $c_A(\lambda) = \det(\lambda I - A) = 0$.
  2. Find eigenvectors: for each $\lambda$, solve $(\lambda I - A)\mathbf{x} = \mathbf{0}$ by Gaussian elimination; collect all basic solutions.
  3. Count: if total basic eigenvectors $= n$, $A$ is diagonalizable.
  4. Form P: columns are the $n$ basic eigenvectors; $D = \operatorname{diag}$ of corresponding eigenvalues.
‚öÝ Not Diagonalizable: $A = \begin{pmatrix}1&1\\0&1\end{pmatrix}$ has $c_A(x) = (x-1)^2$. The eigenvalue $\lambda = 1$ has algebraic multiplicity 2 but only one basic eigenvector $\begin{pmatrix}1\\0\end{pmatrix}$ (geometric mult $= 1 < 2$). Not diagonalizable.
Similar Matrices
Definition ‚Äî Similar Matrices
Square matrices $A$ and $B$ are similar (written $A \sim B$) if there exists an invertible matrix $P$ such that:
$$B = P^{-1}AP$$
Properties of Similar Matrices
If A ~ B, then:
  • $A$ and $B$ have the same characteristic polynomial: $c_A(x) = c_B(x)$
  • A and B have the same eigenvalues (same roots of c_A)
  • $\det A = \det B$
  • $\operatorname{tr} A = \operatorname{tr} B$ (trace is preserved)
  • A invertible ‚ü∫ B invertible
  • A diagonalizable ‚ü∫ B diagonalizable
Relation to Diagonalizability
$A$ is diagonalizable $\iff$ $A$ is similar to a diagonal matrix $D$. The diagonalizing matrix $P$ gives the similarity transformation. Different orderings of eigenvectors give different but equivalent forms.
Caution
Same eigenvalues does NOT imply similar. E.g., $A = \begin{pmatrix}2&1\\0&2\end{pmatrix}$ and $B = \begin{pmatrix}2&0\\0&2\end{pmatrix} = 2I$ both have eigenvalue $2$ (mult $2$), but $A$ is not diagonalizable while $B = 2I$ is. They are not similar.

Linear Dynamical Systems
Definition 3.8 ‚Äî Linear Dynamical System
A sequence v‚ÇÄ, v‚ÇÅ, v‚ÇÇ, ‚Ä¶ in ‚Ñù‚Åø is a linear dynamical system if v‚ÇÄ is given and:
$$\mathbf{v}_{k+1} = A\mathbf{v}_k \quad \text{for each } k \geq 0$$
This implies $\mathbf{v}_k = A^k\mathbf{v}_0$ for all $k \geq 0$.
Theorem 3.3.7 ‚Äî Exact & Approximate Solution
Let $A$ be diagonalizable with eigenvalues $\lambda_1, \ldots, \lambda_n$ and basic eigenvectors $\mathbf{x}_1, \ldots, \mathbf{x}_n$. Let $P = [\mathbf{x}_1 \cdots \mathbf{x}_n]$ and $\mathbf{b} = P^{-1}\mathbf{v}_0 = [b_1, \ldots, b_n]^T$. Then the exact formula is:
$$\mathbf{v}_k = b_1\lambda_1^k\mathbf{x}_1 + b_2\lambda_2^k\mathbf{x}_2 + \cdots + b_n\lambda_n^k\mathbf{x}_n$$
If A has a dominant eigenvalue Œª‚ÇÅ (|Œª‚ÇÅ| > |Œª·µ¢| for all i ‚â• 2), then for large k:
$$\mathbf{v}_k \approx b_1 \lambda_1^k \mathbf{x}_1$$
