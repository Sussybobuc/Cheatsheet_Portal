LAW · CH5 · §5.1–§5.6
§5.1 Subspaces and Spanning
Subspace of $\mathbb{R}^n$ — Definition
Definition 5.1 — Subspace
A set U of vectors in $\mathbb{R}^n$ is a subspace of $\mathbb{R}^n$ if it satisfies all three conditions:
  • S1. The zero vector 0 is in U.
  • S2. If x and y are in U, then x + y is in U. (closed under addition)
  • S3. If x is in U, then ax is in U for every scalar a. (closed under scalar multiplication)
Trivial Subspaces
$\mathbb{R}^n$ itself and $\{\mathbf{0}\}$ (the zero subspace) are always subspaces. Any subspace other than $\{\mathbf{0}\}$ or $\mathbb{R}^n$ is a proper subspace.
Key Examples in R³ (§5.1)
• Every plane through the origin: M = {v | n·v = 0}; satisfies S1–S3 via dot product linearity.
• Every line through the origin: L = {td | t ∈ ℝ}; satisfies S1–S3 trivially.
Example 5.1.2 — Matrix Subspaces
For an m×n matrix A:
• null A $= \{\mathbf{x} \in \mathbb{R}^n \mid A\mathbf{x} = \mathbf{0}\}$ is a subspace of $\mathbb{R}^n$.
• im A $= \{A\mathbf{x} \mid \mathbf{x} \in \mathbb{R}^n\}$ is a subspace of $\mathbb{R}^m$ (also called col A).
Subspace Test (Exercise 5.1.20)
A nonempty set $U \subseteq \mathbb{R}^n$ is a subspace if and only if S2 and S3 hold (S1 follows automatically if U is nonempty and closed under scalar multiplication with a = 0).
Span — Definition & Smallest Subspace
Definition 5.2 — Span
Given x₁, x₂, …, xₖ ∈ $\mathbb{R}^n$, a vector t₁x₁ + ··· + tₖxₖ (tᵢ ∈ ℝ) is a linear combination. The set of all such combinations is:

$$\text{span}\{\mathbf{x}_1, \ldots, \mathbf{x}_k\} = \{t_1\mathbf{x}_1 + \cdots + t_k\mathbf{x}_k \mid t_i \in \mathbb{R}\}$$
Theorem 5.1.1 — Span is the Smallest Subspace
Let $U = \text{span}\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$ in $\mathbb{R}^n$. Then:
1. $U$ is a subspace of $\mathbb{R}^n$ containing each $\mathbf{x}_i$.
2. If $W$ is any subspace of $\mathbb{R}^n$ containing all $\mathbf{x}_i$, then $U \subseteq W$.
(span is the smallest subspace containing those vectors)
Geometric Meaning
• span{d} = $\mathbb{R}^n$ line through origin with direction d ≠ 0.
• span{v, w} = plane through origin (if v, w nonparallel in R³).
Spanning Sets for null A and im A
• Let $\mathbf{x}_1, \ldots, \mathbf{x}_k$ be the basic solutions of $A\mathbf{x} = \mathbf{0}$ (§5.1):  $\text{null}(A) = \text{span}\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$.
• Let $\mathbf{c}_1, \ldots, \mathbf{c}_n$ be the columns of $A$:  $\text{im}(A) = \text{span}\{\mathbf{c}_1, \ldots, \mathbf{c}_n\}$.
Membership Test (Example 5.1.4)
$\mathbf{p} \in \text{span}\{\mathbf{x}, \mathbf{y}\} \Leftrightarrow s\mathbf{x} + t\mathbf{y} = \mathbf{p}$ has a solution.
Equate components → linear system. Consistent ⟹ in span; inconsistent ⟹ not in span.
Example 5.1.6 — Standard Basis
$\mathbb{R}^n = \text{span}\{\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n\}$ where $\mathbf{e}_j$ = column $j$ of $I_n$.
§5.2–§5.3 Independence and Basis
Linear Independence
Definition 5.3 — Linear Independence
A set {x₁, x₂, …, xₖ} is linearly independent if:

$$t_1\mathbf{x}_1 + t_2\mathbf{x}_2 + \cdots + t_k\mathbf{x}_k = \mathbf{0} \implies t_1 = t_2 = \cdots = t_k = 0$$

i.e., the only trivial combination vanishes.
Independence Test — Procedure
  1. Set $t_1\mathbf{x}_1 + \cdots + t_k\mathbf{x}_k = \mathbf{0}$.
  2. Equate components → homogeneous linear system.
  3. Show the only solution is $t_1 = \cdots = t_k = 0$ (no free variables in RREF).
Theorem 5.2.2 — Column Independence via Matrix
For $A$ an $m \times n$ matrix with columns $\mathbf{c}_1, \ldots, \mathbf{c}_n$:
1. $\{\mathbf{c}_1, \ldots, \mathbf{c}_n\}$ is independent in $\mathbb{R}^m$ $\Leftrightarrow$ $A\mathbf{x}=\mathbf{0}$ implies $\mathbf{x}=\mathbf{0}$.
2. $\text{span}\{\mathbf{c}_1, \ldots, \mathbf{c}_n\} = \mathbb{R}^m$ $\Leftrightarrow$ $A\mathbf{x}=\mathbf{b}$ is consistent for every $\mathbf{b}$.
Theorem 5.2.3 — Square Matrix Criterion
For an $n \times n$ matrix $A$, these are equivalent:
1. $A$ is invertible.  2. Columns are independent.  3. Columns span $\mathbb{R}^n$.
4. Rows are independent.  5. Rows span all $1 \times n$ rows.
Key Facts
• $\{\mathbf{0}\}$ is always dependent ($1\cdot\mathbf{0} = \mathbf{0}$ is nontrivial).
• $\{\mathbf{x}\}$ is independent $\Leftrightarrow$ $\mathbf{x} \neq \mathbf{0}$.
• Nonzero rows of any row-echelon matrix are always independent (Example 5.2.6).
• $\det(A) \neq 0 \Leftrightarrow$ rows (or columns) of square $A$ are independent.
Basis — Definition & Key Theorems
Definition 5.4 — Basis
A set B = {x₁, …, xₘ} of vectors in a subspace U ⊆ $\mathbb{R}^n$ is a basis of U if:

1. $\{\mathbf{x}_1, \ldots, \mathbf{x}_m\}$ is linearly independent.
2. $U = \text{span}\{\mathbf{x}_1, \ldots, \mathbf{x}_m\}$.   ($B$ spans $U$)
Theorem 5.2.4 — Fundamental Theorem
If $U \subseteq \mathbb{R}^n$ is spanned by $m$ vectors and contains $k$ linearly independent vectors, then $k \leq m$.
Theorem 5.2.5 — Invariance Theorem
If $\{\mathbf{x}_1, \ldots, \mathbf{x}_m\}$ and $\{\mathbf{y}_1, \ldots, \mathbf{y}_k\}$ are both bases of $U$, then $m = k$.
(All bases of a subspace have the same number of vectors.)
Standard Basis of $\mathbb{R}^n$ (Example 5.2.10)
$\{\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n\}$ (columns of $I_n$) is a basis of $\mathbb{R}^n$.   $\dim(\mathbb{R}^n) = n$, $\dim(\{\mathbf{0}\}) = 0$.
Geometry of Proper Subspaces (Example 5.2.14)
In $\mathbb{R}^2$: $\dim(U) = 1 \Leftrightarrow U$ is a line through origin.
In $\mathbb{R}^3$: $\dim(U) = 1 \Leftrightarrow$ line;  $\dim(U) = 2 \Leftrightarrow$ plane through origin.
Unique Representation & Coordinates
Theorem 5.2.1 — Unique Representation
If $\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$ is independent, then every vector in $\text{span}\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$ has a unique representation as a linear combination of the $\mathbf{x}_i$.
Coordinate Vector
If B = {x₁, …, xₘ} is a basis of U and v ∈ U, then the coordinate vector of v relative to B is:

$$\mathbf{v} = c_1\mathbf{x}_1 + c_2\mathbf{x}_2 + \cdots + c_m\mathbf{x}_m \implies [\mathbf{v}]_B = (c_1, c_2, \ldots, c_m)$$
Theorem 5.3.6 — Expansion Theorem (Fourier Expansion)
If $\{\mathbf{f}_1, \ldots, \mathbf{f}_m\}$ is an orthogonal basis of $U \subseteq \mathbb{R}^n$, then for any $\mathbf{x} \in U$:

$$\mathbf{x} = \frac{\mathbf{x}\cdot\mathbf{f}_1}{\|\mathbf{f}_1\|^2}\mathbf{f}_1 + \frac{\mathbf{x}\cdot\mathbf{f}_2}{\|\mathbf{f}_2\|^2}\mathbf{f}_2 + \cdots + \frac{\mathbf{x}\cdot\mathbf{f}_m}{\|\mathbf{f}_m\|^2}\mathbf{f}_m$$

The coefficients $t_i = \mathbf{x}\cdot\mathbf{f}_i/\|\mathbf{f}_i\|^2$ are the Fourier coefficients. If orthonormal: $t_i = \mathbf{x}\cdot\mathbf{f}_i$.
Finding Coordinates
Given basis B = {x₁, x₂} and v ∈ U, solve:
$c_1\mathbf{x}_1 + c_2\mathbf{x}_2 = \mathbf{v}$  $\to$  the solution $(c_1, c_2) = [\mathbf{v}]_B$.
For orthogonal basis: use Expansion Theorem directly (no system to solve).
Dimension — Existence & Extension Theorems
Definition 5.5 — Dimension
If {x₁, …, xₘ} is any basis of U, the dimension of U is:
$\dim(U) = m$
Well-defined by the Invariance Theorem.   $\dim(\{\mathbf{0}\}) = 0$.
Theorem 5.2.6 — Existence of Basis
Let $U \neq \{\mathbf{0}\}$ be a subspace of $\mathbb{R}^n$. Then:
1. $U$ has a basis and $\dim(U) \leq n$.
2. Any independent set in $U$ can be extended to a basis of $U$ (by adding standard basis vectors).
3. Any spanning set for $U$ can be reduced to a basis (by deleting vectors).
Theorem 5.2.7 — Labour-Saving Result
Let $\dim(U) = m$. For a set $B$ of exactly $m$ vectors in $U$:
$B$ is independent $\Leftrightarrow$ $B$ spans $U$  (only one needs verification!)
Theorem 5.2.8 — Dimension Comparison
If $U \subseteq W$ are subspaces of $\mathbb{R}^n$:
1. $\dim(U) \leq \dim(W)$.
2. If $\dim(U) = \dim(W)$, then $U = W$.
Dimension Extremes
$\dim(U) = 0 \Leftrightarrow U = \{\mathbf{0}\}$;  $\dim(U) = n \Leftrightarrow U = \mathbb{R}^n$.
§5.4–§5.5 Fundamental Subspaces & Rank
Row Space
Definition 5.10 — Row Space
The row space of an m×n matrix A:

$\text{row}(A)$ = span of the rows of $A$  $= \text{col}(A^T) \subseteq \mathbb{R}^n$
Lemma 5.4.1 — Row Operations Preserve row A
If $A \to B$ by elementary row operations, then $\text{row}(A) = \text{row}(B)$.
(Column operations preserve $\text{col}(A)$.)
Lemma 5.4.2 + Theorem 5.4.1 — Basis for row A
If R is a row-echelon form of A (obtained by row operations):
• The nonzero rows of $R$ are independent and span $\text{row}(R) = \text{row}(A)$.
• Therefore: the $r$ nonzero rows of $R$ form a basis of $\text{row}(A)$.
• $\dim(\text{row}(A)) = \text{rank}(A) = r$.
Example 5.4.1
U = span{(1,1,2,3),(2,4,1,0),(1,5,−4,−9)} — reduce to REF:
Basis of U = {(1,1,2,3), (0,2,−3,−6)}  (nonzero rows of REF).
 $\to \dim(\text{row}(A)) = 2 = \text{rank}(A)$.
Corollary 5.4.1 — Rank Symmetry
$\text{rank}(A) = \text{rank}(A^T)$.  Row rank equals column rank for any matrix.
Column Space
Column Space (Definition 5.10)
The column space of an m×n matrix A:

$\text{col}(A) = \text{span}\{\mathbf{c}_1, \mathbf{c}_2, \ldots, \mathbf{c}_n\} = \text{im}(A) \subseteq \mathbb{R}^m$
Theorem 5.4.1 — Basis for col A
Let A → R (row-echelon form) with leading 1s in columns j₁, j₂, …, jᵣ. Then:
• Columns j₁, …, jᵣ of the original matrix A are a basis of col A.
• $\dim(\text{col}(A)) = \text{rank}(A) = r$.
(Use original $A$ columns, NOT the REF columns!)
Theorem 5.4.1 — Key Identity
For any m×n matrix A of rank r:

$$\dim(\text{col}(A)) = \dim(\text{row}(A)) = \text{rank}(A) = r$$
Corollaries 5.4.2–5.4.4
• $\text{rank}(A) \leq m$ and $\text{rank}(A) \leq n$ (bounded by both dimensions).
• $\text{rank}(UA) = \text{rank}(AV) = \text{rank}(A)$ for any invertible $U$, $V$.
• $\text{rank}(AB) \leq \text{rank}(A)$ and $\text{rank}(AB) \leq \text{rank}(B)$.
Example 5.4.2
A is 3×4 of rank 2: basis of col A = two specific columns of A (the pivot-column positions from the REF of A).
$\dim(\text{col}(A)) = \dim(\text{row}(A)) = 2$.
Null Space
Null Space (§5.1, §5.4)
The null space of an m×n matrix A:

$$\text{null}(A) = \{\mathbf{x} \in \mathbb{R}^n \mid A\mathbf{x} = \mathbf{0}\}$$
Theorem 5.4.2 (1) — Basis for null A
Let $A$ have rank $r$. The $n-r$ basic solutions of $A\mathbf{x} = \mathbf{0}$ given by the Gaussian algorithm form a basis of $\text{null}(A)$.
• $\dim(\text{null}(A)) = \text{nullity}(A) = n - r$.
• One basic solution per free variable (assign 1 to one free variable, 0 to others).
Definition 5.13 — Eigenspace (§5.5)
For an n×n matrix A and eigenvalue λ, the eigenspace:
$E_\lambda(A) = \text{null}(\lambda I - A)$ is a subspace of $\mathbb{R}^n$.
Example 5.4.3 — Finding null A and im A
For $A$ $3\times 4$ of rank $r = 2$:  $\dim(\text{null}(A)) = 4 - 2 = 2$.
Reduce $[A|\mathbf{0}]$ to RREF $\to$ identify 2 free variables $\to$ 2 basic solutions $\mathbf{x}_1, \mathbf{x}_2$.
$\text{null}(A) = \text{span}\{\mathbf{x}_1, \mathbf{x}_2\}$;  $\text{im}(A)$ = span of 2 pivot columns of $A$.
Eigenspace Dimension (§5.5 — Lemma 5.5.3)
If $\lambda$ has algebraic multiplicity $m$, then $\dim(E_\lambda(A)) \leq m$.
$A$ is diagonalizable (when $c_A(x)$ factors completely) $\Leftrightarrow$ $\dim(E_\lambda(A)) = $ multiplicity of $\lambda$ for every eigenvalue $\lambda$.
Rank-Nullity Theorem
Theorem 5.4.2 — Rank-Nullity Theorem
For an $m \times n$ matrix $A$ of rank $r$:

1.  $\dim(\text{null}(A)) = n - r$   (nullity $= n - r$)
2.  $\dim(\text{im}(A)) = \dim(\text{col}(A)) = r$

$$\text{rank}(A) + \text{nullity}(A) = n$$ ($n$ = number of columns of $A$)
Summary table for m×n matrix A of rank r
SubspaceLives inDimensionBasis from
$\text{row}(A)$$\mathbb{R}^n$$r$Nonzero rows of REF of $A$
$\text{col}(A) = \text{im}(A)$$\mathbb{R}^m$$r$Pivot columns of $A$ (not REF)
$\text{null}(A)$$\mathbb{R}^n$$n - r$Basic solutions of $A\mathbf{x} = \mathbf{0}$
Theorem 5.4.3 — Full Column Rank (rank A = n)
Equivalent: rows of $A$ span $\mathbb{R}^n$; columns of $A$ are independent in $\mathbb{R}^m$; $A^TA$ invertible; left inverse $CA = I_n$ exists; $A\mathbf{x} = \mathbf{0} \Rightarrow \mathbf{x} = \mathbf{0}$.
Theorem 5.4.4 — Full Row Rank (rank A = m)
Equivalent: columns of $A$ span $\mathbb{R}^m$; rows of $A$ are independent; $AA^T$ invertible; right inverse $AC = I_m$ exists; $A\mathbf{x} = \mathbf{b}$ consistent for every $\mathbf{b}$.
Applications
• A $3\times 6$ matrix has $\text{nullity} \geq 3$ (always has free variables).
• Rank-nullity is the key tool for counting degrees of freedom in $A\mathbf{x} = \mathbf{b}$.
§5.3 & §5.6 Orthogonality, Coordinates & Best Approximation
Orthogonal Complement
Orthogonal Complement (§5.3)
For a subspace U ⊆ $\mathbb{R}^n$, the orthogonal complement is:

$$U^\perp = \{\mathbf{v} \in \mathbb{R}^n \mid \mathbf{v}\cdot\mathbf{u} = 0 \text{ for all } \mathbf{u} \in U\}$$
Fundamental Subspace Theorem
For any m×n matrix A:

• $\mathbb{R}^n = \text{row}(A) \oplus \text{null}(A)$   (direct sum, orthogonal)
• $\mathbb{R}^m = \text{col}(A) \oplus \text{null}(A^T)$  (direct sum, orthogonal)

This means every $\mathbf{x} \in \mathbb{R}^n$ decomposes uniquely as:
$\mathbf{x} = \mathbf{x}_{\text{row}} + \mathbf{x}_{\text{null}}$,   $\mathbf{x}_{\text{row}} \in \text{row}(A)$,  $\mathbf{x}_{\text{null}} \in \text{null}(A)$.
Properties of U⊥ (§5.3)
• $(U^\perp)^\perp = U$   (double complement returns $U$).
• $\dim(U) + \dim(U^\perp) = n$.
• $U \cap U^\perp = \{\mathbf{0}\}$.
• $(\text{row}(A))^\perp = \text{null}(A)$;  $(\text{col}(A))^\perp = \text{null}(A^T)$.
Theorem 5.3.5 — Orthogonal ⟹ Independent
Every orthogonal set of nonzero vectors in $\mathbb{R}^n$ is linearly independent.
Proof: take dot product of vanishing combination with each xᵢ.
Theorem 5.3.4 — Pythagoras in $\mathbb{R}^n$
If {x₁, …, xₖ} is an orthogonal set:
$$\|\mathbf{x}_1 + \cdots + \mathbf{x}_k\|^2 = \|\mathbf{x}_1\|^2 + \cdots + \|\mathbf{x}_k\|^2$$
Projection & Best Approximation
Projection onto a Subspace
For U ⊆ $\mathbb{R}^n$ and v ∈ $\mathbb{R}^n$, projU(v) is the unique closest vector in U to v:

$\mathbf{v} = \text{proj}_U(\mathbf{v}) + \mathbf{v}^\perp$  where  $\mathbf{v}^\perp \in U^\perp$.
If {f₁, …, fₖ} is an ONB for U (Expansion Theorem) $$\text{proj}_U(\mathbf{v}) = (\mathbf{v}\cdot\mathbf{f}_1)\mathbf{f}_1 + (\mathbf{v}\cdot\mathbf{f}_2)\mathbf{f}_2 + \cdots + (\mathbf{v}\cdot\mathbf{f}_k)\mathbf{f}_k$$
Theorem 5.6.1 — Best Approximation Theorem (§5.6)
Let A be m×n and b ∈ $\mathbb{R}^m$. Consider Ax = b (possibly inconsistent). Any solution z of the normal equations:

$$(A^TA)\mathbf{z} = A^T\mathbf{b}$$ minimises $\|\mathbf{b} - A\mathbf{z}\|$ over all $\mathbf{x} \in \mathbb{R}^n$ (best approximation to a solution).
If columns of $A$ are independent: $A^TA$ is invertible and $\mathbf{z} = (A^TA)^{-1}A^T\mathbf{b}$ uniquely.
Theorem 5.6.2 — Least Squares Line (§5.6)
Given data points (x₁,y₁), …, (xₙ,yₙ) with at least two distinct xᵢ. Form:
M = [1 xᵢ] matrix (n×2),  y = (y₁,…,yₙ)ᵀ. Solve normal equations:
$(M^TM)\mathbf{z} = M^T\mathbf{y}$  $\to$  least squares line $y = z_0 + z_1 x$.
Gram-Schmidt Preview (§8.1)
Every subspace U ⊆ $\mathbb{R}^n$ has an orthogonal basis. The Gram-Schmidt algorithm constructs one from any given basis and enables explicit projection computation.
Coordinates & Coordinate Isomorphism
The Coordinate Map CB
Let B = {b₁, …, bₘ} be a basis of a subspace U ⊆ $\mathbb{R}^n$. The coordinate map:

$C_B: U \to \mathbb{R}^m$   defined by   $C_B(\mathbf{v}) = [\mathbf{v}]_B$

is an isomorphism: it preserves addition and scalar multiplication.
Properties of the Coordinate Isomorphism
• [v + w]B = [v]B + [w]B   (preserves addition).
• [cv]B = c[v]B   (preserves scalar multiplication).
• CB is a bijection: dim(U) = m iff U ≅ $\mathbb{R}^m$.
Definition 5.8 — Orthonormal Set (§5.3)
A set $\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$ is orthonormal if it is orthogonal and $\|\mathbf{x}_i\| = 1$ for each $i$.
From an orthogonal set $\{\mathbf{x}_1, \ldots, \mathbf{x}_k\}$: normalise via $\mathbf{x}_i \to \mathbf{x}_i/\|\mathbf{x}_i\|$.
Theorem 5.3.1 — Properties of Dot Product in $\mathbb{R}^n$
For x, y, z ∈ $\mathbb{R}^n$, scalar a:
#Property
1$\mathbf{x}\cdot\mathbf{y} = \mathbf{y}\cdot\mathbf{x}$
2$\mathbf{x}\cdot(\mathbf{y}+\mathbf{z}) = \mathbf{x}\cdot\mathbf{y} + \mathbf{x}\cdot\mathbf{z}$
3$(a\mathbf{x})\cdot\mathbf{y} = a(\mathbf{x}\cdot\mathbf{y})$
4$\|\mathbf{x}\|^2 = \mathbf{x}\cdot\mathbf{x} \geq 0$; $= 0$ iff $\mathbf{x} = \mathbf{0}$
Theorem 5.3.2 — Cauchy–Schwarz Inequality
$|\mathbf{x}\cdot\mathbf{y}| \leq \|\mathbf{x}\| \|\mathbf{y}\|$; equality iff one is a multiple of the other.
Basis Extension/Contraction & Diagonalization
Theorem 5.2.6 — Extension & Contraction
In any subspace U ≠ {0} of $\mathbb{R}^n$:
• Any independent set in U can be extended to a basis of U.
• Any spanning set for U can be contracted (reduced) to a basis.
• In an n-dimensional space: n independent vectors form a basis; n spanning vectors form a basis.
Definition 5.11 — Similar Matrices (§5.5)
A and B are similar (write $A \sim B$) if $B = P^{-1}AP$ for some invertible $P$.
Similarity is an equivalence relation (reflexive, symmetric, transitive).
Theorem 5.5.1 — Shared Properties of Similar Matrices
If $A \sim B$ then $A$ and $B$ have the same:
determinant, rank, trace, characteristic polynomial, eigenvalues.
Theorem 5.5.3 — Diagonalization Criterion
An n×n matrix A is diagonalizable iff $\mathbb{R}^n$ has a basis $\{\mathbf{x}_1, \ldots, \mathbf{x}_n\}$ of eigenvectors of $A$. Then $P = [\mathbf{x}_1 \cdots \mathbf{x}_n]$ gives $P^{-1}AP = \text{diag}(\lambda_1, \ldots, \lambda_n)$.
Theorem 5.5.4 — Distinct Eigenvalues ⟹ Independence
Eigenvectors corresponding to distinct eigenvalues are linearly independent.
Theorem 5.5.6 — Complete Diagonalization Criterion
When $c_A(x)$ factors completely: $A$ is diagonalizable iff $\dim(E_\lambda(A)) = $ multiplicity of $\lambda$ for every eigenvalue $\lambda$.
Theorem 5.5.7 — Symmetric Matrices Have Real Eigenvalues
If $A$ is a real symmetric matrix ($A^T = A$), then all eigenvalues of $A$ are real.
