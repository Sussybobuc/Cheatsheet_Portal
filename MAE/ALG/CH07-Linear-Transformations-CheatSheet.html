Linear Transformations — Chapter 7

Linear maps between abstract vector spaces, kernel, image, rank-nullity, matrix of a transformation, and isomorphisms.

LAW · CH7 · §7.1–§7.4
§7.1 Definition and Examples
Axioms T1–T2, basic consequences, and key families of linear maps
Linear Transformation — Definition 7.1
Definition 7.1 — Linear Transformation
A function $T: V \to W$ is a linear transformation if it satisfies two axioms for all $\mathbf{v}, \mathbf{v}_1 \in V$ and all $r \in \mathbb{R}$:
AxiomConditionName
T1$T(\mathbf{v} + \mathbf{v}_1) = T(\mathbf{v}) + T(\mathbf{v}_1)$Additivity
T2$T(r\mathbf{v}) = rT(\mathbf{v})$Homogeneity
Equivalent Single Condition
$T(a\mathbf{u} + b\mathbf{v}) = aT(\mathbf{u}) + bT(\mathbf{v})$ for all $\mathbf{u}, \mathbf{v} \in V$ and $a, b \in \mathbb{R}$ (combines T1 and T2).
Theorem 7.1.1 — Basic Consequences
For any linear transformation $T: V \to W$:
  • T(0) = 0  — T maps the zero vector to the zero vector (proof: T(0) = T(0·v) = 0·T(v) = 0)
  • T(−v) = −T(v)  — T preserves negatives
  • T(r1v1 + ··· + rkvk) = r1T(v1) + ··· + rkT(vk)  — T preserves all linear combinations
Operator
When $V = W$, $T$ is called a linear operator on $V$.
Examples of Linear Transformations
Example 7.1.1 — Three Special Operators
For any vector spaces V, W:
MapActionName
1V1V(v) = vIdentity on V
00(v) = 0 for all vZero transformation
aI(aI)(v) = avScalar operator (a ∈ ℝ)
Matrix Transformation — TA
For an $m \times n$ matrix $A$, define $T_A: \mathbb{R}^n \to \mathbb{R}^m$ by $T_A(\mathbf{x}) = A\mathbf{x}$. Every linear map $\mathbb{R}^n \to \mathbb{R}^m$ has this form (Thm 2.6.2).
Example 7.1.4 — Calculus Maps on Pn
Differentiation: D : Pn → Pn−1; D[p(x)] = p′(x)  (linear by calculus)
Integration: I : Pn → Pn+1; I[p(x)] = ∫0x p(t) dt  (linear by calculus)
Example 7.1.2 — Matrix Operations
Transposition: R : Mmn → Mnm; R(A) = AT  (linear: (A+B)T=AT+BT)
Trace: S : Mnn → ℝ; S(A) = tr A  (linear by trace properties)
Evaluation: Ea : Pn → ℝ; Ea(p) = p(a)  (linear; Example 7.1.3)
Theorem 7.1.3 — Defined by Basis Images
Given basis {b1,…,bn} of V and any vectors w1,…,wn in W, there exists a unique linear T : V → W with T(bi) = wi. The action: T(v1b1+···+vnbn) = v1w1+···+vnwn.
§7.2 Kernel and Image
The two fundamental subspaces of a linear transformation and the Dimension Theorem
Kernel (Null Space) — Definition 7.2
Definition 7.2 — Kernel
The kernel of $T: V \to W$ is:
$\ker T = \{ \mathbf{v} \in V \mid T(\mathbf{v}) = \mathbf{0} \}$
Theorem 7.2.1(1) — Subspace
ker T is always a subspace of V.
Proof: T(0)=0 ✓; if T(v)=T(v₁)=0 then T(v+v₁)=0 and T(rv)=0 ✓.
Matrix case $$\ker(T_A) = \text{null}(A) = \{\mathbf{x} \in \mathbb{R}^n \mid A\mathbf{x} = \mathbf{0}\}$$
Theorem 7.2.2 — Injectivity Criterion
$T$ is one-to-one (injective) if and only if $\ker T = \{\mathbf{0}\}$.
Proof ($\Rightarrow$): If $\mathbf{v} \in \ker T$, then $T(\mathbf{v})=\mathbf{0}=T(\mathbf{0})$, so $\mathbf{v}=\mathbf{0}$.
Proof ($\Leftarrow$): $T(\mathbf{v})=T(\mathbf{v}_1) \Rightarrow T(\mathbf{v}-\mathbf{v}_1)=\mathbf{0} \Rightarrow \mathbf{v}-\mathbf{v}_1=\mathbf{0} \Rightarrow \mathbf{v}=\mathbf{v}_1$.
Example — D : Pn → Pn−1
ker D = span{1} = {constant polynomials}; dim(ker D) = 1.
Nullity
$\text{nullity}(T) = \dim(\ker T)$. For $T_A$: $\text{nullity} = \dim(\text{null}\, A) = n - \text{rank}\, A$.
Image (Range) — Definition 7.2
Definition 7.2 — Image
The image of $T: V \to W$ is:
$\text{im}\, T = \{ T(\mathbf{v}) \mid \mathbf{v} \in V \} = T(V)$
Theorem 7.2.1(2) — Subspace
im T is always a subspace of W.
Proof: T(0)=0 ∈ im T ✓; w=T(v), w₁=T(v₁) ⟹ w+w₁=T(v+v₁) ∈ im T ✓.
Matrix case $$\text{im}(T_A) = \text{col}(A) = \text{column space of }A$$
Surjectivity Criterion
$T$ is onto (surjective) if and only if $\text{im}\, T = W$.
Theorem 7.2.3 — Matrix Transformations
For $T_A: \mathbb{R}^n \to \mathbb{R}^m$ with $A$ an $m \times n$ matrix:
  • TA is onto iff rank A = m  (column space fills ℝm)
  • TA is one-to-one iff rank A = n  (null space trivial)
Rank
$\text{rank}(T) = \dim(\text{im}\, T)$. For $T_A$: $\text{rank}(T_A) = \text{rank}(A)$.
Rank-Nullity (Dimension) Theorem — Theorem 7.2.4
Theorem 7.2.4 — Dimension Theorem
Let $T: V \to W$ be a linear transformation where $\ker T$ and $\text{im}\, T$ are both finite dimensional. Then $V$ is also finite dimensional and:
Rank-Nullity Theorem $$\dim V = \dim(\ker T) + \dim(\text{im}\, T)$$ i.e., $\dim V = \text{nullity}(T) + \text{rank}(T)$
Proof Sketch
Let {T(e1),…,T(er)} be a basis of im T (ei ∈ V) and {f1,…,fk} a basis of ker T. Then B = {e1,…,er, f1,…,fk} is a basis of V (spans V via the image decomposition; independent via the two separate independence conditions).
Theorem 7.2.5 — Extending a Kernel Basis
If {e1,…,er, er+1,…,en} is a basis of V with {er+1,…,en} a basis of ker T, then {T(e1),…,T(er)} is a basis of im T, so rank(T) = r.
Application: D : Pn → Pn−1
dim(ker D)=1; dim Pn=n+1; so dim(im D)=n=dim Pn−1. Hence D is onto.
Application: null A
For m×n matrix A of rank r: dim(null A) = n − r (from Dimension Thm applied to TA).
Key Corollary
If dim V = dim W and T : V → W is linear, then T injective ⟺ T surjective ⟺ T is an isomorphism (Theorem 7.3.3).
§7.2–§7.3 Isomorphisms
Bijective linear maps; when two vector spaces are "the same"
Isomorphism — Definition 7.4
Definition 7.4 — Isomorphism
$T: V \to W$ is an isomorphism if $T$ is linear AND bijective (one-to-one + onto).
$V$ and $W$ are isomorphic, written $V \cong W$, if such a $T$ exists.
Equivalent Conditions (Thm 7.3.5) $T$ is an isomorphism iff $\exists$ linear $S: W \to V$ with $ST = 1_V$ and $TS = 1_W$.
Such $S$ is unique, also an isomorphism, called $T^{-1}$: $$T^{-1}[T(\mathbf{v})] = \mathbf{v}$$ for all $\mathbf{v} \in V$.
Theorem 7.3.2 — Dimension Criterion
For finite-dimensional $V$ and $W$ over $\mathbb{R}$:
$V \cong W \;\Longleftrightarrow\; \dim V = \dim W$
Corollary 7.3.2
Every n-dimensional real vector space V is isomorphic to ℝⁿ via the coordinate isomorphism CB : V → ℝⁿ defined by fixing a basis B = {b1,…,bn}.
Examples
M22 ≅ P3 ≅ ℝ⁴ (all dim 4);
R : Mmn → Mnm by R(A) = AT is an isomorphism;
M22 symmetric matrices ≅ P2 (both dim 3).
Properties of Isomorphisms — Theorem 7.3.1
Theorem 7.3.1 — Isomorphism ↔ Basis Preservation
For finite-dimensional V and W, the following are equivalent for T : V → W:
  1. T is an isomorphism
  2. For every basis {e1,…,en} of V, the set {T(e1),…,T(en)} is a basis of W
  3. There exists a basis {e1,…,en} of V such that {T(e1),…,T(en)} is a basis of W
Theorem 7.3.3 — dim V = dim W Shortcut
If $\dim V = \dim W = n$, then $T: V \to W$ is an isomorphism whenever it is either one-to-one or onto (not necessary to check both).
What Isomorphisms Preserve
$T$ isomorphism $\Rightarrow$ $T$ preserves: linear independence, spanning sets, bases, dimension, zero vector, negatives, all linear combinations.
Equivalence Relation (Corollary 7.3.1)
≅ is an equivalence relation: V ≅ V; V ≅ W ⟹ W ≅ V; V ≅ U and U ≅ W ⟹ V ≅ W.
Finding T−1 via Basis
If T(ei) = fi defines an isomorphism, then T−1(fi) = ei. For T : P1 → P1 with T(1) = 1+x, T(x) = −1, the inverse carries 1+x ↦ 1, −1 ↦ x (Example 7.3.6).
§7.3 Composition of Linear Maps
Building new linear transformations by chaining existing ones
Composition — Definition 7.5 & Theorem 7.3.4
Definition 7.5 — Composite ST
Given $T: V \to W$ and $S: W \to U$, the composite $ST: V \to U$ is defined by:
$(ST)(\mathbf{v}) = S[T(\mathbf{v})]$ for all $\mathbf{v} \in V$.
Order matters: ST ≠ TS in general. Even when both composites are defined (square operators V → V), they may differ. E.g., transposition and row-swap give ST ≠ TS.
Theorem 7.3.4 — Properties
For V T→ W S→ U R→ Z:
  • ST is linear whenever S and T are linear
  • T·1V = T and 1W·T = T  (identity is neutral)
  • (RS)T = R(ST)  (associativity)
Non-commutativity
TS requires U ⊆ V to even be defined. Even for V = W = U, ST ≠ TS typically. Matrix analogy: AB ≠ BA in general.
Example 7.3.5
S : M22 → M22 swaps rows; T : M22 → M22 transposes.
ST ≠ TS (swap-then-transpose ≠ transpose-then-swap).
Composition, Inverses & Powers
Composition of Isomorphisms
If T : V → W and S : W → U are both isomorphisms, then ST : V → U is also an isomorphism with (ST)−1 = T−1S−1.
Inverse of Composite $$(ST)^{-1} = T^{-1}S^{-1}$$ (reverse order, like matrix inverses)
Powers of a Linear Operator
For T : V → V, define Tk = T∘T∘···∘T (k times). Then:
T0 = 1V; Tk+1 = T·Tk; (Tk)−1 = (T−1)k when T is invertible.
Example 7.3.8 — T³ = 1
T : ℝ³ → ℝ³ with T(x,y,z) = (z,x,y). Then T²(x,y,z) = (y,z,x) and T³(x,y,z) = (x,y,z) = 1. So T(T²) = 1 = (T²)T, giving T−1 = T².
Thm 7.3.5 — One-Sided Implies Two-Sided
For finite-dimensional V ≅ W, if ST = 1V, then automatically TS = 1W (since dim V = dim W forces injectivity ⟺ surjectivity).
§7.4 Matrix of a Linear Transformation
Encoding T : V → W as a matrix relative to chosen bases; change-of-basis similarity
Matrix of T Relative to Bases
Setup
Let $T: V \to W$ be linear; choose basis $\mathcal{B} = \{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$ for $V$ and basis $\mathcal{D} = \{\mathbf{d}_1,\ldots,\mathbf{d}_m\}$ for $W$.
Definition — Matrix MBD(T)
The matrix of $T$ relative to $\mathcal{B}$ and $\mathcal{D}$ is the $m \times n$ matrix $M^{\mathcal{B}}_{\mathcal{D}}(T)$ whose $j$-th column $= [T(\mathbf{b}_j)]_{\mathcal{D}}$ (the $\mathcal{D}$-coordinate vector of $T(\mathbf{b}_j)$).
Key Formula $$[T(\mathbf{v})]_{\mathcal{D}} = M^{\mathcal{B}}_{\mathcal{D}}(T) \cdot [\mathbf{v}]_{\mathcal{B}}$$ for all $\mathbf{v} \in V$
Special case: V = W, B = D
When $T: V \to V$ and $\mathcal{B} = \mathcal{D}$, write $M_{\mathcal{B}}(T)$. The formula becomes $[T(\mathbf{v})]_{\mathcal{B}} = M_{\mathcal{B}}(T)[\mathbf{v}]_{\mathcal{B}}$.
Standard Basis
If $\mathcal{B}$ and $\mathcal{D}$ are the standard bases of $\mathbb{R}^n$ and $\mathbb{R}^m$, then $M^{\mathcal{B}}_{\mathcal{D}}(T_A) = A$ (the original matrix $A$).
Algorithm — Computing MBD(T)
Algorithm
To find MBD(T) for T : V → W:
  1. Fix basis B = {b1,…,bn} for V and basis D = {d1,…,dm} for W
  2. For each j = 1,…,n: apply T to bj to get T(bj) ∈ W
  3. Express T(bj) = c1jd1 + c2jd2 + ··· + cmjdm as a D-linear combination
  4. Place (c1j, c2j,…,cmj)T as column j of MBD(T)
Result $$M^{\mathcal{B}}_{\mathcal{D}}(T) = \bigl[[T(\mathbf{b}_1)]_{\mathcal{D}} \;\big|\; [T(\mathbf{b}_2)]_{\mathcal{D}} \;\big|\; \cdots \;\big|\; [T(\mathbf{b}_n)]_{\mathcal{D}}\bigr]$$
Example — D : P2 → P1
B = {1, x, x²}, D = {1, x}.
D(1)=0=[0,0]D; D(x)=1=[1,0]D; D(x²)=2x=[0,2]D.
MBD(D) = [[0,1,0],[0,0,2]]
Composition Rule
If $T: V \to W$ and $S: W \to U$, then $M(ST) = M(S) \cdot M(T)$ — matrix multiplication corresponds to composition.
Change of Basis & Similarity
Setup — Two Bases for V
Let $T: V \to V$ be a linear operator. Fix two bases $\mathcal{B}$ and $\mathcal{B}'$ for $V$. Let $P = P_{\mathcal{B} \to \mathcal{B}'}$ be the transition matrix from $\mathcal{B}$ to $\mathcal{B}'$.
Change-of-Basis Formula $$M_{\mathcal{B}'}(T) = P^{-1} \cdot M_{\mathcal{B}}(T) \cdot P$$
Definition — Similar Matrices
Two matrices $A$ and $A'$ are similar if $A' = P^{-1}AP$ for some invertible $P$.
Key Theorem
Matrices $M_{\mathcal{B}}(T)$ and $M_{\mathcal{B}'}(T)$ of the same linear operator $T$ (in different bases) are always similar. Conversely, any two similar matrices represent the same linear operator in suitably chosen bases.
Diagonalization Connection
If $\mathcal{B} = \{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ is a basis of eigenvectors of $T$, then $M_{\mathcal{B}}(T) = \text{diag}(\lambda_1,\ldots,\lambda_n)$ is diagonal. This is the motivation for diagonalization studied in Ch. 3.
Invariants Under Similarity
det, trace, rank, characteristic polynomial, eigenvalues — all preserved under similarity (same for all matrix representations of T).
One-Sided Inverses & Inverse Matrix
Left Inverse (Injective T)
If T : V → W is one-to-one, there exists a linear left inverse L : W → V with L∘T = 1V.
Construction: Extend {T(e1),…,T(en)} to a basis of W; send extra basis vectors anywhere; send T(ei) ↦ ei.
Right Inverse (Surjective T)
If T : V → W is onto, there exists a linear right inverse R : W → V with T∘R = 1W.
Construction: Choose any v with T(v)=w; set R(w)=v (linearly over a basis of W).
Two-Sided Inverse (Isomorphism)
$T$ is an isomorphism iff it has a two-sided inverse $T^{-1}$. In this case:
Inverse Matrix Formula $$M_{\mathcal{B}}(T^{-1}) = \bigl[M_{\mathcal{B}}(T)\bigr]^{-1}$$ (matrix of the inverse = inverse of the matrix)
Theorem 7.3.27 — Existence Summary
$T$ injective $\Leftrightarrow$ $\exists$ left inverse $L$ with $LT = 1_V$.
$T$ surjective $\Leftrightarrow$ $\exists$ right inverse $R$ with $TR = 1_W$.
$T$ bijective $\Leftrightarrow$ $\exists$ unique two-sided inverse $T^{-1}$.
