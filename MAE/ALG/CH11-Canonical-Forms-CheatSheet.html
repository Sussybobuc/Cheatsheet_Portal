§11.1 Invariant Subspaces
T-invariant subspaces, restriction operators, and block triangular form
Invariant Subspace
Definition — T-Invariant Subspace
A subspace $U \subseteq V$ is T-invariant if $T(U) \subseteq U$: applying $T$ to any vector in $U$ produces another vector in $U$.
Standard T-Invariant Subspaces
  • {0} — the zero subspace is always T-invariant
  • V — the whole space is always T-invariant
  • $\ker(T)$ — if $T(u) = \mathbf{0}$ then $T(u) \in \ker(T)$
  • $\text{im}(T)$ — if $v \in \text{im}(T)$ then $T(v) \in \text{im}(T)$
  • Eigenspace $E_\lambda$ — $T(u) = \lambda u \in E_\lambda$ for all $u \in E_\lambda$
Restriction T|U
If $U$ is $T$-invariant, the restriction $T|_U : U \to U$ is a well-defined linear operator on $U$.
If B is a basis of U extended to a basis B′ of V, then MB′(T) is block triangular.
Block Diagonal (direct sum)
If $V = U_1 \oplus U_2$ with both $T$-invariant, then for bases $B_i$ of $U_i$:
$M_B(T) = \text{diag}(M_{B_1}(T|_{U_1}),\, M_{B_2}(T|_{U_2}))$
Block Triangular Form  Core Theorem
Theorem 11.1.1 — Block Triangulation
Let $A$ be $n\times n$ with real eigenvalues $\lambda_1,\ldots,\lambda_k$ and char poly $c_A(x) = (x-\lambda_1)^{m_1}\cdots(x-\lambda_k)^{m_k}$.

Then an invertible $P$ exists such that $$P^{-1}AP = \text{diag}(U_1,\ldots,U_k)$$ where each $U_i$ is an $m_i\times m_i$ upper triangular matrix with every diagonal entry equal to $\lambda_i$.
Schur's Theorem (complex version)
Every complex square matrix is unitarily similar to an upper triangular matrix. Every complex matrix can be triangulated with eigenvalues on the diagonal.
Generalized Eigenspace (Definition 11.1)
$G_{\lambda_i}(A) = \ker[(\lambda_i I - A)^{m_i}]$
• $\dim G_{\lambda_i}(A) = m_i$ (Lemma 11.1.1)
• $E_{\lambda_i} \subseteq G_{\lambda_i}(A)$
• $G_{\lambda_i}(A)$ is $T$-invariant (since $(\lambda I - A)^m$ commutes with $A$)
Ascending Chain
Eλ ⊆ null[(λI−A)²] ⊆ ··· ⊆ Gλ(A)
Climbing this chain builds a basis for $G_\lambda(A)$; bases of $G_{\lambda_i}$ union to a basis of $\mathbb{R}^n$.
§11.2 The Jordan Canonical Form
Jordan blocks, nilpotent operators, and the complete Jordan canonical form
Jordan Block (Definition 11.2)
Definition — Jordan Block Jk(λ)
$J_k(\lambda)$ is the $k\times k$ matrix with $\lambda$ on the main diagonal, 1s on the superdiagonal, and 0s elsewhere.
J₁(λ) = [λ] J₂(λ) = ⎡λ 1⎤ J₃(λ) = ⎡λ 1 0⎤ ⎣0 λ⎦ ⎢0 λ 1⎥ ⎣0 0 λ⎦
Key Properties of Jk(λ)
  • Eigenvalue: $\lambda$ is the only eigenvalue; alg mult $= k$, geom mult $= 1$
  • Char poly: $c_{J_k(\lambda)}(x) = (x-\lambda)^k$
  • Nilpotent shift: $(J_k(\lambda)-\lambda I)^k = 0$ but $(J_k(\lambda)-\lambda I)^{k-1} \neq 0$
  • Decomposition: $J_k(\lambda) = \lambda I_k + J_k(0)$, nilpotent part $J_k(0)$
Nilpotent Shift Example
For $J_3(0)$: $J_3(0)^2$ shifts twice along superdiagonal; $J_3(0)^3 = 0$.
$\ker(J_3(0)) \subset \ker(J_3(0)^2) \subset \ker(J_3(0)^3) = \mathbb{R}^3$ with dims $1 < 2 < 3$.
Jordan Canonical Form  Key Theorem
Theorem 11.2.3 — Jordan Canonical Form
Every complex $n\times n$ matrix $A$ is similar to a Jordan matrix:
$$J = \text{diag}(J_{k_1}(\lambda_1),\, J_{k_2}(\lambda_2),\, \ldots,\, J_{k_m}(\lambda_m))$$
Except for the order of Jordan blocks, J is uniquely determined by A.
Similarity Criterion
$A \sim B$ ($A$ and $B$ are similar) if and only if they have the same Jordan canonical form — same blocks, same sizes, same eigenvalues up to reordering.
Why ℂ is Required
Over ℝ, char poly may not split into linear factors (e.g., x²+1 has no real roots).
Over ℂ, every polynomial splits completely → JCF exists for every matrix.
Diagonalizable iff All Blocks are 1×1
$A$ is diagonalizable $\Leftrightarrow$ every Jordan block in its JCF is $J_1(\lambda) = [\lambda]$ $\Leftrightarrow$ geometric mult $=$ algebraic mult for every eigenvalue.
Jordan Form and Multiplicities
Algebraic Multiplicity
$\text{alg mult}(\lambda) = $ power of $(x-\lambda)$ in $c_A(x)$
$=$ total size of all Jordan blocks for $\lambda$
$=\dim G_\lambda(A)$
Geometric Multiplicity
$\text{geom mult}(\lambda) = \dim E_\lambda(A) = \dim\ker(\lambda I - A)$
$=$ number of Jordan blocks for $\lambda$
$\text{geom mult}(\lambda) \leq \text{alg mult}(\lambda)$ always
Jordan Structure Summary
For eigenvalue λ in the JCF of A:
• # Jordan blocks for λ = geometric multiplicity
• Sum of block sizes for λ = algebraic multiplicity
• Single block of size m ⟺ geom mult = 1
ConditionImplication
geom = alg for all λA is diagonalizable
geom < alg for some λBlocks of size ≥ 2 exist
All eigenvalues distinctAlways diagonalizable
Generalized Eigenvectors
Generalized Eigenvector of Rank k
$v$ is a generalized eigenvector of rank $k$ for $\lambda$ if:
$$( A-\lambda I)^k v = \mathbf{0} \quad\text{but}\quad (A-\lambda I)^{k-1} v \neq \mathbf{0}$$
Generalized Eigenspace Kλ
$K_\lambda = \ker((A-\lambda I)^n) = \ker((A-\lambda I)^{\text{alg mult}})$
$= G_\lambda(A)$ as defined in §11.1
$\dim(K_\lambda) = $ algebraic multiplicity of $\lambda$
Primary Decomposition (Lemma 11.1.3)
$$V = K_{\lambda_1} \oplus K_{\lambda_2} \oplus \cdots \oplus K_{\lambda_r}$$ The generalized eigenspaces partition $V$ into a direct sum.
Jordan Chain
A Jordan chain for $\lambda$ starting at $v$ (rank $m$): $\{v,\,(A-\lambda I)v,\,(A-\lambda I)^2 v,\,\ldots,\,(A-\lambda I)^{m-1}v\}$.
This independent set forms the basis for one Jordan block $J_m(\lambda)$.
Independence Claim (Lemma 11.2.1 proof)
If $T^{m-1}u \neq \mathbf{0}$ ($T$ nilpotent), then $\{u,\,Tu,\,T^2 u,\,\ldots,\,T^{m-1}u\}$ is linearly independent.
Proof: apply Tm−1 to a vanishing combination; coefficients vanish one by one.
§11.3 Cayley-Hamilton and Minimal Polynomial
Polynomial annihilators, minimal polynomial, and algorithmic computation of JCF
Cayley-Hamilton Theorem  Key Theorem
Theorem 11.1.2 — Cayley-Hamilton
Every square matrix satisfies its own characteristic polynomial:
$$c_A(A) = \mathbf{0}$$ (the zero matrix)
Proof Sketch (via Block Triangulation)
Write $c_A(x) = \prod_i(x-\lambda_i)^{m_i}$ and use $P^{-1}AP = \text{diag}(U_1,\ldots,U_k)$:
$c_A(U_i) = \prod_j(U_i-\lambda_j I)^{m_j} = 0$ for each $i$
(the factor $(U_i-\lambda_i I)^{m_i} = 0$ since $U_i$ is upper triangular with $\lambda_i$ on diagonal)
$\therefore\; P^{-1}c_A(A)P = \text{diag}[c_A(U_1),\ldots,c_A(U_k)] = \mathbf{0}$
Example 11.1.3
$A = \begin{pmatrix}1&3\\-1&2\end{pmatrix}$, $c_A(x) = x^2 - 3x + 5$.
$c_A(A) = A^2 - 3A + 5I = \begin{pmatrix}-2&9\\-3&1\end{pmatrix} - \begin{pmatrix}3&9\\-3&6\end{pmatrix} + \begin{pmatrix}5&0\\0&5\end{pmatrix} = \mathbf{0}$ ✓
Applications
  • Inverse via polynomial: $A^{-1} = r_0 I + r_1 A + \cdots + r_{n-1}A^{n-1}$ when $\det(A) \neq 0$
  • Power reduction: every $A^k$ ($k \geq n$) reduces to a poly of degree $\leq n-1$ in $A$
  • Annihilator: $c_A(x)$ annihilates $A$; minimal poly $m_A(x)$ divides $c_A(x)$
Minimal Polynomial
Definition — Minimal Polynomial mA(x)
$m_A(x)$ is the monic polynomial of smallest degree such that $m_A(A) = \mathbf{0}$.
Key Properties
  • Divisibility: $m_A(x)$ divides $c_A(x)$
  • Same roots: $m_A$ and $c_A$ have the same roots (same eigenvalues)
  • Degree: $\deg(m_A)$ divides $\deg(c_A) = n$
  • Annihilator: $p(A) = \mathbf{0}$ $\Leftrightarrow$ $m_A(x) \mid p(x)$
Minimal Polynomial & Jordan Form
For eigenvalue $\lambda_i$, let $s_i = $ size of the largest Jordan block for $\lambda_i$.
Then: $$m_A(x) = (x-\lambda_1)^{s_1}(x-\lambda_2)^{s_2}\cdots(x-\lambda_k)^{s_k}$$
Diagonalizability Criterion
$A$ is diagonalizable $\Leftrightarrow$ $m_A(x)$ has no repeated roots
(i.e., $m_A(x) = (x-\lambda_1)(x-\lambda_2)\cdots(x-\lambda_k)$ with distinct $\lambda_i$)
Example
If $\text{JCF} = \text{diag}(J_3(2),\,J_2(2),\,J_1(5))$:
$c_A(x) = (x-2)^5(x-5)$,  $m_A(x) = (x-2)^3(x-5)$
(largest block for $\lambda=2$ has size 3; only one block for $\lambda=5$)
Algorithm: Computing the Jordan Canonical Form  Step-by-Step
  1. Find eigenvalues: compute $c_A(\lambda) = \det(\lambda I - A)$ and find all roots (algebraic multiplicities $m_i$).
  2. For each eigenvalue $\lambda$: compute the null spaces of $(A-\lambda I)^k$ for $k = 1,2,3,\ldots$ Track the dimensions: $d_k = \dim\ker(A-\lambda I)^k$.
  3. Count blocks by size:
    $$\#\text{ blocks of size } \geq k = d_k - d_{k-1}$$ $$\#\text{ blocks of exact size } k = (d_k - d_{k-1}) - (d_{k+1} - d_k)$$
  4. Total Jordan blocks for $\lambda$ $= d_1 = \dim\ker(A-\lambda I) = $ geometric multiplicity.
  5. Build $P$: find a generalized eigenvector chain for each block; assemble $P$ from these chains. Then $$A = PJP^{-1}.$$
Block Size Count — Example
Suppose $\lambda=2$ with alg mult 5; compute:
$\dim\ker(A-2I) = 2$,  $\dim\ker(A-2I)^2 = 4$,  $\dim\ker(A-2I)^3 = 5$

# blocks size $\geq 1$: $2-0 = 2$
# blocks size $\geq 2$: $4-2 = 2$
# blocks size $\geq 3$: $5-4 = 1$

$\therefore$ blocks of size 1: $2-2 = 0$; size 2: $2-1 = 1$; size 3: $1$
JCF for $\lambda=2$: $\text{diag}(J_3(2),\,J_2(2))$ ✓
Rank–Nullity Shortcut
$\dim\ker(A-\lambda I)^k = n - \text{rank}(A-\lambda I)^k$
So track ranks: $r_k = \text{rank}(A-\lambda I)^k$
# blocks of size $\geq k$: $r_{k-1} - r_k$
Nilpotent Matrices
Definition — Nilpotent
$N$ is nilpotent if $N^k = \mathbf{0}$ for some $k \geq 1$.
Equivalent Characterizations
  • All eigenvalues of N are 0
  • $c_N(x) = x^n$ (only root is $0$)
  • $N^n = \mathbf{0}$ always (where $n = $ size of matrix)
  • JCF consists entirely of Jordan blocks $J_{k_i}(0)$
Lemma 11.2.1 — Nilpotent JCF
If $T : V \to V$ is nilpotent, $V$ has a basis $B$ such that $$M_B(T) = \text{diag}(J_{n_1}(0),\, J_{n_2}(0),\, \ldots,\, J_{n_k}(0))$$
Strictly Upper Triangular ⟹ Nilpotent
Any strictly upper triangular matrix (zeros on and below diagonal) is nilpotent.
More generally, A is nilpotent iff it is similar to a strictly upper triangular matrix.
Example — N³ = 0
$N = J_3(0)$: $\ker(N) \subset \ker(N^2) \subset \ker(N^3) = \mathbb{R}^3$; dims $1 < 2 < 3$.
Applications of Jordan Form
Matrix Exponential
$e^A = I + A + A^2/2! + \cdots = Pe^J P^{-1}$

For a Jordan block:
e^{Jₖ(λ)} = eλ ⎡1 1 1/2! ··· 1/(k-1)!⎤ ⎢0 1 1 ··· 1/(k-2)!⎥ ⎢ ⋱ ⎥ ⎣0 0 0 ··· 1 ⎦
Systems of ODEs
$y' = Ay$; solution: $y(t) = e^{At} y(0) = Pe^{Jt}P^{-1}y(0)$
The Jordan form reveals the growth rates ($\text{Re}\,\lambda_i$) and polynomial-exponential terms.
Computing f(A) via Jordan Form
For any function $f$ with a convergent power series:
$f(A) = P f(J) P^{-1}$
where $f(J) = \text{diag}(f(J_{k_1}(\lambda_1)),\,\ldots,\,f(J_{k_m}(\lambda_m)))$

For a Jordan block: $f(J_k(\lambda))$ has $f^{(j)}(\lambda)/j!$ in position $(i,\,i+j)$.
Recurrence via Cayley-Hamilton
Since $c_A(A) = \mathbf{0}$, we get $A^n = -(a_{n-1}A^{n-1} + \cdots + a_0 I)$.
Any sequence defined by $x_{k+n} = -a_0 x_k - \cdots - a_{n-1}x_{k+n-1}$
has closed form expressible via eigenvalues of $A$.
