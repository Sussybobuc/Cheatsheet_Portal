§9.1 Coordinate Vectors
Coordinate Vector
Definition 9.1
Given an ordered basis $\mathcal{B} = \{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$ for $V$, every $v \in V$ has a unique representation $v = c_1\mathbf{b}_1 + \cdots + c_n\mathbf{b}_n$. The coordinate vector of $v$ with respect to $\mathcal{B}$ is:
Coordinate Vector (Definition 9.1) $$[\mathbf{v}]_{\mathcal{B}} = C_{\mathcal{B}}(\mathbf{v}) = [c_1, c_2, \ldots, c_n]^T \in \mathbb{R}^n$$
Theorem 9.1.1 — CB is an Isomorphism
If $\dim V = n$ and $\mathcal{B}$ is any ordered basis, then $C_{\mathcal{B}} : V \to \mathbb{R}^n$ is an isomorphism. Its inverse: $C_{\mathcal{B}}^{-1}([c_1,\ldots,c_n]^T) = c_1\mathbf{b}_1 + \cdots + c_n\mathbf{b}_n$.
Key Facts
  • $C_{\mathcal{B}}(\mathbf{b}_j) = \mathbf{e}_j$ (column $j$ of $I_n$)
  • Coordinates are unique; a basis is a minimal spanning set
  • In $\mathbb{R}^n$: solve $[\mathbf{b}_1 | \cdots | \mathbf{b}_n]\mathbf{x} = \mathbf{v}$ by row reduction to get $[\mathbf{v}]_{\mathcal{B}}$
Example 9.1.1
$\mathcal{B} = \{(1,1,0),(1,0,1),(0,1,1)\}$, $\mathbf{v} = (2,1,3)$:
$(2,1,3) = 0\cdot(1,1,0) + 2\cdot(1,0,1) + 1\cdot(0,1,1)$
$\therefore C_{\mathcal{B}}(\mathbf{v}) = [0, 2, 1]^T$
Properties of Coordinates
Linearity of the Coordinate Map
Since $C_{\mathcal{B}} : V \to \mathbb{R}^n$ is a linear isomorphism, for all $u, v \in V$ and scalar $c$:
Additivity & Homogeneity $$[\mathbf{u} + \mathbf{v}]_{\mathcal{B}} = [\mathbf{u}]_{\mathcal{B}} + [\mathbf{v}]_{\mathcal{B}}$$ $$[c\mathbf{u}]_{\mathcal{B}} = c[\mathbf{u}]_{\mathcal{B}}$$
Coordinate Map $C_{\mathcal{B}}$ Preserves All Vector-Space Structure
Span, independence, dimension, and every linear combination are preserved. $V$ and $\mathbb{R}^n$ are indistinguishable as abstract vector spaces.
Why Order Matters
  • $\{\mathbf{b}_1,\mathbf{b}_2\}$ and $\{\mathbf{b}_2,\mathbf{b}_1\}$ give different coordinate vectors for the same $v$
  • $C_{\mathcal{B}}(\mathbf{b}_i) = \mathbf{e}_i$: swapping basis order permutes coordinate rows
  • All linear operations (addition, scalar mult, rank) correspond exactly in $\mathbb{R}^n$
Example — Non-Standard Basis in ℝ²
$\mathcal{B} = \{[1,1]^T, [1,-1]^T\}$, $\mathbf{v} = [3,1]^T$:
$c_1[1,1] + c_2[1,-1] = [3,1]$ $\Rightarrow$ $c_1 = 2$, $c_2 = 1$
$\therefore [\mathbf{v}]_{\mathcal{B}} = [2, 1]^T$
§9.2 Change of Basis Matrix
Transition Matrix PB′→B
Definition 9.4 — Change Matrix
For ordered bases $\mathcal{B}$ and $\mathcal{B}'$ of vector space $V$, the transition matrix from $\mathcal{B}'$ to $\mathcal{B}$ is:
Transition Matrix (Theorem 9.2.2 eq. 9.1) $$P_{\mathcal{B}'\to\mathcal{B}} = \bigl[[\mathbf{b}'_1]_{\mathcal{B}} \mid [\mathbf{b}'_2]_{\mathcal{B}} \mid \cdots \mid [\mathbf{b}'_n]_{\mathcal{B}}\bigr]$$
Key Formula — Coordinate Conversion
$P_{\mathcal{B}'\to\mathcal{B}}$ converts $\mathcal{B}'$-coordinates to $\mathcal{B}$-coordinates:

$[\mathbf{v}]_{\mathcal{B}} = P_{\mathcal{B}'\to\mathcal{B}} \cdot [\mathbf{v}]_{\mathcal{B}'}$ for all $v \in V$
When B = Standard Basis E (in ℝⁿ)
  • $P_{\mathcal{B}'\to E} = [\mathbf{b}'_1 | \cdots | \mathbf{b}'_n]$ (stack $\mathcal{B}'$ vectors as columns)
  • $[\mathbf{v}]_E = \mathbf{v}$ (standard coordinates are the vector itself)
Example 9.2.1 — P₂ Basis Change
$\mathcal{B} = \{1, x, x^2\}$, $\mathcal{B}' = \{1, (1-x), (1-x)^2\}$:
$x = 1-(1-x)$, $x^2 = 1-2(1-x)+(1-x)^2$
$P_{\mathcal{B}'\to\mathcal{B}} = [[1,1,1],[0,-1,-2],[0,0,1]]$
$C_{\mathcal{B}}(p) = [a,b,c]^T$ $\Rightarrow$ $C_{\mathcal{B}'}(p) = P_{\mathcal{B}'\to\mathcal{B}}[a,b,c]^T$
Computing Transition Matrices
General Method
Express each $\mathcal{B}'$ vector $\mathbf{b}'_j$ as a linear combination of $\mathcal{B}$ vectors. The coefficient columns form $P_{\mathcal{B}'\to\mathcal{B}}$.
Row-Reduction Method ($\mathbb{R}^n$) $$[\,P_{\mathcal{B}} \mid P_{\mathcal{B}'}\,] \xrightarrow{\text{row ops}} [\,I \mid P_{\mathcal{B}\to\mathcal{B}'}\,]$$ where $P_{\mathcal{B}} = [\mathbf{b}_1|\cdots|\mathbf{b}_n]$, $P_{\mathcal{B}'} = [\mathbf{b}'_1|\cdots|\mathbf{b}'_n]$
Theorem 9.2.2 — Properties of Change Matrices
  1. $P_{\mathcal{B}\to\mathcal{B}} = I_n$
  2. $P_{\mathcal{B}'\to\mathcal{B}}$ is invertible: $(P_{\mathcal{B}'\to\mathcal{B}})^{-1} = P_{\mathcal{B}\to\mathcal{B}'}$
  3. Chain rule: $P_{E\leftarrow D} \cdot P_{D\leftarrow\mathcal{B}} = P_{E\leftarrow\mathcal{B}}$
Quick Reference
  • $P_{\mathcal{B}\to\mathcal{B}'} = (P_{\mathcal{B}'\to\mathcal{B}})^{-1}$
  • If $\mathcal{B}' = E$ (standard): $P_{\mathcal{B}\to E} = [\mathbf{b}_1|\cdots|\mathbf{b}_n]$; $P_{E\to\mathcal{B}} = ([\mathbf{b}_1|\cdots|\mathbf{b}_n])^{-1}$
  • If $\mathcal{B}$ is orthonormal in $\mathbb{R}^n$: $P_{\mathcal{B}\to E}$ is orthogonal $\Rightarrow$ $P_{E\to\mathcal{B}} = P_{\mathcal{B}\to E}^T$
Chaining Basis Changes & Transition Matrix Algebra
Chaining Multiple Basis Transitions
For bases B, B′, B″ of V:

PB→B″ = PB′→B″ · PB→B′

Each arrow in a chain of basis changes corresponds to left-multiplying by a transition matrix. The notation PD←B reads right-to-left: "from B, to D."
Three-Basis Chain $$[\mathbf{v}]_{\mathcal{B}''} = P_{\mathcal{B}'\to\mathcal{B}''} \cdot P_{\mathcal{B}\to\mathcal{B}'} \cdot [\mathbf{v}]_{\mathcal{B}}$$
Standard Basis as Pivot
In $\mathbb{R}^n$, route all changes through $E$:
$P_{\mathcal{B}\to\mathcal{B}'} = (P_{\mathcal{B}'\to E})^{-1} \cdot P_{\mathcal{B}\to E}$

Example — ℝ³ Two-Step Transition
$\mathcal{B} \to E \to \mathcal{B}'$:
$P_{\mathcal{B}\to E} = [\mathbf{b}_1|\mathbf{b}_2|\mathbf{b}_3]$, $P_{\mathcal{B}'\to E} = [\mathbf{b}'_1|\mathbf{b}'_2|\mathbf{b}'_3]$
$P_{\mathcal{B}\to\mathcal{B}'} = (P_{\mathcal{B}'\to E})^{-1} \cdot P_{\mathcal{B}\to E}$
§9.2–§9.3 Matrix of a Linear Transformation & Similarity
Matrix of T Relative to a Basis MB(T)
Definition 9.3 — B-Matrix of an Operator
For a linear operator $T : V \to V$ and ordered basis $\mathcal{B} = \{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$, the $\mathcal{B}$-matrix of $T$ is:
B-Matrix (Definition 9.3 / Theorem 9.2.1) $$M_{\mathcal{B}}(T) = \bigl[[T(\mathbf{b}_1)]_{\mathcal{B}} \mid [T(\mathbf{b}_2)]_{\mathcal{B}} \mid \cdots \mid [T(\mathbf{b}_n)]_{\mathcal{B}}\bigr]$$
Theorem 9.2.1 — Defining Property
$M_{\mathcal{B}}(T)$ is the unique matrix satisfying:

$[T(\mathbf{v})]_{\mathcal{B}} = M_{\mathcal{B}}(T) \cdot [\mathbf{v}]_{\mathcal{B}}$ for all $v \in V$
Further Properties
  • Composition: $M_{\mathcal{B}}(ST) = M_{\mathcal{B}}(S) \cdot M_{\mathcal{B}}(T)$
  • $T$ invertible $\Leftrightarrow$ $M_{\mathcal{B}}(T)$ invertible; then $M_{\mathcal{B}}(T^{-1}) = [M_{\mathcal{B}}(T)]^{-1}$
  • Standard matrix of $T_A : \mathbb{R}^n\to\mathbb{R}^n$ w.r.t. standard basis $E$ is $A$ (Example 9.1.4)
How MB(T) Changes Under a New Basis
Theorem 9.2.3 — Similarity of Operator Matrices
Let $\mathcal{B}$ and $\mathcal{B}'$ be ordered bases of $V$, $P = P_{\mathcal{B}\to\mathcal{B}'}$ (transition matrix from $\mathcal{B}$ to $\mathcal{B}'$). Then:

$M_{\mathcal{B}'}(T) = P^{-1} M_{\mathcal{B}}(T) P$

$M_{\mathcal{B}}(T)$ and $M_{\mathcal{B}'}(T)$ are similar matrices — same operator, different coordinate systems.
Commutative Diagram (Coordinate Level) $$M_{\mathcal{B}'}(T) = P^{-1} M_{\mathcal{B}}(T) P$$ $[\mathbf{v}]_{\mathcal{B}}$ ——[$\times M_{\mathcal{B}}(T)$]——→ $[T(\mathbf{v})]_{\mathcal{B}}$
  ↓ $\times P$                       ↓ $\times P$
$[\mathbf{v}]_{\mathcal{B}'}$ ——[$\times M_{\mathcal{B}'}(T)$]——→ $[T(\mathbf{v})]_{\mathcal{B}'}$
Derivation Sketch
$[\mathbf{v}]_{\mathcal{B}'} = P\cdot[\mathbf{v}]_{\mathcal{B}}$. Using $M_{\mathcal{B}}(T)$ on $\mathcal{B}$-side and $M_{\mathcal{B}'}(T)$ on $\mathcal{B}'$-side, the commuting diagram gives $P\cdot M_{\mathcal{B}}(T) = M_{\mathcal{B}'}(T)\cdot P$, hence the result.
Diagonalization as Change of Basis
Theorem 9.2.4 — Realizing Similarity via Bases
Let $A$ be an $n\times n$ matrix, $E$ the standard basis, $P$ invertible with columns $\mathbf{b}_1,\ldots,\mathbf{b}_n$, and $\mathcal{B} = \{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$. Then:

$M_{\mathcal{B}}(T_A) = P^{-1}AP$
Diagonalization — Geometric View
Choose $\mathcal{B}$ = eigenvectors $\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ of $A$:

$T_A(\mathbf{v}_j) = \lambda_j\mathbf{v}_j$ $\Rightarrow$ $M_{\mathcal{B}}(T_A) = \operatorname{diag}(\lambda_1,\ldots,\lambda_n) = D$

With $P = [\mathbf{v}_1 | \cdots | \mathbf{v}_n]$:   $P^{-1}AP = D$

$A$ is diagonalizable $\Leftrightarrow$ $\mathbb{R}^n$ has a basis of eigenvectors of $A$.
Example 9.2.3
$A = \begin{bmatrix}10 & -18\\6 & -11\end{bmatrix}$, $P = \begin{bmatrix}2 & -1\\-3 & 2\end{bmatrix}$, $D = \operatorname{diag}(1,-2)$:
$P^{-1}AP = D$ ✓.   $\mathcal{B} = \{[2,-3]^T,[-1,2]^T\}$ (eigenvectors),
$\therefore M_{\mathcal{B}}(T_A) = D$.
Similarity Revisited
Geometric Meaning
$A$ and $B$ are similar $\Leftrightarrow$ they represent the same linear transformation in different bases (Theorems 9.2.3–9.2.4).
Similarity Invariants (Thm. 5.5.1 / §9.2)
InvariantIntrinsic Property of T

Similarity is an Equivalence Relation
  • Reflexive: $A = I^{-1}AI$
  • Symmetric: $B = P^{-1}AP$ $\Rightarrow$ $A = PBP^{-1}$
  • Transitive: $A\sim B$ and $B\sim C$ $\Rightarrow$ $A\sim C$
Orthonormal Change of Basis & Rotation Matrices
Orthonormal COB → Orthogonal Transition Matrix
If $\mathcal{B}$ and $\mathcal{B}'$ are both orthonormal bases of $\mathbb{R}^n$, then $P_{\mathcal{B}\to\mathcal{B}'}$ is an orthogonal matrix ($P^{-1} = P^T$), simplifying all computations.
Orthonormal Similarity Formula $$M_{\mathcal{B}'}(T) = P^T M_{\mathcal{B}}(T) P$$ (since $P^{-1} = P^T$ when columns are orthonormal)
Applications
  • Rotation/reflection matrices arise as orthonormal COB matrices
  • Gram-Schmidt produces orthonormal $\mathcal{B}$ $\Rightarrow$ orthogonal $P$
  • Spectral Theorem: sym. $A = PDP^T$ with orthogonal $P$
Example 9.2.7 — Rotation About a Line in ℝ³
Rotate about axis $\mathbf{d} = \tfrac{1}{3}[2,1,2]^T$ through angle $\theta$:
1. Find orthonormal $\mathcal{B}_0 = \{\mathbf{d}, \mathbf{f}, \mathbf{g}\}$ ($\mathbf{f}, \mathbf{g} \perp \mathbf{d}$, $\perp$ each other).
2. In $\mathcal{B}_0$: $M_{\mathcal{B}_0}(R) = \operatorname{diag}(1, R_\theta)$ where $R_\theta = \begin{bmatrix}\cos\theta & -\sin\theta\\\sin\theta & \cos\theta\end{bmatrix}$.
3. $P = [\mathbf{d}|\mathbf{f}|\mathbf{g}]$ is orthogonal $\Rightarrow$ $P^{-1} = P^T$.
4. $M_E(R) = P^T M_{\mathcal{B}_0}(R) P$ (standard matrix of $R$).

Geometric insight: working in a natural basis first, then converting to standard coordinates via orthogonal P.
