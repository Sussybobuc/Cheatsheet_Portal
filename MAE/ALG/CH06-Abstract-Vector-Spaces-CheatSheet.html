§6.1 Vector Space Axioms
The 10 Axioms — Definition 6.1
Definition 6.1 — Vector Space
A vector space consists of a nonempty set V with vector addition ($\mathbf{u}+\mathbf{v}$) and scalar multiplication ($a\mathbf{v}$, $a\in\mathbb{R}$) obeying ten axioms. Axioms A1 and S1 express closure.
LabelAxiomName
A1u + v ∈ VClosure under addition
A2u + v = v + uCommutativity
A3u + (v + w) = (u + v) + wAssociativity of addition
A4∃ 0 ∈ V : v + 0 = v for all vAdditive identity (zero vector)
A5∀ v ∈ V, ∃ −v ∈ V : v + (−v) = 0Additive inverse (negative)
S1av ∈ V for all a ∈ ℝ, v ∈ VClosure under scalar mult.
S2a(v + w) = av + awDistributivity over vector sum
S3(a + b)v = av + bvDistributivity over scalar sum
S4a(bv) = (ab)vAssociativity of scalar mult.
S51v = vMultiplicative identity
Terminology
The element $\mathbf{0}$ in A4 is the zero vector; the element $-\mathbf{v}$ in A5 is the negative of v. Subtraction is defined by $\mathbf{u} - \mathbf{v} = \mathbf{u} + (-\mathbf{v})$. All 10 axioms must hold — changing even one operation can violate the structure (see Example 6.1.4).
Consequences of Axioms — Theorems 6.1.1–6.1.3
Theorem 6.1.1 — Cancellation Law
In any vector space V: if $\mathbf{v} + \mathbf{u} = \mathbf{v} + \mathbf{w}$, then $\mathbf{u} = \mathbf{w}$. (Add $-\mathbf{v}$ to both sides using A5 and A3.)
Theorem 6.1.2 — Unique Solution
The equation $x + \mathbf{v} = \mathbf{u}$ has exactly one solution: $x = \mathbf{u} - \mathbf{v}$.
Theorem 6.1.3 — Scalar Multiplication Properties
For any $\mathbf{v} \in V$ and $a \in \mathbb{R}$:
  • 0·v = 0 — zero scalar kills any vector (proof: 0v+0v=(0+0)v=0v, cancel)
  • a·0 = 0 — any scalar times the zero vector = zero vector
  • If av = 0 then a = 0 or v = 0 — no zero divisors
  • (−1)v = −v — scalar −1 produces the additive inverse
  • (−a)v = −(av) = a(−v)
Uniqueness Results
The zero vector is unique; the negative $-\mathbf{v}$ of each vector is unique. Both uniqueness results follow directly from the Cancellation Law.

Subspaces are automatically vector spaces
Every subspace of $\mathbb{R}^n$ is a vector space: A1 & S1 are the subspace conditions; the other 8 axioms are inherited from the parent space (Example 6.1.3). The same inheritance principle applies to subspaces of any abstract vector space.
Polynomial Space Pn — Example 6.1.6
Definition
$\mathcal{P}_n$ = {$a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n$ | $a_i \in \mathbb{R}$}

Two polynomials are equal iff all corresponding coefficients are equal (x is an indeterminate).
Operations (pointwise) $$(p+q)(x) = (a_0+b_0) + (a_1+b_1)x + \cdots$$ $$(cp)(x) = ca_0 + (ca_1)x + \cdots$$
Standard Basis and Dimension
The set $\{1, x, x^2, \ldots, x^n\}$ is a basis for $\mathcal{P}_n$, so $\dim \mathcal{P}_n = n+1$.
Independence: $a_0+a_1 x+\cdots+a_n x^n=0$ as a polynomial implies all $a_i=0$.
Example
$\mathcal{P}_2$ has basis $\{1, x, x^2\}$, $\dim = 3$.
Another basis: $\{1+x, x+x^2, 1+x^2\}$ (also independent, same count = 3). ✓
Inclusion chain
$\mathcal{P}_0 \subseteq \mathcal{P}_1 \subseteq \mathcal{P}_2 \subseteq \cdots \subseteq \mathcal{P} \subseteq F[a,b]$. Each $\mathcal{P}_n$ is a subspace of $\mathcal{P}$ (Example 6.2.5).
Function Spaces — Example 6.1.7
F[a,b] — all functions on [a,b]
F[a,b]: zero element is $0(x)=0$; negative of f is $(-f)(x) = -f(x)$.
D[a,b] — differentiable functions (subspace)
$D[a,b] \subseteq F[a,b]$: zero function is differentiable; $(f+g)'=f'+g'$; $(af)'=af'$ — closure verified via calculus.
Infinite Dimensionality
$F[a,b]$ and $\mathcal{P}$ are infinite dimensional: for every $n$, $\mathcal{P}_n$ is a subspace of dimension $n+1$. If $\dim \mathcal{P} = m$ were finite, then $\mathcal{P}_m$ ($\dim = m+1$) $\leq m$ — contradiction (Example 6.4.3).
ODE Solution Space
Solutions to $y'' + py' + qy = 0$ ($p$, $q$ constants) form a 2-dimensional subspace of $D[a,b]$: any two independent solutions $\{y_1, y_2\}$ form a basis.
Functions vs. Polynomials
$f = g$ in $F[a,b]$ requires $f(x)=g(x)$ for every $x \in [a,b]$. Two polynomials equal as functions on $[a,b]$ are equal in $P$.
§6.2 Subspaces and Spanning Sets
Subspace Test — Theorem 6.2.1
Definition 6.2 — Subspace
A nonempty subset $U \subseteq V$ is a subspace of $V$ if $U$ is itself a vector space using $V$'s operations.
Theorem 6.2.1 — Subspace Test (3 conditions)
$U$ is a subspace of $V$ if and only if:
  1. 0 ∈ U — the zero vector of V belongs to U
  2. u₁, u₂ ∈ U ⇒ u₁ + u₂ ∈ U — closed under addition
  3. u ∈ U, c ∈ ℝ ⇒ cu ∈ U — closed under scalar mult.
Proof sketch
(2) & (3) give A1 & S1. The other 8 axioms are inherited from V. (1) gives A4 (zero of V lives in U). Negative: $-\mathbf{u} = (-1)\mathbf{u} \in U$ by (3) and Thm 6.1.3, giving A5.
Shortcut
$U$ is a subspace iff $U$ is nonempty and closed under both operations. (Nonempty + closed under scalar mult. implies $\mathbf{0} \in U$ since $\mathbf{0} = 0\cdot\mathbf{u}$.)
Theorem 6.2.2 — Span is the Smallest Subspace
$\text{span}\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ is a subspace of V containing all $\mathbf{v}_i$, and is contained in every subspace that contains all $\mathbf{v}_i$.
Subspace Examples in Abstract Spaces
Matrix subspaces of Mnn
  • $\text{Sym}_n$ $= \{A \in \mathcal{M}_{nn} \mid A^T=A\}$  ✓ subspace, $\dim = n(n+1)/2$
  • Upper triangular matrices in $\mathcal{M}_{nn}$: ✓ subspace
  • $\{X \in \mathcal{M}_{nn} \mid AX = XA\}$ for fixed $A$: ✓ subspace (Example 6.2.3)
Polynomial subspaces of P
  • $U = \{p(x) \in \mathcal{P} \mid p(0) = 0\}$: ✓ subspace (zero poly satisfies it; closure preserved)
  • $U = \{p(x) \in \mathcal{P} \mid p(0) = 1\}$: ✖ NOT a subspace (zero polynomial has $p(0)=0 \neq 1$, fails condition 1)
  • $\mathcal{P}_n$ inside $\mathcal{P}$: ✓ subspace for each $n \geq 0$
Function subspaces of F[a,b]
  • $D[a,b]$: ✓ subspace (differentiable functions)
  • $\{f \mid f(0) = 0\}$: ✓ subspace
  • $\{f \mid f(x) \geq 0 \text{ for all } x\}$: ✖ NOT a subspace (not closed under negative scalar)
Standard spanning sets $$\mathcal{P}_n = \text{span}\{1, x, x^2, \ldots, x^n\} \quad \text{(Example 6.2.9)}$$ $$\mathcal{M}_{mn} = \text{span}\{E_{ij}\} \quad \text{(one 1, all other entries 0; Example 6.2.8)}$$
§6.3 Independence, Basis & Dimension
Linear Independence — Definition 6.4
Definition 6.4 — Linear Independence
$\{\mathbf{v}_1,\ldots,\mathbf{v}_k\} \subseteq V$ is linearly independent iff:

$c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0} \Rightarrow c_1 = \cdots = c_k = 0$
Test procedure in Pn
Write $c_1 p_1 + \cdots + c_k p_k = 0$ as a polynomial; equate coefficients of each power of $x$ (treating x as indeterminate) $\to$ homogeneous system $\to$ solve. Only trivial solution? $\Rightarrow$ independent.
Examples
$\{1, x, x^2\}$ in $\mathcal{P}_2$: independent (standard basis). ✓
$\{1+x, x+x^2, 1+x^2\}$ in $\mathcal{P}_2$: independent (set up $3\times 3$ system, $\det \neq 0$). ✓
$\{\sin x, \cos x\}$ in $F[0,2\pi]$: independent (set $x=0 \Rightarrow c_2=0$; $x=\pi/2 \Rightarrow c_1=0$). ✓
Key Fact — Distinct Degrees
Any set of polynomials of distinct degrees is automatically linearly independent (Example 6.3.4). Leading-term argument: each degree forces its coefficient to zero inductively.
Theorem 6.3.1 — Unique Representation
An independent set $\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ gives every vector in its span a unique representation as a linear combination.
Basis and Dimension — Definitions 6.5–6.6
Definition 6.5 — Basis
$\{\mathbf{e}_1,\ldots,\mathbf{e}_n\}$ is a basis of V iff it is linearly independent and $V = \text{span}\{\mathbf{e}_1,\ldots,\mathbf{e}_n\}$.
Definition 6.6 — Dimension
$\dim V = n$ where n is the number of vectors in any basis. Convention: $\dim\{\mathbf{0}\} = 0$. If no finite basis exists, V is infinite dimensional.
Theorem 6.3.3 — Invariance
All bases of V contain the same number of vectors — so $\dim V$ is well-defined.
Key Dimensions $$\dim(\mathbb{R}^n) = n$$ $$\dim(\mathcal{M}_{mn}) = mn \quad \text{— basis: all } E_{ij} \text{ (matrix with single 1)}$$ $$\dim(\mathcal{P}_n) = n+1 \quad \text{— basis: } \{1, x, x^2, \ldots, x^n\}$$ $$\dim(\mathcal{P}) = \infty, \quad \dim(F[a,b]) = \infty$$
Example — Symmetric 2×2 Matrices
$\text{Sym}_2 = \{A \in \mathcal{M}_{22} \mid A^T=A\}$: basis = $\left\{\begin{bmatrix}1&0\\0&0\end{bmatrix},\begin{bmatrix}0&0\\0&1\end{bmatrix},\begin{bmatrix}0&1\\1&0\end{bmatrix}\right\}$, $\dim \text{Sym}_2 = 3$.
Fundamental Theorem (Exchange) — Theorem 6.3.2
Theorem 6.3.2 — Fundamental Theorem
If V is spanned by $m$ vectors and $\{\mathbf{u}_1,\ldots,\mathbf{u}_k\}$ is linearly independent in $V$, then $k \leq m$.
Steinitz Exchange idea
The $k$ independent vectors can replace $k$ of the $m$ spanning vectors one at a time (the exchange step), maintaining the spanning property. At most $m$ exchanges are possible, so $k \leq m$.
Corollary 1 — All Bases Same Size
If $\{\mathbf{e}_1,\ldots,\mathbf{e}_n\}$ and $\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$ are both bases of V, then $n = m$ (apply the theorem in both directions).
Corollary 2 — Overcrowding
Any $k+1$ vectors in a $k$-dimensional space are linearly dependent.
Application
$\dim \mathcal{P}_2 = 3 \Rightarrow$ any 4 polynomials of degree $\leq 2$ are dependent.
E.g., $\{1, x, x^2, 1+x\}$ is dependent: $1+x = 1\cdot 1 + 1\cdot x + 0\cdot x^2$ is a nontrivial vanishing combination $(-1)(1+x) + 1 + x = 0$.
Basis Extension & Contraction — Theorem 6.4.1
Theorem 6.4.1 — Main Structure Theorem (finite-dim V)
Let V be finite dimensional (spanned by $m$ vectors). Then:
  1. V has a finite basis and dim V ≤ m.
  2. Extension: any independent set can be enlarged to a basis by adding vectors from any fixed basis of V.
  3. Subspace: if U ⊆ V then U is finite-dim, dim U ≤ dim V, and any basis of U extends to a basis of V.
Theorem 6.4.3 — Spanning → Basis
Any spanning set of a finite-dim V can be cut down (by deleting vectors) to a basis.
Theorem 6.4.4 — Labour-Saving Criterion
If $\dim V = n$ and $|S| = n$ vectors in $V$:
$S$ independent $\Leftrightarrow$ $S$ spans $V$ $\Leftrightarrow$ $S$ is a basis.
Only verify one condition for the right-sized set!
Example — Extending in P₃
$\{1+x, 1+x^2\}$ is independent in $\mathcal{P}_3$ ($\dim=4$). The standard basis is $\{1,x,x^2,x^3\}$: adding $x^3$ and $1$ gives $\{1, 1+x, 1+x^2, x^3\}$ — four polynomials of distinct effective degrees $\Rightarrow$ basis. ✓
§6.4 Isomorphism
Isomorphism — Definition & Theorem
Definition — Isomorphism
A map $T: V \to W$ is an isomorphism if T is:
(1) linear: $T(a\mathbf{u} + b\mathbf{v}) = aT(\mathbf{u}) + bT(\mathbf{v})$ for all $a,b \in \mathbb{R}$, $\mathbf{u},\mathbf{v} \in V$
(2) bijective: one-to-one (injective) and onto (surjective)
Isomorphic Spaces V ≅ W
V and W are isomorphic ($V \cong W$) if an isomorphism $T: V \to W$ exists. Isomorphic spaces are algebraically identical.
Theorem — Dimension Characterises Isomorphism
Two finite-dimensional vector spaces V and W (over ℝ) are isomorphic iff:

$\dim V = \dim W$

In particular, every n-dimensional real vector space is isomorphic to $\mathbb{R}^n$.
Key isomorphisms $$\mathcal{P}_n \cong \mathbb{R}^{n+1} \quad \text{via } a_0+a_1 x+\cdots+a_n x^n \mapsto (a_0,a_1,\ldots,a_n)$$ $$\mathcal{M}_{mn} \cong \mathbb{R}^{mn} \quad \text{via entry enumeration}$$
Significance
Any theorem proved in $\mathbb{R}^n$ (independence, span, dimension) automatically holds in every $n$-dimensional vector space via isomorphism.
Coordinate Isomorphism — CB
Coordinate Vector
Fix an ordered basis $\mathcal{B} = \{\mathbf{b}_1, \mathbf{b}_2, \ldots, \mathbf{b}_n\}$ of $V$. Every $\mathbf{v} \in V$ has a unique representation:

$\mathbf{v} = c_1\mathbf{b}_1 + c_2\mathbf{b}_2 + \cdots + c_n\mathbf{b}_n$

The coordinate vector of $\mathbf{v}$ relative to $\mathcal{B}$:
$[\mathbf{v}]_{\mathcal{B}} = (c_1, c_2, \ldots, c_n) \in \mathbb{R}^n$
Coordinate Map CB : V → ℝn
$C_{\mathcal{B}}(\mathbf{v}) = [\mathbf{v}]_{\mathcal{B}}$ — assigns each vector in $V$ its coordinate $n$-tuple.
Theorem — CB is an Isomorphism
The coordinate map $C_{\mathcal{B}}: V \to \mathbb{R}^n$ is an isomorphism: linear and bijective.
Linearity of Coordinates $$[c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k]_{\mathcal{B}} = c_1[\mathbf{v}_1]_{\mathcal{B}} + c_2[\mathbf{v}_2]_{\mathcal{B}} + \cdots + c_k[\mathbf{v}_k]_{\mathcal{B}}$$
Example in P₂
$\mathcal{B} = \{1, x, x^2\}$. Then $[3 - 2x + 5x^2]_{\mathcal{B}} = (3, -2, 5) \in \mathbb{R}^3$.
$\{1+x, x+x^2, 1+x^2\}$ indep. in $\mathcal{P}_2$ $\Leftrightarrow$ $\{(1,1,0),(0,1,1),(1,0,1)\}$ indep. in $\mathbb{R}^3$.
$\det\begin{bmatrix}1&0&1\\1&1&0\\0&1&1\end{bmatrix} = 2 \neq 0$. ✓
Practical Power
Any question about a finite-dimensional abstract space translates to $\mathbb{R}^n$ via $C_{\mathcal{B}}$, can be solved by row reduction there, and translated back.
