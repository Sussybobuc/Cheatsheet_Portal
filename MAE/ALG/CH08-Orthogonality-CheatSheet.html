§8.1 Orthogonal & Orthonormal Sets
Orthogonal & Orthonormal Sets
Definition — Orthogonal Set
A set {v1,…,vk} of nonzero vectors in $\mathbb{R}^n$ is orthogonal if $\mathbf{v}_i \cdot \mathbf{v}_j = 0$ for all $i \neq j$.
Definition — Orthonormal Set
An orthogonal set is orthonormal if additionally $\|\mathbf{v}_i\| = 1$ for every $i$; equivalently $\mathbf{v}_i \cdot \mathbf{v}_j = \delta_{ij}$.
Theorem 5.3.5 — Independence
Every orthogonal set of nonzero vectors is linearly independent.
Normalizing an Orthogonal Set Given orthogonal $\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$, define $$\mathbf{f}_i = \frac{\mathbf{v}_i}{\|\mathbf{v}_i\|}$$ to obtain an orthonormal set with the same span.
Why it matters
Orthogonal bases are the "nice" bases: expansion coefficients are trivial to compute — no solving a linear system needed.
Expansion in an Orthonormal Basis
Expansion Theorem (ONB)
If {f1,…,fn} is an orthonormal basis for $\mathbb{R}^n$, then for any $\mathbf{v} \in \mathbb{R}^n$:

$\mathbf{v} = (\mathbf{v}\cdot\mathbf{f}_1)\mathbf{f}_1 + (\mathbf{v}\cdot\mathbf{f}_2)\mathbf{f}_2 + \cdots + (\mathbf{v}\cdot\mathbf{f}_n)\mathbf{f}_n$
Fourier Coefficients
The scalars v·fi are the Fourier coefficients — the coordinates of v in the ONB.
Parseval's Identity $$\|\mathbf{v}\|^2 = (\mathbf{v} \cdot \mathbf{f}_1)^2 + (\mathbf{v} \cdot \mathbf{f}_2)^2 + \cdots + (\mathbf{v} \cdot \mathbf{f}_n)^2$$
General Orthogonal Basis $\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$ If $U = \text{span}\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$ is orthogonal and $\mathbf{v} \in U$: $$\mathbf{v} = \sum_k \frac{\mathbf{v} \cdot \mathbf{f}_k}{\|\mathbf{f}_k\|^2} \mathbf{f}_k$$
Orthonormal columns
If Q = [f1|…|fn] (columns ONB), then $Q^T Q = I$ and expansion becomes $\mathbf{v} = Q(Q^T\mathbf{v})$.
§8.1 Orthogonal Complement & Projection
Orthogonal Complement
Definition 8.1
If $U$ is a subspace of $\mathbb{R}^n$, the orthogonal complement of U is
$U^\perp = \{ \mathbf{x} \in \mathbb{R}^n \mid \mathbf{x} \cdot \mathbf{y} = 0 \text{ for all } \mathbf{y} \in U \}$
Lemma 8.1.2 — Properties
1. $U^\perp$ is a subspace of $\mathbb{R}^n$.
2. $\{\mathbf{0}\}^\perp = \mathbb{R}^n$ and $(\mathbb{R}^n)^\perp = \{\mathbf{0}\}$.
3. If $U = \text{span}\{\mathbf{x}_1,\ldots,\mathbf{x}_k\}$, then $U^\perp = \{ \mathbf{x} \mid \mathbf{x} \cdot \mathbf{x}_i = 0,\ \forall i \}$.
Theorem 8.1.4 — Dimension & Double Complement
$\dim U + \dim U^\perp = n$   and   $(U^\perp)^\perp = U$
Direct Sum Decomposition $$\mathbb{R}^n = U \oplus U^\perp$$ (unique: every $\mathbf{x} = \mathbf{p} + \mathbf{q}$, $\mathbf{p} \in U$, $\mathbf{q} \in U^\perp$)
Fundamental Theorem of Linear Algebra
$\text{null}\, A = (\text{row}\, A)^\perp$   and   $\text{null}\, A^T = (\text{col}\, A)^\perp$
Orthogonal Projection
Definition 8.2
Let U have orthogonal basis $\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$. The orthogonal projection of $\mathbf{x}$ onto $U$ is:
Projection Formula $$\text{proj}_U(\mathbf{x}) = \sum_{k=1}^{m} \frac{\mathbf{x} \cdot \mathbf{f}_k}{\|\mathbf{f}_k\|^2} \mathbf{f}_k$$
Theorem 8.1.3 — Projection Theorem
Let $\mathbf{p} = \text{proj}_U(\mathbf{x})$. Then:
1. $\mathbf{p} \in U$ and $\mathbf{x} - \mathbf{p} \in U^\perp$.
2. $\mathbf{p}$ is the closest vector in $U$ to $\mathbf{x}$: $\|\mathbf{x}-\mathbf{p}\| < \|\mathbf{x}-\mathbf{y}\|$ for all $\mathbf{y} \in U$, $\mathbf{y} \neq \mathbf{p}$.
(Proof: Pythagorean theorem — $\|\mathbf{x}-\mathbf{y}\|^2 = \|\mathbf{x}-\mathbf{p}\|^2 + \|\mathbf{p}-\mathbf{y}\|^2$)
Theorem 8.1.4 — Projection is Linear
$T(\mathbf{x}) = \text{proj}_U(\mathbf{x})$ is a linear operator with $\text{im}\, T = U$ and $\ker T = U^\perp$.
Independence of basis
projU(x) depends only on x and U, not on which orthogonal basis is chosen for U.
§8.1 Gram-Schmidt Orthogonalization
Gram-Schmidt Algorithm
Theorem 8.1.2 — Setup
Given any basis $\{\mathbf{x}_1,\ldots,\mathbf{x}_m\}$ of subspace $U \subseteq \mathbb{R}^n$, construct an orthogonal basis $\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$ successively:
Construction $$\mathbf{f}_1 = \mathbf{x}_1$$ $$\mathbf{f}_2 = \mathbf{x}_2 - \frac{\mathbf{x}_2 \cdot \mathbf{f}_1}{\|\mathbf{f}_1\|^2}\mathbf{f}_1$$ $$\mathbf{f}_3 = \mathbf{x}_3 - \frac{\mathbf{x}_3 \cdot \mathbf{f}_1}{\|\mathbf{f}_1\|^2}\mathbf{f}_1 - \frac{\mathbf{x}_3 \cdot \mathbf{f}_2}{\|\mathbf{f}_2\|^2}\mathbf{f}_2$$ $$\vdots$$ $$\mathbf{f}_k = \mathbf{x}_k - \sum_{j=1}^{k-1} \frac{\mathbf{x}_k \cdot \mathbf{f}_j}{\|\mathbf{f}_j\|^2}\mathbf{f}_j$$
To get an ONB
Normalize: $\mathbf{q}_k = \mathbf{f}_k/\|\mathbf{f}_k\|$. Multiplying any $\mathbf{f}_k$ by a nonzero scalar leaves subsequent vectors unchanged — clear fractions freely.
Theorem 8.1.2 — Guarantees
1. $\{\mathbf{f}_1,\ldots,\mathbf{f}_m\}$ is an orthogonal basis of $U$.
2. $\text{span}\{\mathbf{f}_1,\ldots,\mathbf{f}_k\} = \text{span}\{\mathbf{x}_1,\ldots,\mathbf{x}_k\}$ for each $k$.

Corollary: Every subspace of $\mathbb{R}^n$ has an orthogonal basis (Theorem 8.1.1).
Example 8.1.1
Rows x1=(1,1,−1,−1), x2=(3,2,0,1), x3=(1,0,1,0):
f1=(1,1,−1,−1);  f2=(2,1,1,2);  f3=(4,−3,7,−6)
§8.4 QR-Factorization
QR-Factorization
Definition 8.6 & Theorem 8.4.1
Let A be m×n with linearly independent columns. A QR-factorization of A is A = QR where:
  • Q is m×n with orthonormal columns
  • R is n×n upper triangular with positive diagonal entries (invertible)
Every such A has a unique QR-factorization (Theorem 8.4.3).
Construction via Gram-Schmidt on columns $\mathbf{c}_1,\ldots,\mathbf{c}_n$ of $A$ Apply G-S $\Rightarrow$ orthonormal $\mathbf{q}_1,\ldots,\mathbf{q}_n$; $$R_{ii} = \|\mathbf{f}_i\| > 0, \quad R_{ij} = \mathbf{c}_j \cdot \mathbf{q}_i \ (i < j), \quad R_{ij} = 0 \ (i > j)$$
Theorem 8.4.2 — Square Case
Every square invertible matrix has A = QR with Q orthogonal, R upper triangular (positive diagonal).
Key Identity (used in Least Squares) $$A^T A = R^T Q^T Q R = R^T R$$
Why QR is used numerically
Computing (ATA)−1 reduces to inverting the upper triangular R — numerically stable, avoids explicit formation of ATA.
Corollary 8.4.1 (Independent Rows)
If A has independent rows: A = LP where P has orthonormal rows and L is invertible lower triangular.
§8.2 Orthogonal Matrices
Orthogonal Matrix — Definition & Conditions
Definition 8.3
An n×n matrix P is orthogonal if any of the following equivalent conditions hold (Theorem 8.2.1):
ConditionStatement
Inverse$P$ invertible and $P^{-1} = P^T$
Product$P^T P = I$ (equivalently $PP^T = I$)
RowsRows of $P$ form an orthonormal set in $\mathbb{R}^n$
ColumnsColumns of $P$ form an orthonormal set in $\mathbb{R}^n$
Caution
Merely orthogonal rows (not unit length) does not make a matrix orthogonal — the rows must be orthonormal.
Closure (Example 8.2.3)
If $P$, $Q$ orthogonal: $PQ$ is orthogonal; $P^{-1} = P^T$ is orthogonal.
Properties of Orthogonal Matrices
Isometry — Preserves Geometry (Exercise 8.2.24)
If $P$ is orthogonal and $\mathbf{x}, \mathbf{y} \in \mathbb{R}^n$:

1. $\|P\mathbf{x}\| = \|\mathbf{x}\|$  (preserves length)
2. $(P\mathbf{x})\cdot(P\mathbf{y}) = \mathbf{x}\cdot\mathbf{y}$  (preserves dot product)
3. $\|P\mathbf{x}-P\mathbf{y}\| = \|\mathbf{x}-\mathbf{y}\|$  (preserves distance)
4. $\det P = \pm 1$ (Exercise 8.2.18)
Theorem 8.2.5 — Triangulation
If $A$ is $n\times n$ with $n$ real eigenvalues, $\exists$ orthogonal $P$ such that $P^T A P$ is upper triangular.
Corollary 8.2.1
If $\lambda_1,\ldots,\lambda_n$ are the real eigenvalues of $A$:
$\det A = \lambda_1\lambda_2\cdots\lambda_n$   and   $\text{tr}\, A = \lambda_1+\cdots+\lambda_n$
Standard Examples
Rotation Rθ (det = +1, preserves orientation);
Reflection (det = −1, reverses orientation);   Permutation matrices.
§8.2 Symmetric Matrices & Orthogonal Diagonalization
Properties of Symmetric Matrices
Definition
A matrix A is symmetric if $A = A^T$.
Theorem 5.5.7 — Real Eigenvalues
Every eigenvalue of a real symmetric matrix is real.
Proof: λ‖v‖² = (Av)·v = v·(Av) = v·(λv) = λ̄‖v‖² ⟹ λ = λ̄ ∈ ℝ.
Theorem 8.2.3 — Self-Adjoint Property
If A is symmetric: (Ax)·y = x·(Ay) for all x,y ∈ ℝn.
Proof: (Ax)·y = (Ax)Ty = xTATy = xTAy.
Theorem 8.2.4 — Orthogonal Eigenvectors
Eigenvectors of a symmetric matrix corresponding to distinct eigenvalues are orthogonal.
Proof: $\lambda(\mathbf{x}\cdot\mathbf{y}) = (A\mathbf{x})\cdot\mathbf{y} = \mathbf{x}\cdot(A\mathbf{y}) = \mu(\mathbf{x}\cdot\mathbf{y})$, so $(\lambda-\mu)(\mathbf{x}\cdot\mathbf{y}) = 0$.
Repeated eigenvalues
Apply Gram-Schmidt within each eigenspace to produce orthonormal eigenvectors.
Orthogonal Diagonalization
Definition 8.4
A is orthogonally diagonalizable if $\exists$ orthogonal $P$ with $P^T A P = D$ diagonal.
Theorem 8.2.2 — Principal Axis Theorem
The following are equivalent for an n×n matrix A:
1. $A$ has an orthonormal set of $n$ eigenvectors.
2. $A$ is orthogonally diagonalizable ($P^T A P = D$).
3. $A$ is symmetric.
Algorithm — Orthogonal Diagonalization
  1. Find all distinct (real) eigenvalues $\lambda_1,\ldots,\lambda_k$
  2. Find a basis for each eigenspace $E_{\lambda_i}$
  3. Apply Gram-Schmidt within each eigenspace
  4. Normalize all eigenvectors; form P = [q1|…|qn]
  5. Result: $P^T A P = D = \text{diag}(\lambda_1,\ldots,\lambda_n)$
Example 8.2.5
A = [[8,−2,2],[−2,5,4],[2,4,5]]; eigenvalues 0 (mult 1) and 9 (mult 2);
E9 requires Gram-Schmidt → get orthonormal P.
§8.2 Spectral Theorem & Spectral Decomposition
Spectral Theorem
Theorem 8.2.2 — Real Symmetric Matrices
Every real symmetric matrix A is orthogonally diagonalizable:
$A = PDP^T$, $P$ orthogonal, $D = \text{diag}(\lambda_1,\ldots,\lambda_n)$.
Spectral Decomposition (Rank-1 Sum) $$A = \lambda_1 \mathbf{f}_1 \mathbf{f}_1^T + \lambda_2 \mathbf{f}_2 \mathbf{f}_2^T + \cdots + \lambda_n \mathbf{f}_n \mathbf{f}_n^T$$
Terminology
The set of distinct eigenvalues of A is the spectrum of A. The orthonormal eigenvectors fi are the principal axes of A.
Each fifiT
Each outer product $\mathbf{f}_i\mathbf{f}_i^T$ is a rank-1 matrix representing the orthogonal projection onto the eigenspace for $\lambda_i$.
Theorem 8.6.8 — Complex (Hermitian) Spectral Theorem
If A is hermitian (AH = A), there exists a unitary matrix U with UHAU diagonal.
Theorem 8.6.5 — Hermitian Properties
If A is hermitian:
1. All eigenvalues of A are real.
2. Eigenvectors for distinct eigenvalues are orthogonal in ℂn.
Theorem 8.6.9 — Normal Matrices
$A$ is unitarily diagonalizable $\Leftrightarrow$ $A$ is normal ($AA^H = A^H A$).
Normal includes: symmetric, hermitian, unitary, and orthogonal matrices.
Theorem 8.6.7 — Schur's Theorem
Every complex n×n matrix is unitarily triangularizable: ∃ unitary U with UHAU = T upper triangular (eigenvalues on diagonal).
§8.3 Positive Definite Matrices
Positive Definite Matrices
Definition 8.5
A square matrix A is positive definite (SPD) if it is symmetric and all eigenvalues λ > 0.
Theorem 8.3.1 — Invertibility
Every positive definite matrix A is invertible with det A > 0.
Theorem 8.3.2 — Quadratic Form Criterion
A symmetric A is positive definite ⟺ xTAx > 0 for every nonzero x.
Proof: using PTAP = D: xTAx = Σλiyi² (with y = PTx).
Theorem 8.3.3 — Equivalent Conditions
For symmetric n×n A, the following are equivalent:
1. A is positive definite.
2. det((r)A) > 0 for r = 1,…,n  (all leading principal minors positive).
3. A = UTU for upper triangular U with positive diagonal.
ATA is Always PSD
$A^T A$ is symmetric positive semidefinite for any $A$; positive definite $\Leftrightarrow$ $A$ has full column rank.
Cholesky Factorization
Theorem 8.3.3 — Unique Factorization
Every positive definite matrix A has a unique factorization A = UTU where U is upper triangular with positive diagonal entries. This is the Cholesky factorization.
Algorithm
  1. Row-reduce A → upper triangular U1 using only row ops that add multiples of a row to a lower row (preserves positive diagonal)
  2. Divide each row i of U1 by √(diagonal entry of row i) → U
Then A = UTU.
Example 8.3.3
A = [[10,5,2],[5,3,2],[2,2,3]]; verify det((1)A)=10, det((2)A)=5, det A=3 > 0 ✓
U1 = [[10,5,2],[0,½,1],[0,0,3/5]]; U = [[√10, 5/√10, 2/√10],[0, 1/√2, √2],[0, 0, √(3/5)]]
Positive Semidefinite
A is positive semidefinite if symmetric and all λ ≥ 0, i.e., xTAx ≥ 0 for all x.
§8.4 / §5.6 Least-Squares Approximation
Least-Squares Problem & Solution via QR
Problem Setup
Given a (possibly inconsistent) system Ax = b (m equations, n unknowns, m > n), find z that minimizes ‖Az − b‖² over all z ∈ ℝn.
Best Approximation
The minimizer satisfies Az = projcol(A)(b), i.e., b − Az ⊥ col(A).
Normal Equations (§5.6)
z is a least-squares solution ⟺ it satisfies the normal equations:

$$( A^T A )\, \mathbf{z} = A^T \mathbf{b}$$ Unique least-squares solution when A has full column rank:
$\mathbf{z} = (A^T A)^{-1} A^T \mathbf{b}$
Via QR (Numerically Stable)
If $A = QR$ (full column rank), then $A^T A = R^T R$, so:

$$\mathbf{z} = R^{-1} Q^T \mathbf{b}$$ No need to form ATA explicitly — just back-substitute through R.
Minimum Residual $$\|\mathbf{b} - A\mathbf{z}\| = \|\mathbf{b} - \text{proj}_{\text{col}(A)}(\mathbf{b})\|$$
Gram matrix
$G = A^T A$ is symmetric positive semidefinite; positive definite $\Leftrightarrow$ $A$ full column rank $\Leftrightarrow$ unique least-squares solution.
Pseudoinverse
When $A$ has full column rank, the map $A^+ = (A^T A)^{-1} A^T$ is the left inverse / Moore-Penrose pseudoinverse.
§8.5 Computing Eigenvalues
Power Method
Setup
Finds the dominant eigenvalue λ of A iteratively.
Algorithm
Start with $\mathbf{x}_0$; iterate $\mathbf{x}_{k+1} = A\mathbf{x}_k$ (so $\mathbf{x}_k = A^k\mathbf{x}_0$).
Rayleigh Quotient (eigenvalue estimate) $$r_k = \frac{\mathbf{x}_k \cdot \mathbf{x}_{k+1}}{\|\mathbf{x}_k\|^2} \to \lambda_1 \text{ (dominant eigenvalue)}$$
Convergence
If $\mathbf{x}_0$ has nonzero component along the dominant eigenvector and $|\lambda_i/\lambda_1| < 1$ for $i > 1$, then
$(1/\lambda_1^k)\mathbf{x}_k \to$ dominant eigenvector.
Limitations
Slow if $|\lambda_2/\lambda_1| \approx 1$; fails if $\mathbf{x}_0 \perp$ dominant eigenvector; not recommended for large dense matrices.
QR-Algorithm
Iteration
Set $A_1 = A$; repeatedly factor and reverse:
$A_k = Q_k R_k \Rightarrow A_{k+1} = R_k Q_k$
Key Property
Ak+1 = Qk−1AkQk, so each Ak is similar to A (same eigenvalues). If A has real eigenvalues with distinct absolute values, Ak → upper triangular with eigenvalues on diagonal.
  • If $A$ symmetric $\Rightarrow$ $A_k \to$ diagonal (Exercise 8.5.4)
  • Complex eigenvalue pairs → 2×2 diagonal blocks
  • $A_k = (Q_1 Q_2 \cdots Q_k)(R_k \cdots R_1)$ is a QR-factorization of $A^k$
Practical Enhancements
Shifting: factor Ak−skI → Ak+1 = RkQk+skI (accelerates convergence).
Hessenberg pre-reduction: use Householder matrices to put A in upper Hessenberg form first.
§8.6 Complex Matrices — Hermitian, Unitary & Normal
Hermitian, Unitary & Normal Matrices
Standard Inner Product on ℂⁿ (Definition 8.7)
⟨z,w⟩ = z1w̄1 + ··· + znw̄n   (conjugate the second argument)
‖z‖ = √⟨z,z⟩ = √(|z1|²+···+|zn|²)
Conjugate Transpose (Definition 8.9)
AH = (Ā)T = conjugate transpose of A. AH = AT when A is real.
Hermitian (Definition 8.10)
A is hermitian if $A^H = A$ (equivalently $\bar{A} = A^T$).   Real symmetric $\subseteq$ hermitian.
Unitary (Definition 8.12)
U is unitary if $U^{-1} = U^H$.   Real orthogonal $\subseteq$ unitary.
Theorem 8.6.5 — Hermitian Eigenvalues
Let A be hermitian. Then:
1. All eigenvalues of $A$ are real.
2. Eigenvectors for distinct eigenvalues are orthogonal in $\mathbb{C}^n$.
Theorem 8.6.8 — Spectral Theorem (Complex)
If $A$ is hermitian, $\exists$ unitary $U$ with $U^H A U$ diagonal (all diagonal entries real).
Theorem 8.6.9 — Normality
$A$ is unitarily diagonalizable $\Leftrightarrow$ $A$ is normal ($AA^H = A^H A$).
Theorem 8.6.10 — Cayley-Hamilton
Every $n\times n$ complex matrix satisfies its own characteristic polynomial: $c_A(A) = 0$.
