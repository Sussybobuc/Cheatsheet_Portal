§2.1 Matrix Foundations
Notation, special matrices, addition, scalar multiplication, transpose
Matrix Notation & Special Types
Definition 2.1 — Matrix
A rectangular array of numbers. An m×n matrix A has entries aij in row i and column j. Notation: A = [aij].
TypeConditionNotation
Squarem = n (n×n)An×n
Row matrixm = 1 (1×n)row vector
Column matrixn = 1 (m×1)column vector
Zero matrixevery entry = 00 or 0mn
Identity matrixsquare; 1s on diagonal, 0s elsewhereI or In
Diagonal matrixsquare; all off-diagonal entries = 0diag(d1,…,dn)
Matrix Equality
A = B if and only if: (1) they have the same size, and (2) aij = bij for all i, j.
Rows vs Columns Convention
Size is always stated as rows × columns. The (i, j)-entry lies in row i (first index) and column j (second index).
Addition, Scalar Multiplication & Properties
Definition 2.1 — Matrix Addition
If A and B have the same size: A + B = [aij + bij]. Not defined for different sizes.
Definition 2.2 — Scalar Multiple
For any matrix A and scalar k: kA = [kaij]. Thus −A = [−aij] and A−B = A+(−B).
Theorem 2.1.1 — Properties of Addition & Scalar Mult.
For m×n matrices A, B, C and scalars k, p:
  1. $A + B = B + A$   (commutative)
  2. $A + (B + C) = (A + B) + C$   (associative)
  3. $\mathbf{0} + A = A$   (zero matrix)
  4. $A + (-A) = \mathbf{0}$   (additive inverse)
  5. $k(A + B) = kA + kB$   (distributive)
  6. $(k + p)A = kA + pA$   (distributive)
  7. $(kp)A = k(pA)$   (scalar assoc.)
  8. $1 \cdot A = A$
Key Facts
$kA = \mathbf{0}$ implies $k = 0$ or $A = \mathbf{0}$ (converse is also true).
$\mathbf{0}A = \mathbf{0}$ and $k\mathbf{0} = \mathbf{0}$ for all $A$, $k$.
Matrix Transpose
Definition 2.3 — Transpose
If A is m×n, the transpose AT is the n×m matrix whose rows are the columns of A. Formally: if A = [aij] then AT = [aji].
Theorem 2.1.2 — Transpose Properties
  1. If $A$ is $m \times n$ then $A^T$ is $n \times m$.
  2. $(A^T)^T = A$
  3. $(kA)^T = kA^T$
  4. $(A + B)^T = A^T + B^T$
Symmetric Matrix
A square matrix $A$ is symmetric if $A = A^T$. Symmetric matrices are necessarily square.
Skew-Symmetric Matrix
A square matrix $W$ is skew-symmetric if $W^T = -W$. All diagonal entries of a skew-symmetric matrix are zero.
Decomposition
Every square matrix $A$ can be written uniquely as $A = S + W$ where $S = \tfrac{1}{2}(A + A^T)$ is symmetric and $W = \tfrac{1}{2}(A - A^T)$ is skew-symmetric.
Geometric view Forming $A^T$ = "flip $A$ about its main diagonal" (rotate 180° about the diagonal line). Transpose of a $3 \times 2$ matrix is $2 \times 3$.
§2.2–§2.3 Matrix-Vector & Matrix Multiplication
Column-combination definition, dot product rule, properties
Matrix-Vector Multiplication
Definition 2.5 — Matrix-Vector Product
Let A = [a1 a2 … an] (m×n, written in terms of columns) and x = (x1, …, xn)T. Then:
$$A\mathbf{x} = x_1\mathbf{a}_1 + x_2\mathbf{a}_2 + \cdots + x_n\mathbf{a}_n$$

Ax is a linear combination of the columns of A with coefficients from x. Ax is an m-vector.

Theorem 2.2.1 — Systems in Matrix Form
  1. Every linear system has matrix form $A\mathbf{x} = \mathbf{b}$ ($A$ = coefficient matrix, $\mathbf{b}$ = constant matrix, $\mathbf{x}$ = variables).
  2. System $A\mathbf{x} = \mathbf{b}$ is consistent iff $\mathbf{b}$ is a linear combination of the columns of $A$.
  3. If a1,…,an are columns of A, then x solves Ax = b iff x1a1+⋯+xnan = b.
Theorem 2.2.2 — Linearity of Ax
For matrix A and vectors x, y and scalar k:
  1. $A(\mathbf{x} + \mathbf{y}) = A\mathbf{x} + A\mathbf{y}$
  2. $A(k\mathbf{x}) = k(A\mathbf{x})$
In particular, if $A\mathbf{x}_0 = \mathbf{0}$ and $A\mathbf{x}_1 = \mathbf{b}$, then $\mathbf{x}_1 + t\mathbf{x}_0$ ($t$ any scalar) is an infinite family of solutions to $A\mathbf{x} = \mathbf{b}$.
Matrix Multiplication
Definition 2.9 — Matrix Product
Let $A$ be $m \times n$ and $B = [\mathbf{b}_1\ \mathbf{b}_2\ \cdots\ \mathbf{b}_k]$ be $n \times k$ (written in terms of its columns). The product $AB$ is the $m \times k$ matrix:
$$AB = A[\mathbf{b}_1\ \mathbf{b}_2\ \cdots\ \mathbf{b}_k] = [A\mathbf{b}_1\ A\mathbf{b}_2\ \cdots\ A\mathbf{b}_k]$$
Theorem 2.3.2 — Dot Product Rule
The dot product rule: the $(i,j)$-entry of $AB$ = dot product of row $i$ of $A$ with column $j$ of $B$.
Compatibility Rule
$AB$ is defined only if the number of columns of $A$ equals the number of rows of $B$.
$(m \times n)(n \times k) \to m \times k$
⚠ Non-commutativity: $AB$ and $BA$ are generally not equal — they may not even be the same size. Changing the order of factors in a product can change (or destroy) the result. This is the single most common error in matrix algebra.
Example — A² = 0 with A ≠ 0
$A = \begin{bmatrix} 6 & 9 \\ -4 & -6 \end{bmatrix}$: $A^2 = \mathbf{0}$ despite $A \neq \mathbf{0}$. Also $AB \neq BA$ for generic matrices.
Properties of Matrix Multiplication
Theorem 2.3.3 — Core Properties
Let a be any scalar; A, B, C matrices of compatible sizes:
  1. $IA = A$ and $AI = A$   (identity)
  2. $A(BC) = (AB)C$   (associative)
  3. $A(B + C) = AB + AC$   (left-distributive)
  4. $(B + C)A = BA + CA$   (right-distributive)
  5. $a(AB) = (aA)B = A(aB)$
  6. $(AB)^T = B^T A^T$   (reversal rule)
Theorem 2.3.1 — Associativity via Vectors
If A is m×n and B is n×k, then for all $\mathbf{x} \in \mathbb{R}^k$:
$A(B\mathbf{x}) = (AB)\mathbf{x}$
This is how matrix multiplication is motivated: it corresponds to composition of matrix transformations.
Theorem 2.3.4 — Block Multiplication
If matrices A and B are partitioned compatibly into blocks, AB can be computed using ordinary matrix multiplication with blocks as entries.
Commuting Matrices
A and B commute if $AB = BA$. $(A-B)(A+B) = A^2 - B^2$ if and only if $AB = BA$.
Transpose of product — reversal rule $$(AB)^T = B^T A^T$$ $$(ABC)^T = C^T B^T A^T$$ More generally: $$(A_1 A_2 \cdots A_k)^T = A_k^T \cdots A_2^T A_1^T$$
§2.4 Matrix Inverses
Definition, uniqueness, 2×2 formula, Gauss-Jordan algorithm, properties
Inverse Definition & 2×2 Formula
Definition 2.11 — Matrix Inverse
If A is a square matrix, a matrix B is an inverse of A if $AB = I$ and $BA = I$. A matrix that has an inverse is called invertible (or non-singular).
Theorem 2.4.1 — Uniqueness of Inverse
If B and C are both inverses of A, then B = C.
Proof: $B = IB = (CA)B = C(AB) = CI = C$.

If A is invertible its unique inverse is written A−1, satisfying AA−1 = I = A−1A.

2×2 Inverse Formula (Example 2.4.4) For $A = \begin{bmatrix} a & b \\ c & d \end{bmatrix}$: $$\det A = ad - bc$$ $$\operatorname{adj} A = \begin{bmatrix} d & -b \\ -c & a \end{bmatrix}$$ $$A^{-1} = \frac{1}{\det A} \cdot \operatorname{adj} A, \quad \text{provided } \det A \neq 0$$ $A$ is invertible iff $\det A \neq 0$.
Solving Ax = b with Inverse
If $A$ ($n \times n$) is invertible, the system $A\mathbf{x} = \mathbf{b}$ has unique solution $\mathbf{x} = A^{-1}\mathbf{b}$ (Theorem 2.4.2).
Corollary 2.4.1
For square $A$, $C$: if $AC = I$, then automatically $CA = I$. Both $A$ and $C$ are invertible with $C = A^{-1}$.
Computing Inverses — Gauss-Jordan Algorithm
Matrix Inversion Algorithm
If A is invertible, a sequence of elementary row operations carries A to I, and the same operations carry I to A−1:
$$\left[A \mid I\right] \xrightarrow{\text{row operations}} \left[I \mid A^{-1}\right]$$
  1. Form the augmented matrix [A | In].
  2. Perform row operations to reduce A to reduced row-echelon form.
  3. If A reduces to In, the right block is A−1.
  4. If A does not reduce to In, A has a row of zeros → A is not invertible.
Theorem 2.4.3 — Algorithm is Effective
For any n×n matrix A, either A can be reduced to I by row operations (algorithm produces A−1), or it cannot (A−1 does not exist).
Example 2.4.6 — 3×3 Inverse
A = [[2,7,1],[1,4,−1],[1,3,0]]
Apply [A|I] → [I|A−1] via row operations (swap rows, eliminate).
Result: A−1 = (1/2)[[−3,−3,11],[1,1,−3],[1,−1,−1]]
Corollary 2.4.2 — Rank Condition
An $n \times n$ matrix $A$ is invertible if and only if $\text{rank}(A) = n$.
Properties of Inverses
Theorem 2.4.4 — Inverse Properties
All matrices square of the same size:
  1. $I^{-1} = I$
  2. $(A^{-1})^{-1} = A$
  3. $(AB)^{-1} = B^{-1}A^{-1}$   (reversal)
  4. $(A_1 A_2 \cdots A_k)^{-1} = A_k^{-1} \cdots A_2^{-1} A_1^{-1}$
  5. $(A^k)^{-1} = (A^{-1})^k$ for $k \geq 1$
  6. $(aA)^{-1} = (1/a)A^{-1}$ for $a \neq 0$
  7. $(A^T)^{-1} = (A^{-1})^T$
⚠ Order reversal: $(AB)^{-1} = B^{-1}A^{-1}$, NOT $A^{-1}B^{-1}$. This is because matrix multiplication is not commutative.
Corollary 2.4.1 — Invertibility of Transpose
A square matrix $A$ is invertible if and only if $A^T$ is invertible.
Cancellation Laws (Example 2.4.7)
If A is invertible:
• $AB = AC \Rightarrow B = C$  (left cancellation)
• $BA = CA \Rightarrow B = C$  (right cancellation)
But mixed cancellation may fail: $AB = CA$ does not imply $B = C$.
Invertible Matrix Theorem (Theorem 2.4.5)
Theorem 2.4.5 — Equivalent Conditions for Invertibility
The following are equivalent for an n×n matrix A:
#Condition
(1)A is invertible (A−1 exists).
(2)The homogeneous system $A\mathbf{x} = \mathbf{0}$ has only the trivial solution $\mathbf{x} = \mathbf{0}$.
(3)$A$ can be reduced to the identity matrix $I_n$ by elementary row operations.
(4)For every column $\mathbf{b}$, $A\mathbf{x} = \mathbf{b}$ has at least one solution.
(5)There exists an n×n matrix C such that AC = In.
Proof Chain
$(1)\Rightarrow(2)\Rightarrow(3)\Rightarrow(4)\Rightarrow(5)\Rightarrow(1)$. Key step: $(3)\Rightarrow(4)$ because row-reducing $[A|\mathbf{b}]$ always produces a solution; $(5)\Rightarrow(1)$ because $AC = I$ forces $CA = I$ (Corollary 2.4.1).
Consequences $$\text{rank}(A) = n \;\Leftrightarrow\; A \text{ invertible} \quad \text{(Corollary 2.4.2)}$$ $A$ has a row of zeros $\Rightarrow$ $A$ is NOT invertible. $A$ has a column of zeros $\Rightarrow$ $A$ is NOT invertible. Each row (column) of $A$ sums to 0 $\Rightarrow$ NOT invertible.
§2.5 Elementary Matrices
Three types, left-multiplication = row operation, product representation
Elementary Matrices & Row Operations
Definition 2.12 — Elementary Matrix
An n×n matrix E is an elementary matrix if it is obtained from In by a single elementary row operation.
TypeOperation on IInverse Operation
IInterchange rows p and qInterchange rows p and q (self-inverse)
IIMultiply row p by k ≠ 0Multiply row p by 1/k
IIIAdd k × row p to row q (p ≠ q)Subtract k × row p from row q
Lemma 2.5.1 — Left Multiplication = Row Operation
Performing an elementary row operation on $A$ is the same as left-multiplying $A$ by the corresponding elementary matrix $E$:
$EA$ = $A$ with that row operation applied.
Lemma 2.5.2 — Every E is Invertible
Every elementary matrix E is invertible, and E−1 is also elementary of the same type (corresponding to the inverse row operation).
Example (2×2)
E1=[[0,1],[1,0]] (type I, swap rows) — self-inverse
E2=[[1,0],[0,9]] (type II, scale row 2 by 9) — inverse has 1/9
E3=[[1,5],[0,1]] (type III, add 5×row2 to row1) — inverse has −5
Factoring as Product of Elementary Matrices
Theorem 2.5.1 — Row Reduction = Left Multiplication by Invertible U
If A is m×n and A →B by elementary row operations with elementary matrices E1, E2, …, Ek (in order), then B = UA where U = Ek⋯E2E1 is invertible. Moreover, U can be computed from:
$$[A \mid I_m] \;\longrightarrow\; [B \mid U]$$
Theorem 2.5.2 — Invertible ⟺ Product of Elementaries
A square matrix is invertible if and only if it is a product of elementary matrices.
Factorization Formula If $A \to I$ via $E_1, E_2, \ldots, E_k$ (in order), then: $$A^{-1} = E_k E_{k-1} \cdots E_2 E_1$$ $$A = E_1^{-1} E_2^{-1} \cdots E_k^{-1}$$ (product of elementary matrices, each of the same type as $E_i$)
Theorem 2.5.3 — Smith Normal Form
For any m×n matrix A of rank r, there exist invertible U (m×m) and V (n×n) such that:
$$UAV = \begin{bmatrix} I_r & 0 \\ 0 & 0 \end{bmatrix}$$
This is the Smith normal form of A — the "nicest" matrix reachable by both row and column operations.
Row-Equivalent Matrices
A and B are row-equivalent (A ~ B) if A = UB for some invertible U. Every matrix is row-equivalent to a unique reduced row-echelon form (Theorem 2.5.4).
§2.7 LU-Factorization
Triangular factorization, algorithm, solving Ax = b efficiently
LU-Factorization
Definition 2.14 — LU-Factorization
$A = LU$ where $L$ is lower triangular and invertible, $U$ is upper triangular and row-echelon. Exists when A can be reduced to row-echelon form without row interchanges.
Theorem 2.7.1 — Existence
If $A$ can be lower-reduced (no row interchanges) to row-echelon matrix $U$, then $A = LU$ where $L$ is invertible lower triangular.
Theorem 2.7.2 — PLU-Factorization
For any $m \times n$ matrix $A$: there exists a permutation matrix $P$ such that $PA$ has an LU-factorization $PA = LU$ (PLU-factorization of $A$).
Theorem 2.7.3 — Uniqueness
If $A = LU$ and $\text{rank}(A) = m$ ($U$ has no zero rows), then $L$ and $U$ are uniquely determined.
LU-Algorithm (Theorem 2.7.1)
  1. Let A1 = A. Find the leading column c1 of A1; create the first leading 1 and zeros below (lower reduction only).
  2. Let A2 = rows 2–m of result. Find leading column c2 of A2; repeat.
  3. Continue for r steps (r = rank A) until row-echelon U is reached.
  4. Construct L: place c1, c2, …, cr at the bottom of the first r columns of Im.
Solving Ax = b via LU Given $A = LU$: $$\text{Step 1: Solve } L\mathbf{y} = \mathbf{b} \quad \text{(forward substitution)}$$ $$\text{Step 2: Solve } U\mathbf{x} = \mathbf{y} \quad \text{(back substitution)}$$ Then $A\mathbf{x} = L(U\mathbf{x}) = L\mathbf{y} = \mathbf{b}$. ✓ Advantage: If the same $A$ with different $\mathbf{b}$'s must be solved, factor once and forward/back-substitute repeatedly.
Lemma 2.7.1 — Triangular Closure
(1) Product of lower (upper) triangular matrices is lower (upper) triangular.
(2) A triangular n×n matrix is invertible iff all diagonal entries are nonzero; its inverse is also triangular of the same type.
§2.6 Linear Transformations
Definition, matrix of a transformation, rotations, reflections, projections
Linear Transformations
Definition 2.13 — Linear Transformation
$T: \mathbb{R}^n \to \mathbb{R}^m$ is a linear transformation if for all $\mathbf{x}, \mathbf{y} \in \mathbb{R}^n$ and all scalars $a$:
  • T1: $T(\mathbf{x} + \mathbf{y}) = T(\mathbf{x}) + T(\mathbf{y})$  (preserves addition)
  • T2: $T(a\mathbf{x}) = aT(\mathbf{x})$  (preserves scalar multiplication)
Immediate Consequences
$T(\mathbf{0}) = \mathbf{0}$, $T(-\mathbf{x}) = -T(\mathbf{x})$, and $T$ preserves all linear combinations (Theorem 2.6.1):
$$T(a_1\mathbf{x}_1 + \cdots + a_k\mathbf{x}_k) = a_1 T(\mathbf{x}_1) + \cdots + a_k T(\mathbf{x}_k)$$
Theorem 2.6.2 — Matrix of a Linear Transformation
Every linear transformation $T: \mathbb{R}^n \to \mathbb{R}^m$ is induced by a unique $m \times n$ matrix $A$. Specifically, A = [T(e1) T(e2) ⋯ T(en)] where e1,…,en are the standard basis vectors of Rn. Then:
$$T(\mathbf{x}) = A\mathbf{x} \text{ for all } \mathbf{x} \in \mathbb{R}^n$$
Theorem 2.6.3 — Composition = Matrix Product
If $T_A: \mathbb{R}^k \to \mathbb{R}^n$ and $T_B: \mathbb{R}^n \to \mathbb{R}^m$ are matrix transformations, their composite $T_B \circ T_A: \mathbb{R}^k \to \mathbb{R}^m$ is induced by the matrix $BA$ (product of the individual matrices).
Theorem 2.4.6 — Invertibility of Matrix Transformation
The matrix transformation TA: Rn → Rn (induced by A) has an inverse iff A is invertible. The inverse transformation is TA⁻¹, and the fundamental identities hold:
$T^{-1}[T(\mathbf{x})] = \mathbf{x}$ and $T[T^{-1}(\mathbf{x})] = \mathbf{x}$ for all $\mathbf{x}$.
Geometric Transformations of R²
Theorem 2.6.4 — Rotation by Angle θ
Counterclockwise rotation $R_\theta: \mathbb{R}^2 \to \mathbb{R}^2$ is linear with matrix:
$$R_\theta = \begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix}$$
Rπ/2 = [[0,−1],[1,0]]; Rπ = [[−1,0],[0,−1]].
Composition: $R_{\theta+\varphi} = R_\theta \circ R_\varphi$ — gives addition formulas for $\cos$ and $\sin$.
Theorem 2.6.5 — Reflection in y = mx
Reflection $Q_m: \mathbb{R}^2 \to \mathbb{R}^2$ in the line $y = mx$ is linear with matrix:
$$Q_m = \frac{1}{1+m^2} \begin{bmatrix} 1-m^2 & 2m \\ 2m & m^2-1 \end{bmatrix}$$
m = 0 (x-axis): [[1,0],[0,−1]];   y-axis: [[−1,0],[0,1]];   y = x (m=1): [[0,1],[1,0]].
Theorem 2.6.6 — Projection onto y = mx
Projection $P_m: \mathbb{R}^2 \to \mathbb{R}^2$ onto the line $y = mx$ is linear with matrix:
$$P_m = \frac{1}{1+m^2} \begin{bmatrix} 1 & m \\ m & m^2 \end{bmatrix}$$
Note $P_m \circ P_m = P_m$ (idempotent), so the matrix $A$ satisfies $A^2 = A$.

§2.2 Vectors in Rⁿ & Matrix Transformations
n-tuples, matrix transformation, identity and zero transformations
Vectors, Rⁿ, and the Matrix Transformation
Definition 2.4 — Rⁿ
$\mathbb{R}^n$ denotes the set of all ordered $n$-tuples $(a_1, a_2, \ldots, a_n)$ of real numbers. Elements are called vectors or n-vectors, written as rows or columns. Rn is closed under addition and scalar multiplication.
Definition 2.8 — Matrix Transformation
For any $m \times n$ matrix $A$, multiplication by $A$ defines a transformation $T_A: \mathbb{R}^n \to \mathbb{R}^m$ by $T_A(\mathbf{x}) = A\mathbf{x}$. This is the matrix transformation induced by A.
TransformationMatrix ANotes
Zero transformationZero matrix 0T(x) = 0 for all x
Identity transformation 1RⁿInT(x) = x for all x
Translation by wNot a matrix transform!T(x) = x + w (w ≠ 0) has no matrix
Theorem 2.2.5 — A Determined by Its Action
If $A$ and $B$ are $m \times n$ matrices such that $A\mathbf{x} = B\mathbf{x}$ for every $\mathbf{x} \in \mathbb{R}^n$, then $A = B$. Thus a matrix transformation is entirely determined by its action on Rn.
Geometric Examples (§2.2)
Reflection in x-axis: [[1,0],[0,−1]];   reflection in y=x: [[0,1],[1,0]];
Rotation 90° CCW: [[0,−1],[1,0]].
Applications: Input-Output & Markov Chains (§2.8–§2.9)
§2.8 — Input-Output (Leontief) Model
E = [eij] is the input-output matrix (eij = fraction of industry j's output consumed by industry i). A stochastic matrix has 0 ≤ eij ≤ 1 and each column sums to 1 (closed model). Equilibrium prices satisfy $E\mathbf{p} = \mathbf{p}$, i.e., $(I - E)\mathbf{p} = \mathbf{0}$.
Theorem 2.8.1 — Equilibrium Exists
For any stochastic matrix $E$, there is a nonzero $\mathbf{p} \geq \mathbf{0}$ with $E\mathbf{p} = \mathbf{p}$.
Theorem 2.8.2 — Open Model
Let E ≥ 0. Then $(I-E)$ is invertible and $(I-E)^{-1} \geq 0$ if and only if there exists $\mathbf{p} > \mathbf{0}$ with $\mathbf{p} > E\mathbf{p}$ (each industry produces more than it consumes = profitable).
§2.9 — Markov Chains (Definition 2.15)
A Markov chain is a system where the next state depends only on the present state, not history. The transition matrix P is stochastic; state vector at stage $k$ is $\mathbf{s}_k = P \cdot \mathbf{s}_{k-1} = P^k \cdot \mathbf{s}_0$.
Steady-State Vector
A state vector $\mathbf{q} \geq \mathbf{0}$ with $P\mathbf{q} = \mathbf{q}$ (sum of entries $= 1$) is the steady-state distribution. It is the eigenvector of $P$ for eigenvalue $\lambda = 1$.
Open Model — Production Equation $$(I - E)\mathbf{p} = \mathbf{d} \quad \text{where } \mathbf{d} = \text{demand vector (external sector)}$$ $$\mathbf{p} = (I - E)^{-1}\mathbf{d} \quad \text{(exists if each industry runs at profit)}$$
