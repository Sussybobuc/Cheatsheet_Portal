🔗 Relations

Definition

Relation from A to B: R ⊆ A × B

Notation: aRb or (a,b) ∈ R

Relation on A

R ⊆ A × A (from A to itself)

Representations

  • Set notation: list pairs
  • Matrix: rows/cols indexed by elements
  • Digraph: directed graph with edges
  • Arrow diagram: arrows between elements

Domain & Range

Domain: {a | ∃b: aRb}

Range: {b | ∃a: aRb}

R⁻¹ Inverse Relation

Definition

R⁻¹ = {(b,a) | (a,b) ∈ R}

Reverse all ordered pairs

Properties

  • (R⁻¹)⁻¹ = R (inverse of inverse)
  • R ⊆ S ⟹ R⁻¹ ⊆ S⁻¹
  • (R ∪ S)⁻¹ = R⁻¹ ∪ S⁻¹
  • (R ∩ S)⁻¹ = R⁻¹ ∩ S⁻¹

Composition

R ∘ S = {(a,c) | ∃b: aRb ∧ bSc}

Apply R first, then S

✓ Properties

Reflexive

∀a ∈ A: aRa (each relates to itself)

Irreflexive

∀a ∈ A: ¬(aRa) (no element relates to itself)

Symmetric

aRb ⟹ bRa (if a relates b, b relates a)

Antisymmetric

aRb ∧ bRa ⟹ a = b (at most one direction)

Transitive

aRb ∧ bRc ⟹ aRc (chains connect)

Symmetric ∧ Transitive ⟹ Reflexive may fail

≡ Equivalence Relations

Definition

Reflexive + Symmetric + Transitive

Equivalence Class

[a] = {x ∈ A | xRa} (all equivalent to a)

Partition

Equivalence classes partition A (disjoint, covering)

Examples

  • Equality (=) on any set
  • Congruence mod n: a≡b (mod n)
  • Same birthday
  • Homomorphic (graphs)
Number of equivalence classes = number of distinct [a]

≤ Partial Orders

Definition

Reflexive + Antisymmetric + Transitive

Notation: ≤ or ⪯

Poset (Partially Ordered Set)

Set with partial order relation

Total Order

Every two elements comparable: a ≤ b or b ≤ a

Examples

  • ≤ on ℝ (total)
  • ⊆ on power set (partial)
  • Divisibility ∣ (partial)
  • ≤ on products (lexicographic)
Partial ≠ Total unless comparable

📊 Hasse Diagrams

Purpose

Visualize partial order relations

Construction

  • Nodes = elements of poset
  • Edges = direct relations (no transitivity drawn)
  • Smaller elements below, larger above
  • Edges point upward

Example: Divisibility on {1,2,3,4,6,12}

     12
    / |  \
   6  4   (others)
   |  |
   3  2
    \ |
      1

Special Elements

Minimal: no element < it

Maximal: no element > it

Least: < everything | Greatest: > everything

⟲ Closure

Reflexive Closure

Add (a,a) for all a missing in R

r(R) = R ∪ {(a,a) | a ∈ A}

Symmetric Closure

Add (b,a) for each (a,b) in R

s(R) = R ∪ R⁻¹

Transitive Closure

Add (a,c) if path a→b→c exists

t(R) = R ∪ R² ∪ R³ ∪ ... (finite for finite A)

Computing Transitive Closure

Warshall's Algorithm: O(n³) time

t(R) is the minimum transitive relation containing R

🔍 Special Relations

Identity Relation

I = {(a,a) | a ∈ A}

Universal Relation

U = A × A (all pairs)

Empty Relation

∅ (no pairs)

Reflexive Relation

Contains identity: I ⊆ R ⊆ U

Function (Special Relation)

Each element in domain has exactly one output

🎯 Key Theorems

Equivalence ⟹ Partition

Each equivalence relation partitions domain

Each partition defines equivalence relation

Partial Order Properties

Must have all three: reflexive, antisymmetric, transitive

Total Order ⊂ Partial Order

All total orders are partial (extra: comparable)

Hasse Diagram Uniqueness

One unique Hasse diagram per poset

Relation satisfying 2 properties ≠ satisfying all 3
