Propositional Logic Quantifiers Inference Proofs

⊨ Propositions

A proposition is a declarative sentence that is either T (true) or F (false), but not both.

Valid: "2+2=4", "DC is capital of USA"
Invalid: "What time?", "x+1=2"

Propositional Variables

Denoted p, q, r, s, ... representing propositions. Truth values: T or F

Compound Propositions

Form new propositions by combining existing ones using logical operators (connectives).

∧ ∨ ¬ Operators

Negation: ¬p

¬p ≡ "not p" | Truth value is opposite of p

Conjunction: p ∧ q

p ∧ q ≡ "p AND q" | True only when both are true

Disjunction: p ∨ q

p ∨ q ≡ "p OR q" (inclusive) | False only when both are false

Exclusive Or: p ⊕ q

p ⊕ q ≡ "p XOR q" | True when exactly one is true

→ ↔ Conditionals

Conditional: p → q

p → q ≡ "if p then q" | False only when p=T and q=F

p = hypothesis (antecedent) | q = conclusion

Biconditional: p ↔ q

p ↔ q ≡ "p if and only if q" | True when both have same truth value

p ↔ q ≡ (p → q) ∧ (q → p)

Precedence (highest to lowest)

  1. Negation (¬)
  2. Conjunction (∧)
  3. Disjunction (∨)
  4. Conditional (→)
  5. Biconditional (↔)

↔ Conditional Forms

Standard Forms of p → q

  • "if p then q"
  • "p implies q"
  • "p only if q"
  • "q if p"
  • "q whenever p"
  • "p is sufficient for q"
  • "q is necessary for p"

Related Statements

  • Converse: q → p (NOT equivalent)
  • Inverse: ¬p → ¬q (NOT equivalent)
  • Contrapositive: ¬q → ¬p ✓ EQUIVALENT

∀ ∃ Quantifiers

Predicate P(x)

Statement with variable x; becomes proposition when x assigned value

Universal: ∀x P(x)

"for all x, P(x)" | True only if P(x) true for every x in domain

Existential: ∃x P(x)

"there exists x such that P(x)" | True if P(x) true for at least one x

¬(∀x P(x)) ≡ ∃x ¬P(x)
¬(∃x P(x)) ≡ ∀x ¬P(x)

≡ Equivalences

  • Idempotent: p∧p≡p, p∨p≡p
  • Commutative: p∧q≡q∧p
  • Associative: (p∧q)∧r≡p∧(q∧r)
  • Distributive: p∧(q∨r)≡(p∧q)∨(p∧r)
  • De Morgan's: ¬(p∧q)≡¬p∨¬q
  • Absorption: p∨(p∧q)≡p
  • Double Negation: ¬(¬p)≡p
  • Tautology: p∨¬p≡T
  • Contradiction: p∧¬p≡F

⊢ Proof Strategies

Direct Proof

Assume hypothesis, deduce conclusion through logical steps

Proof by Contrapositive

To prove p→q, prove ¬q→¬p (logically equivalent)

Proof by Contradiction

Assume ¬p and derive contradiction; conclude p must be true

Proof by Cases

Show statement true for all possible cases

Existence Proof

Constructive: build example | Non-constructive: show must exist

🔄 Nested Quantifiers

Order Matters!

∀x ∃y P(x,y) ≠ ∃y ∀x P(x,y)

First statement

"For every x, there exists y such that P(x,y)"

Second statement

"There exists y such that for all x, P(x,y)"

Negation of ∀x ∃y P: ∃x ∀y ¬P(x,y)
Negation of ∃x ∀y Q: ∀x ∃y ¬Q(x,y)

Scope Rules

  • ∀ and ∃ have equal precedence
  • Read left to right
  • Quantifier applies to everything following it

⟨x⟩ Variable Scope

Bound Variable

Variable within scope of quantifier: ∀x P(x) – x is bound

Free Variable

Variable not bound by quantifier: P(x) without quantifier – x is free

Domain Restrictions

∀x>0 P(x) means "for all positive x"

Domain Changes Truth Value

Same statement can be true for one domain, false for another

∀x∈ℕ (x² ≥ 0) is TRUE
∀x∈ℝ (x ≥ 0) is FALSE

⊨ Valid Arguments

Argument Structure

Premises (hypotheses) + logical steps → Conclusion

Valid Argument

Conclusion is true whenever ALL premises are true

Invalid Argument (Fallacy)

Counterexample where premises true but conclusion false

Common Fallacies

  • Affirming Consequent: q, p→q ⟹ p (INVALID)
  • Denying Antecedent: ¬p, p→q ⟹ ¬q (INVALID)
  • Begging the Question: Assuming conclusion

💡 Proof Examples

If n is odd, then n² is odd.

Direct Proof

Assume n is odd, so n=2k+1 for integer k. Then n²=(2k+1)²=4k²+4k+1=2(2k²+2k)+1, which is odd. ✓

If n² is even, then n is even.

Proof by Contrapositive

Prove: if n is odd, then n² is odd (shown above). Therefore, if n² is even, n must be even. ✓

√2 is irrational.

Proof by Contradiction

Assume √2 = p/q (p,q coprime). Then 2q²=p², so p² even, thus p even. Let p=2m. Then 2q²=4m², so q²=2m², making q even. But then both even, contradicting coprimality. ✓

🎯 Proof Writing

Structure

  1. State what you'll prove
  2. State assumptions explicitly
  3. Use logical steps (each justified)
  4. Conclude statement proven

Key Phrases

  • "Assume..." (for hypothesis)
  • "Suppose not..." (for contradiction)
  • "By definition..." (justify terms)
  • "Therefore..." (draw conclusion)
  • "This completes the proof" or "Q.E.D."
Pro Tip
Work backwards: start with conclusion, determine what suffices to prove it.

📋 Theorem vs Lemma

Theorem

Major result; often primary goal of proof

Lemma

Minor theorem used to prove a more important result

Corollary

Result that follows easily from theorem

Conjecture

Statement believed true but not yet proven

Note
Organizing proofs with lemmas makes complex results clearer!

🔑 Key Concepts

Tautology

Statement always true (T for all truth assignments)

Contradiction

Statement always false (F for all truth assignments)

Contingency

Statement sometimes true, sometimes false

Satisfiable

Exists at least one assignment making formula true

Unsatisfiable

No assignment makes formula true (contradiction)

p ∨ ¬p is TAUTOLOGY
p ∧ ¬p is CONTRADICTION
