Sample Space Conditional Expected Value Distributions

Ω Sample Space

Definition

S (or Ω) = set of all possible outcomes of experiment

|S| = number of possible outcomes

Event

Subset E ⊆ S (one or more outcomes)

Elementary: single outcome {x}

Compound: multiple outcomes

Examples

  • Coin: S = {H, T}, |S| = 2
  • Die: S = {1,2,3,4,5,6}, |S| = 6
  • Two dice: |S| = 36 ordered pairs
  • Cards: |S| = 52

P(E) Probability

Definition

P(E) = |E| / |S| (for equally likely outcomes)

Number of favorable outcomes / total outcomes

Properties

  • 0 ≤ P(E) ≤ 1 for any event E
  • P(S) = 1 (certain event)
  • P(∅) = 0 (impossible event)
  • P(Ē) = 1 − P(E) (complement)

Examples

Fair coin: P(H) = 1/2

Fair die: P(even) = 3/6 = 1/2

Two dice: P(sum=7) = 6/36 = 1/6

∪ ∩ Events

Union: E₁ ∪ E₂

"E₁ or E₂" (or both)

P(E₁ ∪ E₂) = P(E₁) + P(E₂) − P(E₁ ∩ E₂)

Intersection: E₁ ∩ E₂

"E₁ and E₂" (both occur)

Mutually Exclusive

E₁ ∩ E₂ = ∅ (can't both happen)

P(E₁ ∪ E₂) = P(E₁) + P(E₂)

Disjoint Events

Same as mutually exclusive

P(E₁ ∪ E₂ ∪ E₃) = P(E₁) + P(E₂) + P(E₃)
− P(E₁ ∩ E₂) − P(E₁ ∩ E₃) − P(E₂ ∩ E₃)
+ P(E₁ ∩ E₂ ∩ E₃)

| P(A|B) Conditional

Definition

P(A|B) = probability of A given B occurred

P(A|B) = P(A ∩ B) / P(B) (if P(B) > 0)

Intuition

Restrict sample space to B, then find A proportion

Law of Total Probability

If B₁, B₂, ..., Bₙ partition S:

P(A) = Σᵢ P(A|Bᵢ)·P(Bᵢ)

Bayes' Theorem

P(A|B) = P(B|A)·P(A) / P(B)

Reverse conditional probability

P(A ∩ B) = P(A|B) · P(B)

⊥ Independence

Definition

Events A and B independent if:

P(A ∩ B) = P(A) · P(B)

Or equivalently: P(A|B) = P(A)

Intuition

Occurrence of B doesn't affect probability of A

For Multiple Events

A₁, A₂, ..., Aₙ mutually independent if:

For any subset: P(∩ᵢ Aᵢ) = ∏ P(Aᵢ)

Examples

Two fair coin flips (each independent)

Rolling two dice

NOT: drawing cards without replacement

𝔼[X] Expected Value

Definition

E[X] = Σ x · P(X=x) (weighted average)

Properties

  • E[c] = c (constant)
  • E[X + Y] = E[X] + E[Y]
  • E[cX] = c · E[X]
  • If X, Y indep: E[XY] = E[X]E[Y]

Examples

Fair die: E[X] = (1+2+3+4+5+6)/6 = 3.5

Coin (H=1, T=0): E[X] = 0.5

Fair die doubled: E[2X] = 2·3.5 = 7

E[X] = Average outcome (long-run mean)

σ² Variance

Definition

Var(X) = E[(X − E[X])²] = E[X²] − (E[X])²

Measures spread around mean

Standard Deviation

σ = √Var(X)

Properties

  • Var(c) = 0 (constant has no variance)
  • Var(X+c) = Var(X) (shifting doesn't matter)
  • Var(cX) = c² · Var(X)
  • If indep: Var(X+Y) = Var(X) + Var(Y)
Larger variance = more spread

🎲 Distributions

Uniform Distribution

All n outcomes equally likely: P(X=k) = 1/n

E[X] = (n+1)/2 (for 1 to n)

Bernoulli (p)

Binary outcome: P(X=1)=p, P(X=0)=1−p

E[X] = p | Var(X) = p(1−p)

Binomial (n, p)

n independent Bernoulli trials

P(X=k) = C(n,k)·pᵏ·(1−p)ⁿ⁻ᵏ

E[X] = np | Var(X) = np(1−p)

Geometric (p)

Trials until first success

P(X=k) = (1−p)ᵏ⁻¹·p

E[X] = 1/p

📋 Examples

Coin Flips

P(3 heads in 5 flips) = C(5,3)·(1/2)⁵ = 10/32

Dice Roll

P(sum > 7 with 2 dice)

= P(8,9,10,11,12) = (5+6+5+4+3)/36 = 23/36

Conditional Example

P(B|A) given P(A)=0.3, P(B)=0.4, P(A∩B)=0.12

= 0.12 / 0.3 = 0.4

Bayes Example

Disease test: P(disease|+) given P(+|disease)=0.99

P(+|no disease)=0.01, P(disease)=0.001

💡 Counting for Probability

Permutations

P(n,k) = n!/(n−k)! (order matters)

Combinations

C(n,k) = n!/(k!(n−k)!) (order doesn't matter)

Usage

P(E) = (# favorable) / (# possible)

Use combinations for unordered outcomes

Example

P(2 from deck is pair) = C(4,2)·13 / C(52,2)
