⊆ Sets Basics

A set is an unordered collection of distinct objects (elements)

Notation

a ∈ A (a belongs to A) | a ∉ A (a doesn't belong)

Roster Method

V = {a, e, i, o, u} (list all elements)

Set-Builder

O = {x | x odd, x < 10} or {x ∈ Z⁺ | x is odd}

Special Sets

  • ℕ = {0, 1, 2, ...} (natural numbers)
  • ℤ = {..., -2, -1, 0, 1, 2, ...} (integers)
  • ℚ = {p/q | p, q ∈ ℤ, q ≠ 0} (rationals)
  • ℝ (reals) | ℂ (complex)
  • ∅ (empty set) | {∅} (set containing empty set)

⊂ Subset Relations

Subset: A ⊆ B

Every element of A is in B: ∀x(x ∈ A → x ∈ B)

Proper Subset: A ⊂ B

A ⊆ B and A ≠ B (A strictly contained in B)

Equality: A = B

A ⊆ B AND B ⊆ A (same elements)

∅ ⊆ S for any set S
S ⊆ S for any set S
Proof Technique
To prove A = B: show A ⊆ B, then show B ⊆ A

∪ ∩ Set Operations

Union: A ∪ B

All elements in A or B (or both): {x | x ∈ A ∨ x ∈ B}

Intersection: A ∩ B

Elements in both A and B: {x | x ∈ A ∧ x ∈ B}

Complement: Ā or A'

Elements in U but not in A: {x | x ∈ U, x ∉ A}

Difference: A − B

Elements in A but not B: {x | x ∈ A, x ∉ B} = A ∩ B̄

Symmetric Difference: A ⊕ B

Elements in A or B but not both: (A − B) ∪ (B − A)

⟨⟩ Set Laws & Properties

2ˢ Power Set

Definition

P(S) = set of all subsets of S (including ∅ and S itself)

Cardinality

If |S| = n, then |P(S)| = 2ⁿ

Example

S = {0, 1, 2}

P(S) = {∅, {0}, {1}, {2}, {0,1}, {0,2}, {1,2}, {0,1,2}}

|P(S)| = 2³ = 8

P(∅) = {∅}
P({a}) = {∅, {a}}

Used For

Enumerating all combinations, determining all possible states

× Cartesian Product

Definition

A × B = {(a, b) | a ∈ A, b ∈ B} (ordered pairs)

Key Property

Order matters: (a, b) ≠ (b, a) unless a = b

A × B ≠ B × A in general

Cardinality

|A × B| = |A| · |B|

Multiple Sets

A × B × C = {(a, b, c) | a ∈ A, b ∈ B, c ∈ C}

Aⁿ = A × A × ... × A (n times)

Example

A = {1, 2}, B = {a, b, c}

A × B = {(1,a), (1,b), (1,c), (2,a), (2,b), (2,c)}

ƒ Functions Basics

Definition

Function f: A → B assigns to each element a ∈ A exactly one element b ∈ B

f(a) = b means element a maps to element b

Terminology

  • Domain: set A (inputs)
  • Codomain: set B (possible outputs)
  • Range/Image: {f(a) | a ∈ A} (actual outputs)
  • Pre-image of b: {a | f(a) = b}

Functions vs Relations

Function: each input has exactly one output

Relation: input may have 0, 1, or multiple outputs

🔄 Function Types

Injective (One-to-One)

Different inputs map to different outputs: f(a) = f(b) → a = b

Pre-image of each element ≤ 1

Surjective (Onto)

Every element of codomain is mapped to: ∀b ∈ B ∃a ∈ A with f(a) = b

Range = Codomain

Bijective (One-to-One Correspondence)

Both injective AND surjective

Perfect pairing between A and B

If |A| = |B| finite, bijection exists iff injection exists iff surjection exists

Bijection ⟺ Bijection has inverse ƒ⁻¹: B → A

⟶ Function Operations

Composition: (ƒ ∘ g)(a) = ƒ(g(a))

Apply g first, then f | Note: not commutative

Inverse: ƒ⁻¹

If f: A → B bijection, then ƒ⁻¹: B → A

ƒ(ƒ⁻¹(b)) = b and ƒ⁻¹(ƒ(a)) = a

Image of Set S

ƒ(S) = {ƒ(a) | a ∈ S} (where S ⊆ A)

Pre-image of Set T

ƒ⁻¹(T) = {a ∈ A | ƒ(a) ∈ T} (where T ⊆ B)

ƒ: A → B is bijection ⟺ ƒ inverse exists

|A| Cardinality

Finite Cardinality

|A| = n means set A has exactly n distinct elements

Equal Cardinality

|A| = |B| iff there exists bijection between A and B

Infinite Cardinality

Countably Infinite: |A| = |ℕ| (can list all elements)

Uncountably Infinite: |A| > |ℕ| (cannot list all)

Examples

  • ℤ, ℚ are countably infinite (bijection with ℕ)
  • ℝ, ℂ are uncountably infinite
  • Power set P(ℕ) is uncountable
|A| ≤ |P(A)| for any set A
|P(A)| > |A| (Cantor's theorem)

📊 Sequences

Definition

Function f: ℤ⁺ → S (or ℕ → S) that lists elements in order

Usually written as {aₙ} where aₙ = f(n)

Types

  • Arithmetic: aₙ = a₁ + (n-1)d (constant difference d)
  • Geometric: aₙ = a₁ · rⁿ⁻¹ (constant ratio r)
  • Recursive: each term defined using previous terms

Summation: ∑

∑ᵏ₌₁ⁿ aₖ = a₁ + a₂ + ... + aₙ

Common Sums

∑ᵏ₌₁ⁿ k = n(n+1)/2

∑ᵏ₌₁ⁿ k² = n(n+1)(2n+1)/6

🔗 Relations

Definition

Relation from A to B: R ⊆ A × B (subset of Cartesian product)

(a, b) ∈ R means a is related to b

Relation on A

R ⊆ A × A (set related to itself)

Representation

  • Set notation: pairs listed or described
  • Matrix: rows = first elements, columns = second
  • Digraph: nodes for elements, arrows for relations
  • Table: rows and columns indexed

Function vs Relation

Every function is a relation (special case)

Not every relation is a function

📐 Interval Notation

Closed Interval: [a, b]

{x | a ≤ x ≤ b} (includes endpoints)

Open Interval: (a, b)

{x | a < x < b} (excludes endpoints)

Half-Open: [a, b), (a, b]

[a, b) = {x | a ≤ x < b} (includes left, excludes right)

Infinite Intervals

[a, ∞) = {x | x ≥ a}

(−∞, b) = {x | x < b}

(−∞, ∞) = ℝ

Length of [a, b] = b − a

💡 Applications

Database Design

Graph Theory

Vertices, edges as relations; adjacency via functions

Counting

Cardinality used in combinatorics (permutations, combinations)

Encoding

ASCII maps characters to integers (bijection)
