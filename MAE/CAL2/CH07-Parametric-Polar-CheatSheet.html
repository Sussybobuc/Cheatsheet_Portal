§7.1 Parametric Equations
Parametric Curves
Definition
A parametric curve is defined by:
x = f(t),   y = g(t),   t ∈ [α, β]
The variable t is the parameter.
Eliminating the Parameter
Solve one equation for t, substitute into the other to get y = f(x) (if possible).
  • Line: x = x₀+at, y = y₀+bt
  • Circle: x = r cos t, y = r sin t, t ∈ [0, 2π]
  • Ellipse: x = a cos t, y = b sin t
  • Cycloid: x = r(t−sin t), y = r(1−cos t)
Orientation
The direction of increasing t traces the orientation of the curve. A curve may be traversed multiple times as t varies.
Calculus with Parametric Curves
First Derivative dy/dx
If x = f(t), y = g(t) and dx/dt ≠ 0:
First Derivative $$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{g\'(t)}{f\'(t)}$$
Second Derivative d²y/dx²
Second Derivative $$\frac{d^2y}{dx^2} = \frac{\dfrac{d}{dt}\!\left(\dfrac{dy}{dx}\right)}{dx/dt}$$
Tangent Lines
  • Horizontal tangent: dy/dt = 0 and dx/dt ≠ 0
  • Vertical tangent: dx/dt = 0 and dy/dt ≠ 0
  • Singular point (cusp): dx/dt = dy/dt = 0
Example
x = t², y = t³ − 3t: dx/dt = 2t, dy/dt = 3t²−3
dy/dx = (3t²−3)/(2t)
Horizontal tangents: t = ±1 → points (1, ±2)
§7.2 Arc Length & Surface Area — Parametric
Arc Length (Parametric)
Theorem
Arc length of x = f(t), y = g(t), α ≤ t ≤ β:
Parametric Arc Length $$L = \int_\alpha^\beta \sqrt{\left(\frac{dx}{dt}\right)^2+\left(\frac{dy}{dt}\right)^2}\,dt$$
Area Under Parametric Curve $$A = \int_\alpha^\beta y(t)\,x\'(t)\,dt \quad\text{(curve traced left to right)}$$
Example — Circle
x = r cos t, y = r sin t, t ∈ [0, 2π]
dx/dt = −r sin t, dy/dt = r cos t
L = ∫02π√(r²sin²t + r²cos²t) dt = ∫02π r dt = 2πr ✓
Surface Area of Revolution (Parametric)
Surface Area — About x-axis $$S = 2\pi\int_\alpha^\beta y(t)\sqrt{\left(\tfrac{dx}{dt}\right)^2+\left(\tfrac{dy}{dt}\right)^2}\,dt$$
Surface Area — About y-axis $$S = 2\pi\int_\alpha^\beta x(t)\sqrt{\left(\tfrac{dx}{dt}\right)^2+\left(\tfrac{dy}{dt}\right)^2}\,dt$$
Pattern
Both surface area formulas follow: 2π · (distance from axis) · (arc length element ds)
where ds = √[(dx/dt)² + (dy/dt)²] dt
§7.3 Polar Coordinates
Polar System & Conversions
Polar Coordinates
A point is given as ( r, θ ) where r is the radial distance and θ is the angle.
Polar ↔ Rectangular $$x = r\cos\theta \qquad y = r\sin\theta$$ $$r^2 = x^2+y^2 \qquad \tan\theta = \frac{y}{x} \qquad r = \sqrt{x^2+y^2}$$
Multiple Representations
(r, θ) = (r, θ + 2πk) = (−r, θ + π) — each point has infinitely many polar representations.
Example Conversions
(3, π/4) → x = 3cos(π/4) = 3√2/2, y = 3√2/2
(−2, 3) → r = √13, θ = π + arctan(3/2) (Q III)

§7.4 Calculus in Polar Coordinates
Slopes & Tangent Lines in Polar
Converting to Parametric
For r = f(θ): x = r cos θ = f(θ)cos θ, y = r sin θ = f(θ)sin θ, with parameter θ.
Slope of Polar Curve $$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} \quad\text{where}\quad \frac{dy}{d\theta} = r'\sin\theta+r\cos\theta,\quad \frac{dx}{d\theta} = r'\cos\theta-r\sin\theta$$
Tangent Lines
  • Horizontal: dy/dθ = 0, dx/dθ ≠ 0
  • Vertical: dx/dθ = 0, dy/dθ ≠ 0
Example — Cardioid
r = 1 + cos θ: r′ = −sin θ
dy/dθ = −sin θ·sin θ + (1+cos θ)cos θ = cos θ + cos 2θ
Horizontal tangents: solve cos θ + cos 2θ = 0
Area & Arc Length in Polar
Area Enclosed by Polar Curve
Area swept out by r = f(θ) from θ = α to θ = β:
Polar Area $$A = \frac{1}{2}\int_\alpha^\beta [f(\theta)]^2\,d\theta = \frac{1}{2}\int_\alpha^\beta r^2\,d\theta$$
Area Between Two Polar Curves (router ≥ rinner) $$A = \frac{1}{2}\int_\alpha^\beta \bigl(r_{\text{outer}}^2 - r_{\text{inner}}^2\bigr)\,d\theta$$
Arc Length in Polar $$L = \int_\alpha^\beta \sqrt{r^2+\left(\frac{dr}{d\theta}\right)^2}\,d\theta$$
Example — Circle r = 2
A = (1/2)∫02π 4 dθ = (1/2)(4)(2π) = 4π ✓
Example — One Petal of r = cos 2θ
One petal: θ ∈ [−π/4, π/4]
A = (1/2)∫−π/4π/4 cos²(2θ) dθ = π/8
§7.5 Conic Sections in Polar Coordinates
Polar Equations of Conics
Focus-Directrix Definition
Eccentricity e: ratio of distance from focus to distance from directrix. One focus at pole.
Polar Conic — Horizontal Directrix $$r = \frac{ep}{1 \pm e\sin\theta}$$
Polar Conic — Vertical Directrix $$r = \frac{ep}{1 \pm e\cos\theta}$$
eConic
e = 0Circle
0 < e < 1Ellipse
e = 1Parabola
e > 1Hyperbola
Focal Parameter p
p = distance from focus to directrix (semi-latus rectum). To use: make the constant term in denominator = 1 by dividing; then the coefficient of sin/cos is e.
