§6.1 Power Series & Radius of Convergence
Power Series Definition
Definition
A power series centered at a:
General Form $$\sum_{n=0}^{\infty} c_n(x-a)^n = c_0 + c_1(x-a) + c_2(x-a)^2 + \cdots$$
Convergence Behavior
For a power series centered at a, exactly one of three cases holds:
(1) Converges only at x = a
(2) Converges absolutely for all x ∈ ℝ (R = ∞)
(3) ∃ R > 0: converges absolutely for |x−a| < R, diverges for |x−a| > R
Radius of Convergence R
Find R using the Ratio Test:
R = limn→∞ |cn/cn+1|   or   R = 1 / lim|cn|1/n
Interval of Convergence
Definition
The interval of convergence (IOC) is determined by R, then endpoint check:
Steps to Find IOC 1. Apply Ratio Test: find $R$
2. IOC is open interval $(a-R,\,a+R)$ to start
3. Test $x=a-R$ and $x=a+R$ separately
4. Include endpoints where series converges
Endpoint behaviorIOC form
Both diverge(a−R, a+R)
Left converges only[a−R, a+R)
Right converges only(a−R, a+R]
Both converge[a−R, a+R]
Example
Σ xⁿ/n: R = 1, IOC. At x=1: Σ1/n diverges. At x=−1: Σ(−1)ⁿ/n converges (AST). IOC = [−1, 1)
§6.2 Calculus with Power Series
Term-by-Term Differentiation & Integration
Theorem — Differentiation
If f(x) = Σ cn(x−a)n with radius R, then:

f′(x) = Σn=1∞ n·cn(x−a)n−1   (same R)
Theorem — Integration
∫f(x)dx = Σn=0∞ cn(x−a)n+1/(n+1) + C   (same R)
Endpoint Note
Differentiation may lose endpoint convergence; integration may gain it. The radius R stays the same for the interior.
Classic Example
From 1/(1−x) = Σxⁿ (|x|<1):
Differentiate: 1/(1−x)² = Σ n·xⁿ⁻¹
Integrate: −ln(1−x) = Σ xⁿ⁺¹/(n+1)
ln(1+x) = Σ (−1)ⁿ xⁿ⁺¹/(n+1)
Representing Functions as Power Series
Geometric Series Template
Base series: 1/(1−x) = Σn=0∞ xⁿ   (|x|<1)
Derived Series $$\frac{1}{1+x} = \sum_{n=0}^{\infty}(-1)^n x^n \qquad \frac{1}{1-x^2} = \sum_{n=0}^{\infty} x^{2n}$$ $$\frac{1}{1+x^2} = \sum_{n=0}^{\infty}(-1)^n x^{2n} \qquad \arctan x = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{2n+1}$$
Strategy: Algebraic Manipulation
Multiply series, substitute x → xⁿ, differentiate, or integrate to build new power series from known ones.
§6.3 Taylor & Maclaurin Series
Taylor Series Definition
Definition
The Taylor series for f centered at x = a:
Taylor Series $$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n = f(a)+f'(a)(x-a)+\frac{f''(a)}{2!}(x-a)^2+\cdots$$
Maclaurin Series (a = 0)
Maclaurin series:
Maclaurin Series $$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(0)}{n!}\,x^n$$
Taylor Polynomial
The nth partial sum Tn(x) is the nth Taylor polynomial — a polynomial approximation of f near a.

§6.3 Taylor's Theorem & Error Bounds
Taylor's Theorem with Remainder
Taylor's Theorem (Lagrange Form)
If f has n+1 continuous derivatives on I containing a and x, then:

f(x) = Tn(x) + Rn(x)

Rn(x) = f(n+1)(c) · (x−a)n+1 / (n+1)!
for some c between a and x.
Error Bound $$|R_n(x)| \leq \frac{M\,|x-a|^{n+1}}{(n+1)!} \quad\text{where } M = \max|f^{(n+1)}(t)|$$
Example
Error in ex ≈ T₃(x) near x=0, for x∈[0,1]:
|R₃| ≤ e¹ · |x|⁴/4! ≤ e/24 ≈ 0.113
Binomial Series & Applications
Binomial Series
For any real k and |x| < 1:
Binomial Series $$(1+x)^k = \sum_{n=0}^{\infty}\binom{k}{n}x^n = 1+kx+\frac{k(k-1)}{2!}x^2+\frac{k(k-1)(k-2)}{3!}x^3+\cdots$$
  • k = −1: (1+x)−1 = 1−x+x²−x³+… (geometric)
  • k = 1/2: √(1+x) = 1 + x/2 − x²/8 + x³/16 − …
  • k = −1/2: 1/√(1+x) = 1 − x/2 + 3x²/8 − …
Applications
Power series solve non-elementary integrals like ∫e−x²dx, ∫sin(x²)/x dx by substituting the series and integrating term by term.
