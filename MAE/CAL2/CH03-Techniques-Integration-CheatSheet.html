§3.1 Integration by Parts
IBP Formula & Derivation
Theorem — Integration by Parts
Derived from the product rule d(uv) = u dv + v du:

∫ u dv = uv − ∫ v du
LIATE Rule (choose u)
LIATE gives priority for choosing u:
  1. Logarithmic: ln x, logax
  2. Inverse trig: arcsin, arctan, etc.
  3. Algebraic: xn, polynomials
  4. Trigonometric: sin, cos, tan…
  5. Exponential: ex, ax
Example
∫ x ex dx: u = x, dv = exdx
du = dx, v = ex
= xex − ∫ ex dx = xex − ex + C = ex(x−1) + C
IBP Strategies & Special Cases
Repeated IBP (Tabular Method)
For ∫ xn ex dx (or trig): apply IBP n times. Use tabular method: alternate signs, differentiate u-column, integrate dv-column.
Tabular Example — ∫ x² ex dx
SignDiff (u)Int (dv)
+x²ex
−2xex
+2ex
−0ex
Result: ex(x²−2x+2) + C
Circular IBP (Algebraic Loop)
When IBP leads back to the original integral I:
I = (result) − k·I → solve: I = (result)/(1+k) + C
Example: ∫ exsin x dx
Definite Integral IBP $$\int_a^b u\,dv = \bigl[uv\bigr]_a^b - \int_a^b v\,du$$
§3.2 Trigonometric Integrals
∫ sinᵐx cosⁿx dx — Strategy
Three Cases
  • m odd: save sin x · dx for du; use cos²x = 1−sin²x for the rest; sub u = cos x
  • n odd: save cos x · dx for du; use sin²x = 1−cos²x; sub u = sin x
  • m,n both even: use half-angle identities:
    sin²x = (1−cos 2x)/2
    cos²x = (1+cos 2x)/2
Key Pythagorean Identities $$\sin^2 x+\cos^2 x=1 \qquad 1+\tan^2 x=\sec^2 x \qquad 1+\cot^2 x=\csc^2 x$$
Example — n odd
∫ sin²x cos³x dx = ∫ sin²x cos²x cos x dx
= ∫ sin²x(1−sin²x) cos x dx   [u = sin x]
= u³/3 − u⁵/5 + C = sin³x/3 − sin⁵x/5 + C
∫ tanᵐx secⁿx dx & Reduction Formulas
Strategy
  • n even: save sec²x for du; use tan²x = sec²x−1; sub u = tan x
  • m odd: save sec x tan x for du; use tan²x = sec²x−1; sub u = sec x
  • m even, n odd: use reduction formulas
Reduction Formulas $$\int \sin^n x\,dx = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}\int \sin^{n-2}x\,dx$$ $$\int \cos^n x\,dx = \frac{\cos^{n-1}x\sin x}{n} + \frac{n-1}{n}\int \cos^{n-2}x\,dx$$ $$\int \tan^n x\,dx = \frac{\tan^{n-1}x}{n-1} - \int \tan^{n-2}x\,dx$$ $$\int \sec^n x\,dx = \frac{\sec^{n-2}x\tan x}{n-1} + \frac{n-2}{n-1}\int \sec^{n-2}x\,dx$$
Special Values $$\int\tan x\,dx = \ln|\sec x|+C \qquad \int\sec x\,dx = \ln|\sec x+\tan x|+C$$ $$\int\cot x\,dx = \ln|\sin x|+C \qquad \int\csc x\,dx = -\ln|\csc x+\cot x|+C$$
§3.3 Trigonometric Substitution
Three Cases of Trig Substitution
Expression Substitution Identity Used Result θ Range
√(a²−x²) x = a sin θ, dx = a cos θ dθ 1−sin²θ = cos²θ a cos θ −π/2 ≤ θ ≤ π/2
√(a²+x²) x = a tan θ, dx = a sec²θ dθ 1+tan²θ = sec²θ a sec θ −π/2 < θ < π/2
√(x²−a²) x = a sec θ, dx = a sec θ tan θ dθ sec²θ−1 = tan²θ a tan θ 0 ≤ θ < π/2 or π ≤ θ < 3π/2
Back-substitution: Reference Triangle
Draw a right triangle: label sides based on substitution, then read off sin, cos, tan in terms of x. Reference triangle method ensures clean back-substitution.
Example — Case 1
∫ √(9−x²) dx: x = 3 sin θ, dx = 3 cos θ dθ, √(9−x²) = 3 cos θ
= ∫ 9 cos²θ dθ = (9/2)(θ + sin θ cos θ) + C
= (9/2)[arcsin(x/3) + (x/3)(√(9−x²)/3)] + C = (9/2)arcsin(x/3) + (x√(9−x²))/2 + C
§3.4 Partial Fraction Decomposition
Partial Fractions — Four Cases
Prerequisite
Degree of numerator must be < degree of denominator. If not, perform polynomial long division first.
Factor in denominatorPartial fraction term(s)
Linear: (ax+b)A/(ax+b)
Repeated linear: (ax+b)ⁿA₁/(ax+b) + A₂/(ax+b)² + … + Aₙ/(ax+b)ⁿ
Irreducible quadratic: (ax²+bx+c)(Ax+B)/(ax²+bx+c)
Repeated quadratic: (ax²+bx+c)ⁿ(A₁x+B₁)/(…) + … + (Aₙx+Bₙ)/(…)ⁿ
Check Irreducibility
ax²+bx+c is irreducible (over ℝ) when b²−4ac < 0.
Solving & Integrating Partial Fractions
Method
  1. Factor the denominator completely
  2. Write the partial fraction form
  3. Multiply both sides by the full denominator
  4. Expand and equate coefficients (or plug in strategic x values to find constants)
  5. Integrate each partial fraction
Integration of Each Type $$\frac{A}{ax+b} \;\to\; \frac{A}{a}\ln|ax+b|+C$$ $$\frac{A}{(ax+b)^n} \;\to\; \frac{-A}{(n-1)(ax+b)^{n-1}}+C \quad(n\geq 2)$$ $$\frac{Ax+B}{x^2+k^2} \;\to\; \frac{A}{2}\ln(x^2+k^2)+\frac{B}{k}\arctan\!\frac{x}{k}+C$$
Example
∫ 1/((x+1)(x+2)) dx = ∫[1/(x+1) − 1/(x+2)] dx
= ln|x+1| − ln|x+2| + C = ln|(x+1)/(x+2)| + C
§3.7 Improper Integrals
Type I: Infinite Limits
Definition
Type I improper integrals have infinite limits:
One Infinite Limit $$\int_a^{\infty} f(x)\,dx = \lim_{t\to\infty}\int_a^t f(x)\,dx \qquad \int_{-\infty}^b f(x)\,dx = \lim_{t\to-\infty}\int_t^b f(x)\,dx$$
Both Limits Infinite $$\int_{-\infty}^{\infty} f\,dx = \int_{-\infty}^c f\,dx + \int_c^{\infty} f\,dx \quad\text{(split at any }c\text{)}$$
p-integral Test
∫1∞ x−p dx converges if p > 1, diverges if p ≤ 1.
Classic Example
∫1∞ e−x dx = limt→∞[−e−x]1t = 0−(−e−1) = 1/e ✓ converges
Type II: Discontinuous Integrand
Definition
Type II improper integrals have a discontinuity:
Discontinuity at Right Endpoint b $$\int_a^b f(x)\,dx = \lim_{t\to b^-}\int_a^t f(x)\,dx$$
Discontinuity at Left Endpoint a $$\int_a^b f(x)\,dx = \lim_{t\to a^+}\int_t^b f(x)\,dx$$
Discontinuity at Interior Point c $$\int_a^b = \int_a^c + \int_c^b \quad\text{(both must converge)}$$
Caution: ∫−11 1/x dx ≠ 0 — it diverges! Always check for discontinuities inside [a, b].
Comparison Test for Improper Integrals
If 0 ≤ f(x) ≤ g(x) for x ≥ a:
• ∫a∞ g converges ⟹ ∫a∞ f converges
• ∫a∞ f diverges ⟹ ∫a∞ g diverges
