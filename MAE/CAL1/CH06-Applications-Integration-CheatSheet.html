§6.1 Area Between Curves
Area Between Two Curves
Setup
If f(x) ≥ g(x) on [a, b], the area between the curves is: A = ∫ₐᵇ [f(x) − g(x)] dx
Formulas $$A = \int_a^b [f(x)-g(x)]\,dx \quad \text{(vertical slices)}$$ $$A = \int_c^d [f(y)-g(y)]\,dy \quad \text{(horizontal slices)}$$
When Curves Cross
Split the integral at intersection points. On each subinterval, identify which function is on top. Or use: A = ∫ₐᵇ |f(x) − g(x)| dx
Example
Area between y=x² and y=x:
Intersect at x=0, x=1; x ≥ x² on [0,1]
A = ∫₀¹(x−x²)dx = [x²/2 − x³/3]₀¹ = 1/2 − 1/3 = 1/6
Area Strategy — Choosing x or y Integration
Integration by x vs y
Use vertical slices (dx) when functions are given as y = f(x) and top/bottom are easily identified.
Use horizontal slices (dy) when functions are naturally x = f(y) or when multiple pieces arise in x.
Example — Integrate by y
Region between y = x and x = y²:
Rewrite: x = y (right) and x = y² (left) on [0,1]
A = ∫₀¹(y − y²)dy = [y²/2 − y³/3]₀¹ = 1/6
Finding Intersection Points Set $f(x) = g(x)$ and solve for $x$. For $y$-integration: set $f(y)=g(y)$ and solve for $y$. Verify which curve is on top in each interval.
§6.2 Volumes by Slicing — Disk & Washer Methods
Disk Method
Setup
Rotate the region between y = f(x) and the x-axis around the x-axis. Each cross-section is a disk with radius R = f(x).
Disk Formula $$V = \pi\int_a^b [f(x)]^2\,dx \quad \text{(about }x\text{-axis)}$$ $$V = \pi\int_c^d [f(y)]^2\,dy \quad \text{(about }y\text{-axis)}$$
Example
y = √x, [0,4] rotated about x-axis:
V = π∫₀⁴ (√x)² dx = π∫₀⁴ x dx = π[x²/2]₀⁴ = 8π
Washer Method
Setup
Rotate region between f(x) ≥ g(x) ≥ 0 around the x-axis. Each slice is a washer with outer radius R = f(x) and inner radius r = g(x).
Washer Formula $$V = \pi\int_a^b \bigl([f(x)]^2-[g(x)]^2\bigr)\,dx \quad \text{(about }x\text{-axis)}$$ $$V = \pi\int_c^d \bigl([f(y)]^2-[g(y)]^2\bigr)\,dy \quad \text{(about }y\text{-axis)}$$
Example
Region between y=x² and y=x, rotated about x-axis:
R=x, r=x², [0,1]:
V=π∫₀¹(x²−x⁴)dx=π[x³/3−x⁵/5]₀¹=π(1/3−1/5)=2π/15
§6.3 Volumes by Cylindrical Shells
Shell Method
Setup
Use cylindrical shells. For rotation about the y-axis, integrate with respect to x. Shell at position x has radius x, height f(x), thickness dx.
Shell Formulas $$V = 2\pi\int_a^b x\,f(x)\,dx \quad \text{(about }y\text{-axis)}$$ $$V = 2\pi\int_c^d y\,f(y)\,dy \quad \text{(about }x\text{-axis)}$$
Example — Shell about y-axis
y = x − x², [0,1] rotated about y-axis:
V = 2π∫₀¹ x(x−x²)dx = 2π∫₀¹(x²−x³)dx
= 2π[x³/3−x⁴/4]₀¹ = 2π(1/3−1/4) = π/6
Disk/Washer vs Shell
Disk/Washer: slices perpendicular to axis of rotation.
Shells: slices parallel to axis of rotation.
Choose whichever avoids solving for the inverse function.
§6.4 Arc Length & Surface Area
Arc Length
Formula Derivation
Approximate curve with line segments; ds = √(dx² + dy²) = √(1 + (dy/dx)²) dx.
Arc Length Formulas $$L = \int_a^b \sqrt{1+[f'(x)]^2}\,dx$$ $$L = \int_c^d \sqrt{1+[g'(y)]^2}\,dy$$ $$L = \int_\alpha^\beta \sqrt{[x'(t)]^2+[y'(t)]^2}\,dt \quad \text{(parametric)}$$
Example
y = (2/3)x^(3/2) from x=0 to x=3:
y' = x^(1/2); 1+(y')²=1+x
L = ∫₀³√(1+x)dx = [(2/3)(1+x)^(3/2)]₀³ = 14/3
Surface Area of Revolution
Surface Area Formulas $$S = 2\pi\int_a^b f(x)\sqrt{1+[f'(x)]^2}\,dx \quad (f\ge 0)$$ $$S = 2\pi\int_c^d g(y)\sqrt{1+[g'(y)]^2}\,dy$$
Memory Aid
Surface area = ∫ 2π·(radius)·(arc length element)
Radius = distance from axis; ds = √(1+(dy/dx)²)dx
Example
y = √x on [0,4] about x-axis:
y'=1/(2√x), 1+(y')²=1+1/(4x)
S = 2π∫₀⁴√x·√(1+1/(4x))dx = 2π∫₀⁴√(x+1/4)dx
§6.5–6.6 Physical Applications
Work
Definition
Work W done by variable force F(x) over displacement from a to b: W = ∫ₐᵇ F(x) dx (units: joules = N·m)
Important Cases $$W = F\cdot d \quad \text{(constant force)}$$ $$F(x) = kx,\quad W = \int_0^d kx\,dx = \frac{kd^2}{2} \quad \text{(Hooke's Law)}$$ $$W = \rho g\int A(y)\cdot y_{\text{lift}}\,dy \quad \text{(lifting liquid)}$$
Example — Spring
Spring constant k=20 N/m, compress 0.3m from natural length:
W = ∫₀^0.3 20x dx = [10x²]₀^0.3 = 0.9 J
Hydrostatic Force & Pressure
Pressure & Force
Pressure at depth h: P = ρgh (ρ = fluid density, g = gravity).
Hydrostatic force on a vertical plate with width w(y) at depth (surface − y): F = ρg ∫_a^b (surface − y)·w(y) dy
Constants $\rho = 1000\,\text{kg/m}^3$ (SI), $62.4\,\text{lb/ft}^3$ (US) $g = 9.8\,\text{m/s}^2$ (SI), $32\,\text{ft/s}^2$ (US) $\rho g = 9800\,\text{N/m}^3$ (specific weight of water)
Example
Rectangular plate 2m wide, submerged from y=0 (top) to y=3m:
F = ρg∫₀³ y·2 dy = 9800·2·[y²/2]₀³ = 9800·9 ≈ 88,200 N
Moments & Center of Mass
Definition
For a lamina with density ρ(x,y) = ρ (uniform), center of mass (x̄, ȳ) of a planar region R between f(x) and g(x):
Formulas (Uniform Density) $$m = \rho\int_a^b[f(x)-g(x)]\,dx$$ $$M_y = \rho\int_a^b x[f(x)-g(x)]\,dx$$ $$M_x = \frac{\rho}{2}\int_a^b\bigl([f(x)]^2-[g(x)]^2\bigr)\,dx$$ $$\bar{x} = \frac{M_y}{m}, \quad \bar{y} = \frac{M_x}{m}$$ Centroid: same as above with $\rho = 1$
Example — Centroid
Region under y=x² from 0 to 1:
A=1/3; Mᵧ=∫₀¹x·x²dx=1/4; M_x=(1/2)∫₀¹x⁴dx=1/10
x̄=3/4, ȳ=3/10
Exponential Growth & Decay (§6.8)
Model
If dy/dt = ky (growth rate proportional to current value), the solution is: y(t) = y₀·e^(kt) where y₀ = y(0) = initial value and k = growth constant.
Key Applications $$y(t) = y_0 e^{kt}$$ $$k>0\text{: growth}; \quad k<0\text{: decay}$$ $$T_{1/2} = \frac{\ln 2}{|k|}, \quad T_{\text{double}} = \frac{\ln 2}{k}$$ $$A = Pe^{rt} \quad \text{(continuous compounding)}$$ $$T(t) = T_{\text{env}} + (T_0-T_{\text{env}})e^{kt}, \; k<0 \quad \text{(Newton cooling)}$$
Example
Bacteria doubles every 3h; initial count 100.
100·e^(3k) = 200 → k = ln2/3
At t=12h: y = 100·e^(12·ln2/3) = 100·2⁴ = 1600
