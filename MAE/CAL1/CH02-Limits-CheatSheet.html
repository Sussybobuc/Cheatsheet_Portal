§2.1–2.2 Intuitive Limit & One-Sided Limits
Informal Definition of a Limit
Definition (Informal)
lim(x→a) f(x) = L means f(x) gets arbitrarily close to L as x approaches a (but x ≠ a). The value f(a) need not equal L; f(a) may not even exist.
Notation $$\lim_{x\to a} f(x) = L$$ $$\lim_{x\to a^-} f(x) = L^- \quad \text{(left-hand limit)}$$ $$\lim_{x\to a^+} f(x) = L^+ \quad \text{(right-hand limit)}$$
Two-Sided Limit Theorem
lim(x→a) f(x) = L if and only if lim(x→a⁻) f(x) = L AND lim(x→a⁺) f(x) = L.
Example
f(x) = (x²−4)/(x−2) = x+2 for x≠2.
lim(x→2) f(x) = 4, even though f(2) is undefined.
One-Sided Limits & Non-Existence
One-Sided Limits
lim(x→a⁻) f(x) — x approaches a from the left (x < a).
lim(x→a⁺) f(x) — x approaches a from the right (x > a).
When a Limit Does NOT Exist
1. Left ≠ right one-sided limits
2. f(x) oscillates without settling (e.g., sin(1/x) as x→0)
3. f(x) → ±∞ (infinite limit)
Example — Jump Discontinuity
f(x) = { 1 if x < 0, −1 if x ≥ 0 }
lim(x→0⁻) = 1, lim(x→0⁺) = −1 → limit DNE at x=0
⚠ lim(x→0) sin(1/x) does not exist — the function oscillates infinitely fast near 0.
§2.3 The Limit Laws
Limit Laws (assuming both limits exist)
Algebraic Limit Laws $$\lim[f \pm g] = \lim f \pm \lim g$$ $$\lim[f\cdot g] = \lim f \cdot \lim g$$ $$\lim[f/g] = \frac{\lim f}{\lim g} \quad (\lim g \ne 0)$$ $$\lim[c\cdot f] = c\cdot\lim f$$ $$\lim[f^n] = (\lim f)^n$$ $$\lim[\sqrt[n]{f}] = \sqrt[n]{\lim f}$$
Direct Substitution Property $$\text{If }f\text{ is polynomial/rational, }a\in\text{domain:}$$ $$\lim_{x\to a} f(x) = f(a)$$
Example — Factoring
lim(x→3) (x²−9)/(x−3)
= lim(x→3) (x+3)(x−3)/(x−3)
= lim(x→3) (x+3) = 6
Techniques for Indeterminate Forms (0/0)
1. Factor and cancel   2. Multiply by conjugate   3. Simplify complex fractions   4. Use known limits (sin x/x)
Squeeze Theorem
Squeeze Theorem
If g(x) ≤ f(x) ≤ h(x) near a (not necessarily at a), and
lim(x→a) g(x) = lim(x→a) h(x) = L,
then lim(x→a) f(x) = L.
Classic Trig Limits $$\lim_{x\to 0}\frac{\sin x}{x} = 1$$ $$\lim_{x\to 0}\frac{1-\cos x}{x} = 0$$ $$\lim_{x\to 0}\frac{1-\cos x}{x^2} = \frac{1}{2}$$
Example
lim(x→0) x²·sin(1/x):
−x² ≤ x²sin(1/x) ≤ x², both → 0, so limit = 0.
Infinite Limits & Vertical Asymptotes
Infinite Limits
lim(x→a) f(x) = +∞ means f(x) increases without bound as x→a.
lim(x→a) f(x) = −∞ means f(x) decreases without bound.
Vertical Asymptote
x = a is a vertical asymptote if lim(x→a⁺) or lim(x→a⁻) is ±∞.
Examples $$\lim_{x\to 0^+}\frac{1}{x}=+\infty, \quad \lim_{x\to 0^-}\frac{1}{x}=-\infty$$ $$\lim_{x\to 0}\frac{1}{x^2}=+\infty \quad \text{(both sides)}$$ $$\lim_{x\to(\pi/2)^-}\tan x = +\infty$$
⚠ lim(x→0) 1/x = DNE (one-sided limits disagree: +∞ ≠ −∞).
§2.4 Continuity
Continuity at a Point
Definition
f is continuous at x = a if:
1. f(a) is defined
2. lim(x→a) f(x) exists
3. lim(x→a) f(x) = f(a)
Types of Discontinuity
Removable: limit exists but ≠ f(a), or f(a) undefined.
Jump: one-sided limits exist but are unequal.
Infinite: at least one one-sided limit is ±∞.
Oscillatory: limit does not exist (e.g., sin(1/x)).
Example
f(x) = (x²−1)/(x−1): removable discontinuity at x=1.
Redefine f(1) = 2 to make it continuous.
Continuity Theorems & IVT
Continuity of Standard Functions
Polynomials, rational, trig, exponential, and logarithmic functions are continuous on their domains. Sums, differences, products, quotients (denom ≠ 0), and compositions of continuous functions are continuous.
Intermediate Value Theorem (IVT)
If f is continuous on [a, b] and k is between f(a) and f(b), then there exists c ∈ (a, b) such that f(c) = k.
Example — IVT Application
f(x) = x³ − x − 1: f(1) = −1 < 0, f(2) = 5 > 0.
IVT guarantees a root in (1, 2).
§2.3, §2.6 Limits at Infinity & Precise Definition
Limits at Infinity & Horizontal Asymptotes
Definition
lim(x→∞) f(x) = L means f(x) → L as x grows without bound.
y = L is a horizontal asymptote.
Key Limits at Infinity $$\lim_{x\to\infty}\frac{1}{x^n}=0 \quad (n>0)$$ $$\lim_{x\to\infty}e^x=\infty, \quad \lim_{x\to-\infty}e^x=0$$ $$\lim_{x\to\infty}\ln x=\infty, \quad \lim_{x\to 0^+}\ln x=-\infty$$ $$\lim_{x\to\infty}\!\left(1+\frac{1}{x}\right)^x = e$$
Rational Function at Infinity
Divide numerator and denominator by the highest power of x in the denominator.
Example
lim(x→∞) (3x²+2x)/(2x²−5)
= lim(x→∞) (3 + 2/x)/(2 − 5/x²) = 3/2
Epsilon-Delta (Precise) Definition
Formal Definition (§2.5)
lim(x→a) f(x) = L means: for every ε > 0, there exists δ > 0 such that if 0 < |x − a| < δ, then |f(x) − L| < ε.
Symbolic Form $$\forall\,\varepsilon>0,\;\exists\,\delta>0:$$ $$0<|x-a|<\delta \;\Rightarrow\; |f(x)-L|<\varepsilon$$
Example — Prove lim(x→3) 2x = 6
Given ε > 0, need |2x − 6| < ε when |x−3| < δ.
|2x − 6| = 2|x − 3| < 2δ. Set δ = ε/2. ✓
Note
δ depends on ε (and possibly on a). The key idea: we can make outputs arbitrarily close by restricting inputs.
