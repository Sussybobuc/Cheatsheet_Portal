§4.2 Linear Approximation & Differentials
Linear Approximation (Linearization)
Definition
The linearization of f at a is: L(x) = f(a) + f'(a)(x − a) For x near a: f(x) ≈ L(x).
Useful Standard Linearizations (near $x = 0$) $$(1+x)^n \approx 1 + nx$$ $$\sin x \approx x, \quad \cos x \approx 1$$ $$e^x \approx 1 + x, \quad \ln(1+x) \approx x$$ $$\tfrac{1}{1+x} \approx 1 - x$$
Example
Approximate √4.1: f(x)=√x, a=4, f'(x)=1/(2√x)
L(x) = 2 + (1/4)(x−4); L(4.1) = 2 + 0.025 = 2.025
(Exact: √4.1 ≈ 2.0248)
Differentials
Definition
If y = f(x), the differential dy is defined as: dy = f'(x) dx where dx is an independent variable (the change in x). Δy ≈ dy for small Δx = dx.
Error Estimation $$\Delta y = f(x+\Delta x) - f(x) \quad \text{(exact)}$$ $$dy = f'(x)\,dx \quad \text{(approximate)}$$ $$\text{Relative error: }\frac{dy}{y} = \frac{f'(x)\,dx}{f(x)}$$
Example
V = (4/3)πr³, r=5, dr=0.1:
dV = 4πr²dr = 4π(25)(0.1) = 10π ≈ 31.4 cm³
§4.3–4.4 Extrema & Mean Value Theorem
Extrema & Critical Points
Definitions
Absolute (global) max/min: largest/smallest value on entire domain.
Local (relative) max/min: largest/smallest in some open interval.
Critical point c: f'(c) = 0 OR f'(c) does not exist.
Extreme Value Theorem
If f is continuous on [a, b], then f attains both an absolute maximum and minimum on [a, b].
Fermat's Theorem
If f has a local extremum at c and f'(c) exists, then f'(c) = 0.
Closed Interval Method 1. Find all critical points in $(a, b)$ 2. Evaluate $f$ at critical points and endpoints $a$, $b$ 3. Largest value = abs max; smallest = abs min
Rolle's & Mean Value Theorems
Rolle's Theorem
If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then there exists c ∈ (a,b) such that f'(c) = 0.
Mean Value Theorem (MVT)
If f is continuous on [a,b] and differentiable on (a,b), then there exists c ∈ (a,b) such that: $$f'(c) = \frac{f(b)-f(a)}{b-a}$$ (instantaneous rate of change = average rate of change)
Example
f(x)=x²−x on [0,2]: f'(c) = (f(2)−f(0))/(2−0) = 2/2 = 1
f'(x) = 2x−1 = 1 → x = 1. c = 1 ∈ (0,2) ✓
Corollaries of MVT
f'(x) = 0 on (a,b) → f constant on [a,b]
f'(x) > 0 on (a,b) → f increasing on [a,b]
f'(x) < 0 on (a,b) → f decreasing on [a,b]
§4.5 Shape of the Graph
First & Second Derivative Tests
First Derivative Test
At a critical point c where f' changes sign:
• − to +: local minimum
• + to −: local maximum
• no sign change: neither
Second Derivative Test
At critical point c where f'(c) = 0:
f''(c) > 0 → local minimum (concave up)
f''(c) < 0 → local maximum (concave down)
f''(c) = 0 → inconclusive (use 1st derivative test)
Concavity & Inflection Points
Concave up: f''(x) > 0; concave down: f''(x) < 0.
Inflection point: f'' changes sign at c.
Curve Sketching Summary
Systematic Curve Sketching Steps 1. Domain of $f$ 2. Intercepts ($x$: $f=0$; $y$: $x=0$) 3. Symmetry (even/odd) 4. Asymptotes (vertical, horizontal, oblique) 5. Intervals of increase/decrease (sign of $f'$) 6. Local extrema (first or second derivative test) 7. Concavity (sign of $f''$) 8. Inflection points ($f''$ changes sign) 9. Sketch using all gathered info
Example
f(x) = x³ − 3x: f'(x) = 3x²−3 = 3(x−1)(x+1)
Critical pts: x=±1; f''(x)=6x
f''(1)=6>0→local min; f''(−1)=−6<0→local max
Inflection at x=0 (f''=0 changes sign)
§4.7–4.10 Optimization, L'Hôpital, Newton, Antiderivatives
Applied Optimization
Optimization Strategy 1. Identify quantity $Q$ to maximize/minimize 2. Write $Q$ as function of one variable (use constraints) 3. Find domain 4. Find critical points ($Q'=0$ or $Q'$ DNE) 5. Use EVT or first/second derivative test to classify 6. Answer in context with units
Classic Example — Rectangle in Circle
Rectangle inscribed in circle r=5.
x²+y²=25; A = 4xy (total area)
y = √(25−x²); A(x) = 4x√(25−x²)
A'=0 → x=5/√2; Max area = 50 sq units
L'Hôpital's Rule
L'Hôpital's Rule
If lim(x→a) f(x)/g(x) gives 0/0 or ∞/∞, and f, g are differentiable near a (g'≠0 near a), then: lim(x→a) f(x)/g(x) = lim(x→a) f'(x)/g'(x) (Applies also for x→∞, one-sided limits)
Indeterminate Forms & Transformations $$\tfrac{0}{0},\;\tfrac{\infty}{\infty}: \text{ apply L'H directly}$$ $$0\cdot\infty: \text{ write as }\tfrac{0}{1/\infty}\text{ or }\tfrac{\infty}{1/0}$$ $$\infty-\infty: \text{ common denominator or factor}$$ $$0^0,\;1^\infty,\;\infty^0: \text{ take }\ln,\text{ use }0\cdot\infty\text{ form}$$
Examples
lim(x→0) sin x/x = lim cos x/1 = 1
lim(x→∞) x/eˣ = lim 1/eˣ = 0
lim(x→0⁺) x·ln x: ln x/(1/x) → (1/x)/(−1/x²) = −x → 0
Newton's Method
Algorithm
Newton's method approximates roots of f(x)=0: $$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$ Starting from initial guess $x_0$, iterate until $|x_{n+1}-x_n| < \text{tol}$.
Example — Approximate √2
f(x) = x²−2, f'(x) = 2x, x₀ = 1
x₁ = 1 − (1−2)/2 = 1.5
x₂ = 1.5 − (2.25−2)/3 = 1.4167
x₃ ≈ 1.41422 (≈ √2 = 1.41421...)
⚠ Newton's method may fail if x₀ is far from root, f'(xₙ)≈0, or the sequence diverges/cycles.
Antiderivatives
Definition
F is an antiderivative of f on I if F'(x) = f(x) for all x ∈ I. If F is one antiderivative, the general antiderivative is F(x) + C (constant of integration).
Basic Antiderivative Formulas $$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n\ne -1)$$ $$\int \frac{1}{x}\,dx = \ln|x| + C$$ $$\int e^x\,dx = e^x + C$$ $$\int \sin x\,dx = -\cos x + C$$ $$\int \cos x\,dx = \sin x + C$$ $$\int \sec^2 x\,dx = \tan x + C$$ $$\int k\,dx = kx + C$$
Example — Initial Value Problem
f'(x) = 3x²−2x+1, f(0) = 4
f(x) = x³ − x² + x + C; f(0)=C=4
f(x) = x³ − x² + x + 4
