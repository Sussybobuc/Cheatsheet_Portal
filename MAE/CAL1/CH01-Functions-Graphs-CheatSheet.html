§1.1 Function Fundamentals
Definition of a Function
Definition
A function f from set A to set B is a rule that assigns to each element x in A exactly one element y = f(x) in B. A is the domain, B contains the range.
Notation $$f: A \to B, \quad y = f(x)$$ $$\text{domain: all valid } x \text{ values}$$ $$\text{range: } \{f(x) \mid x \in \text{domain}\}$$
Vertical Line Test
A curve in the xy-plane is the graph of a function if and only if no vertical line intersects the curve more than once.
Example
f(x) = √(x − 2): domain = [2, ∞), range = [0, ∞)
g(x) = 1/(x² − 1): domain = ℝ \ {−1, 1}
Domain & Range Techniques
Definition
The natural domain is the largest subset of ℝ for which f(x) is defined. Restrict domain when: denominator = 0, even root of negative, log of non-positive.
Domain Rules $$\text{Polynomials: domain} = \mathbb{R}$$ $$\sqrt{g(x)}: \quad g(x) \ge 0$$ $$\tfrac{1}{g(x)}: \quad g(x) \ne 0$$ $$\ln(g(x)): \quad g(x) > 0$$ $$\tfrac{1}{\sqrt{g(x)}}: \quad g(x) > 0$$
Example
f(x) = √(4 − x²): need 4 − x² ≥ 0 → x² ≤ 4 → domain = [−2, 2]
range = [0, 2]
Note
To find the range, solve y = f(x) for x and determine which y-values give valid x-values, or use the graph.
§1.2 Basic Classes of Functions
Linear & Polynomial Functions
Linear Function
f(x) = mx + b. Slope m = (y₂−y₁)/(x₂−x₁). y-intercept b, x-intercept −b/m.
Polynomial
p(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₁x + a₀, degree n, leading coeff aₙ ≠ 0.
Key Forms $$y = mx + b \quad \text{(slope-intercept)}$$ $$y - y_1 = m(x - x_1) \quad \text{(point-slope)}$$ $$f(x) = ax^2 + bx + c \quad \text{(quadratic)}$$ $$f(x) = a(x-h)^2 + k \quad \text{(vertex form)}$$
Example
f(x) = −x² + 4x − 3 = −(x−2)² + 1
Vertex: (2, 1), opens downward (a = −1 < 0)
Rational & Power Functions
Rational Function
r(x) = p(x)/q(x), where p, q are polynomials, q ≢ 0. Vertical asymptotes where q(x) = 0 (and p ≠ 0).
Power Function
f(x) = xⁿ for integer n, or f(x) = x^(p/q) for rational exponent. Root functions f(x) = x^(1/n) = ⁿ√x.
Horizontal Asymptotes of $r(x) = p(x)/q(x)$ $$\deg p < \deg q \;\Rightarrow\; y = 0$$ $$\deg p = \deg q \;\Rightarrow\; y = \frac{\text{lead}(p)}{\text{lead}(q)}$$ $$\deg p > \deg q \;\Rightarrow\; \text{no H.A. (oblique)}$$
⚠ Common Mistake: √(x²) = |x|, not x. Always use absolute value for even roots of squared expressions.
Algebraic & Transcendental Functions
Algebraic Functions
Built from polynomials using +, −, ×, ÷, and nth roots. Includes radicals like f(x) = √(x² + 1).
Transcendental Functions
Functions that are NOT algebraic: trigonometric, exponential/logarithmic, inverse trig.
Standard Function Summary $$f(x) = mx+b \quad \text{(linear)}$$ $$f(x) = ax^2+bx+c \quad \text{(quadratic)}$$ $$f(x) = ax^3+\cdots \quad \text{(cubic)}$$ $$f(x) = |x| \quad \text{(absolute value)}$$ $$f(x) = \sqrt{x} \quad \text{(square root)}$$
Even & Odd Symmetry
Definition
Even: f(−x) = f(x) for all x. Symmetric about the y-axis.
Odd: f(−x) = −f(x) for all x. Symmetric about the origin.
Quick Test (replace $x$ with $-x$) $$f(-x) = f(x) \;\Rightarrow\; \text{EVEN}$$ $$f(-x) = -f(x) \;\Rightarrow\; \text{ODD}$$ $$\text{otherwise} \;\Rightarrow\; \text{NEITHER}$$
Example
f(x) = x⁴ − 3x²: f(−x) = x⁴ − 3x² = f(x) → EVEN
g(x) = x³ − 5x: g(−x) = −x³ + 5x = −g(x) → ODD
h(x) = x² + x: neither (h(−x) ≠ ±h(x))
§1.3 Trigonometric Functions
Trig Definitions & Key Identities
Unit Circle Definition
For angle θ (radians), point (cos θ, sin θ) on unit circle x² + y² = 1.
tan θ = sin θ / cos θ,   cot θ = cos θ / sin θ
sec θ = 1/cos θ,   csc θ = 1/sin θ
Pythagorean Identities $$\sin^2\theta + \cos^2\theta = 1$$ $$1 + \tan^2\theta = \sec^2\theta$$ $$1 + \cot^2\theta = \csc^2\theta$$
Key Values (radians) $$\begin{array}{c|ccccc}\theta & 0 & \tfrac{\pi}{6} & \tfrac{\pi}{4} & \tfrac{\pi}{3} & \tfrac{\pi}{2}\\\hline \sin\theta & 0 & \tfrac{1}{2} & \tfrac{\sqrt{2}}{2} & \tfrac{\sqrt{3}}{2} & 1\\ \cos\theta & 1 & \tfrac{\sqrt{3}}{2} & \tfrac{\sqrt{2}}{2} & \tfrac{1}{2} & 0\\ \tan\theta & 0 & \tfrac{1}{\sqrt{3}} & 1 & \sqrt{3} & \text{undef}\end{array}$$
Period & Amplitude ($y = A\sin(Bx+C)+D$) $$\text{Period} = \frac{2\pi}{|B|}, \quad \text{Amplitude} = |A|$$ $$\text{Phase shift} = -\frac{C}{B}, \quad \text{Vertical shift} = D$$
Note
Period of sin/cos = 2π; period of tan/cot = π. Trig functions are NOT one-to-one on their natural domains — must restrict domain to define inverses.
§1.4–1.5 Inverse, Exponential & Logarithmic Functions
Inverse Functions
Definition
f⁻¹ is the inverse of f if f(f⁻¹(x)) = x and f⁻¹(f(x)) = x. Requires f to be one-to-one (injective).
Horizontal Line Test
f has an inverse iff no horizontal line intersects its graph more than once.
Key Facts $$\text{domain}(f^{-1}) = \text{range}(f)$$ $$\text{range}(f^{-1}) = \text{domain}(f)$$ $$\text{Graph of }f^{-1}: \text{reflect }f\text{ across }y = x$$
Example
f(x) = 2x − 3; solve y = 2x − 3 for x: x = (y+3)/2
So f⁻¹(x) = (x+3)/2. Check: f(f⁻¹(x)) = 2·(x+3)/2 − 3 = x ✓
⚠ Note: f⁻¹(x) ≠ 1/f(x). The −1 is notation for inverse, NOT a negative exponent.
Inverse Trig Functions
Restricted Domains for Inverses
sin⁻¹(x) = arcsin(x):   domain [−1,1], range [−π/2, π/2]
cos⁻¹(x) = arccos(x):   domain [−1,1], range [0, π]
tan⁻¹(x) = arctan(x):   domain ℝ, range (−π/2, π/2)
Useful Identities $$\sin(\sin^{-1}x) = x \quad x \in [-1,1]$$ $$\sin^{-1}(\sin x) = x \quad x \in \bigl[-\tfrac{\pi}{2},\tfrac{\pi}{2}\bigr]$$ $$\tan^{-1}x + \tan^{-1}\tfrac{1}{x} = \tfrac{\pi}{2} \quad (x>0)$$ $$\sin^{-1}x + \cos^{-1}x = \tfrac{\pi}{2}$$
Example
sin⁻¹(√3/2) = π/3 (since sin(π/3) = √3/2)
tan⁻¹(1) = π/4 (since tan(π/4) = 1)
Exponential Functions
Definition
f(x) = bˣ, base b > 0, b ≠ 1. Domain = ℝ, range = (0, ∞). Natural base e ≈ 2.71828; f(x) = eˣ is the natural exponential.
Laws of Exponents $$b^x \cdot b^y = b^{x+y}$$ $$b^x / b^y = b^{x-y}$$ $$(b^x)^y = b^{xy}$$ $$(ab)^x = a^x b^x$$ $$b^{-x} = \tfrac{1}{b^x}$$
Growth vs Decay
b > 1: exponential growth (increasing)
0 < b < 1: exponential decay (decreasing)
Horizontal asymptote: y = 0
Logarithmic Functions
Definition
logᵦ(x) = y iff bʸ = x. The natural logarithm ln(x) = logₑ(x). Domain = (0, ∞), range = ℝ.
Logarithm Laws $$\ln(xy) = \ln x + \ln y$$ $$\ln(x/y) = \ln x - \ln y$$ $$\ln(x^n) = n\ln x$$ $$\log_b x = \frac{\ln x}{\ln b} \quad \text{(change of base)}$$ $$\ln(e^x) = x, \quad e^{\ln x} = x$$
Example
Solve: e^(2x) = 5 → 2x = ln 5 → x = (ln 5)/2 ≈ 0.805
Solve: ln(x+1) = 3 → x+1 = e³ → x = e³ − 1 ≈ 19.09
§1.2 Transformations & Combinations
Graph Transformations
Transformations of y = f(x)
Starting from a base function, transformations shift, stretch, or reflect the graph.
Transformation Rules $$y = f(x)+c \quad \text{(shift up }c\text{)}$$ $$y = f(x)-c \quad \text{(shift down }c\text{)}$$ $$y = f(x+c) \quad \text{(shift left }c\text{)}$$ $$y = f(x-c) \quad \text{(shift right }c\text{)}$$ $$y = c\,f(x) \quad \text{(vertical stretch/compress)}$$ $$y = f(cx) \quad \text{(horizontal compress/stretch)}$$ $$y = -f(x) \quad \text{(reflect over }x\text{-axis)}$$ $$y = f(-x) \quad \text{(reflect over }y\text{-axis)}$$
Example
y = 2√(x−3) + 1: shift right 3, stretch by 2, shift up 1
Composition & Piecewise Functions
Composition
(f ∘ g)(x) = f(g(x)). Apply g first, then f. Domain: all x in domain of g such that g(x) is in domain of f.
Arithmetic Combinations $$(f+g)(x) = f(x)+g(x)$$ $$(f-g)(x) = f(x)-g(x)$$ $$(f\cdot g)(x) = f(x)\cdot g(x)$$ $$\bigl(\tfrac{f}{g}\bigr)(x) = \tfrac{f(x)}{g(x)},\; g(x)\ne 0$$
Piecewise Function
Defined by different formulas on different parts of the domain.
Example Piecewise $$f(x) = \begin{cases} x^2 & x < 0 \\ 2x+1 & x \ge 0 \end{cases}$$ $$f(-2) = (-2)^2 = 4, \quad f(3) = 2(3)+1 = 7$$
Note
Check continuity at boundary points of piecewise functions by evaluating left and right limits.
