§3.1–3.2 Definition of the Derivative
Derivative at a Point
Definition
The derivative of f at a is: f'(a) = lim(h→0) [f(a+h) − f(a)] / h provided the limit exists. Equivalently: f'(a) = lim(x→a) [f(x) − f(a)] / (x − a)
Geometric Interpretation
f'(a) = slope of the tangent line to y = f(x) at (a, f(a)).
Tangent line equation: y − f(a) = f'(a)(x − a)
Example — From Definition
f(x) = x²: f'(a) = lim(h→0) [(a+h)²−a²]/h
= lim(h→0) [2ah + h²]/h = lim(h→0)(2a+h) = 2a
So f'(x) = 2x.
Derivative as a Function
Definition
f'(x) = lim(h→0) [f(x+h) − f(x)] / h. f is differentiable at a if f'(a) exists; differentiable on (a,b) if at every point.
Differentiability Implies Continuity
If f is differentiable at a, then f is continuous at a. (Converse is FALSE.)
Notations for the Derivative
f'(x), y', dy/dx, d/dx[f(x)], Df(x)
⚠ f(x) = |x| is continuous at 0 but NOT differentiable there — the left and right slopes are −1 and +1.
§3.3 Basic Differentiation Rules
Fundamental Differentiation Rules
Basic Rules $$\frac{d}{dx}[c] = 0 \quad \text{(constant)}$$ $$\frac{d}{dx}[x^n] = nx^{n-1} \quad \text{(power rule)}$$ $$\frac{d}{dx}[c\cdot f] = c\cdot f'$$ $$\frac{d}{dx}[f \pm g] = f' \pm g'$$ $$\frac{d}{dx}[e^x] = e^x, \qquad \frac{d}{dx}[\ln x] = \frac{1}{x}$$
Product & Quotient Rules $$\frac{d}{dx}[fg] = f'g + fg'$$ $$\frac{d}{dx}\!\left[\frac{f}{g}\right] = \frac{f'g - fg'}{g^2}$$
Example — Product Rule
d/dx[x²·eˣ] = 2x·eˣ + x²·eˣ = xeˣ(x+2)
Example — Quotient Rule
d/dx[(x²+1)/(x−1)]
= [(2x)(x−1) − (x²+1)(1)] / (x−1)²
= (x²−2x−1) / (x−1)²
§3.5 Derivatives of Trigonometric Functions
All Six Trig Derivatives
Basic Trig Derivatives $$\frac{d}{dx}[\sin x] = \cos x$$ $$\frac{d}{dx}[\cos x] = -\sin x$$ $$\frac{d}{dx}[\tan x] = \sec^2 x$$
Reciprocal Trig Derivatives $$\frac{d}{dx}[\csc x] = -\csc x\cot x$$ $$\frac{d}{dx}[\sec x] = \sec x\tan x$$ $$\frac{d}{dx}[\cot x] = -\csc^2 x$$
Memory Aid
The "co-" functions (cos, csc, cot) all have a negative sign in their derivative. This follows from the chain rule applied to identities like cos x = sin(π/2 − x).
Example
d/dx[tan x] = d/dx[sin x / cos x] = (cos x · cos x − sin x · (−sin x)) / cos²x = (cos²x + sin²x)/cos²x = 1/cos²x = sec²x
§3.6 The Chain Rule
Chain Rule
Chain Rule
If g is differentiable at x and f is differentiable at g(x), then: d/dx[f(g(x))] = f'(g(x)) · g'(x) In Leibniz notation: dy/dx = (dy/du)·(du/dx)
Chain Rule Examples $$\frac{d}{dx}[\sin(x^2)] = \cos(x^2)\cdot 2x$$ $$\frac{d}{dx}[e^{3x}] = 3e^{3x}$$ $$\frac{d}{dx}[(x^2+1)^5] = 5(x^2+1)^4\cdot 2x$$ $$\frac{d}{dx}[\ln(\sin x)] = \frac{\cos x}{\sin x} = \cot x$$ $$\frac{d}{dx}[\sqrt{x^2+1}] = \frac{x}{\sqrt{x^2+1}}$$
General Power Rule (via Chain)
d/dx[u^n] = n·u^(n−1) · u'
Implicit Differentiation
Method
When y is defined implicitly by F(x, y) = 0, differentiate both sides with respect to x, treating y as a function of x. Whenever differentiating a term in y, multiply by dy/dx. Then solve for dy/dx.
Example — Circle x²+y²=25
Differentiate: 2x + 2y·(dy/dx) = 0
Solve: dy/dx = −x/y
At (3,4): dy/dx = −3/4 (slope of tangent)
Example — x³+y³=6xy
3x² + 3y²(dy/dx) = 6y + 6x(dy/dx)
dy/dx(3y²−6x) = 6y − 3x²
dy/dx = (2y − x²)/(y² − 2x)
§3.7–3.9 Inverse, Exponential & Logarithmic Derivatives
Derivatives of Inverse & Inverse Trig
Inverse Function Theorem
If f is differentiable at a with f'(a) ≠ 0, then f⁻¹ is differentiable at b = f(a) and: (f⁻¹)'(b) = 1 / f'(f⁻¹(b))
Inverse Trig Derivatives $$\frac{d}{dx}[\sin^{-1}x] = \frac{1}{\sqrt{1-x^2}}$$ $$\frac{d}{dx}[\cos^{-1}x] = \frac{-1}{\sqrt{1-x^2}}$$ $$\frac{d}{dx}[\tan^{-1}x] = \frac{1}{1+x^2}$$ $$\frac{d}{dx}[\cot^{-1}x] = \frac{-1}{1+x^2}$$ $$\frac{d}{dx}[\sec^{-1}x] = \frac{1}{|x|\sqrt{x^2-1}}$$ $$\frac{d}{dx}[\csc^{-1}x] = \frac{-1}{|x|\sqrt{x^2-1}}$$
Exponential & Log Derivatives
Exp & Log Derivatives $$\frac{d}{dx}[e^x] = e^x$$ $$\frac{d}{dx}[b^x] = b^x\ln b \quad (b>0)$$ $$\frac{d}{dx}[\ln x] = \frac{1}{x} \quad (x>0)$$ $$\frac{d}{dx}[\ln|x|] = \frac{1}{x} \quad (x\ne 0)$$ $$\frac{d}{dx}[\log_b x] = \frac{1}{x\ln b}$$ $$\frac{d}{dx}[e^u] = e^u u', \quad \frac{d}{dx}[\ln u] = \frac{u'}{u}$$
Logarithmic Differentiation
Take ln of both sides, differentiate implicitly. Useful for products/quotients of many terms or variable exponents.
Example — Logarithmic Diff.
y = xˣ → ln y = x·ln x
(1/y)·y' = ln x + 1
y' = xˣ(ln x + 1)
Higher-Order Derivatives
Definition
The second derivative f''(x) = d²y/dx² = d/dx[f'(x)].
The nth derivative: f^(n)(x) = d^n y / dx^n.
Examples $$f(x)=x^5:\; f'=5x^4,\; f''=20x^3,\; f'''=60x^2$$ $$f^{(4)}=120x,\; f^{(5)}=120,\; f^{(6)}=0$$ $$f(x)=\sin x:\; f'=\cos x,\; f''=-\sin x$$ $$f'''=-\cos x,\; f^{(4)}=\sin x \quad \text{(period 4)}$$
Related Rates
Method
Related rates problems: quantities x, y both change with time t. Use implicit differentiation w.r.t. t.
Strategy 1. Draw diagram; label variables 2. Write equation relating variables 3. Differentiate both sides w.r.t. $t$ 4. Substitute known values; solve for desired rate
Example — Ladder Problem
x² + y² = 25 (5m ladder).
Differentiate: 2x(dx/dt) + 2y(dy/dt) = 0
At x=3, y=4, dx/dt=2: 6(2)+8(dy/dt)=0 → dy/dt = −3/2 m/s
